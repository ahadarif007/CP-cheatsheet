# Regular Expression

> C# regex lives in `System.Text.RegularExpressions`.
> `Regex.Replace`, `Regex.Match`, and `Regex.Matches` cover most use cases.

## Remove Characters by Pattern

```csharp
using System.Text.RegularExpressions;

// Remove all vowels from a string
string noVowels = Regex.Replace(text, "[aeiou]", "");
```

## Check if a Pattern Matches

```csharp
// Check if the string contains only digits
bool allDigits = Regex.IsMatch(input, @"^\d+$");
```

## Extract Matches

```csharp
// Find all words (sequences of word characters)
MatchCollection matches = Regex.Matches(text, @"\w+");

foreach (Match match in matches)
{
    string word = match.Value;
}
```

## Named Groups

```csharp
// Extract parts of a date string
var pattern = @"(?<year>\d{4})-(?<month>\d{2})-(?<day>\d{2})";
var match = Regex.Match("2026-07-20", pattern);

if (match.Success)
{
    string year  = match.Groups["year"].Value;    // "2026"
    string month = match.Groups["month"].Value;   // "07"
    string day   = match.Groups["day"].Value;     // "20"
}
```

## Split by Pattern

```csharp
// Split on one or more non-word characters
string[] tokens = Regex.Split(text, @"\W+");
```

## Compiled Regex (Performance)

```csharp
// Compile once, reuse many times
private static readonly Regex NumberPattern = new Regex(
    @"\d+",
    RegexOptions.Compiled
);

var results = NumberPattern.Matches(text);
```
