# PriorityQueue

> `PriorityQueue<TElement, TPriority>` is C#'s built-in min-heap (added in .NET 6).
> Elements are dequeued in ascending priority order by default.
> For a max-heap, supply a reversed comparer.

## Min-Heap (Default)

```
Enqueue: 3, 1, 2
Internal heap:    1
                 / \
                3   2

Dequeue order: 1, 2, 3
```

```csharp
var minHeap = new PriorityQueue<int, int>();

minHeap.Enqueue(3, 3);   // element=3, priority=3
minHeap.Enqueue(1, 1);   // element=1, priority=1
minHeap.Enqueue(2, 2);   // element=2, priority=2

int smallest = minHeap.Dequeue();   // returns 1
```

## Max-Heap (Reversed Comparer)

```csharp
// Create a max-heap by reversing the comparison
var maxHeap = new PriorityQueue<int, int>(
    Comparer<int>.Create((a, b) => b.CompareTo(a))
);

maxHeap.Enqueue(3, 3);
maxHeap.Enqueue(1, 1);
int largest = maxHeap.Dequeue();   // returns 3
```

## Top K Frequent Elements

Find the K most frequent elements using a min-heap of size K.

```
Input:  nums = [1,1,1,2,2,3], k = 2

Step 1 -- Count frequencies:
  {1:3, 2:2, 3:1}

Step 2 -- Min-heap (keep top K by frequency):
  Process 1(freq=3): heap = [(1,3)]           size=1
  Process 2(freq=2): heap = [(2,2),(1,3)]     size=2
  Process 3(freq=1): heap = [(3,1),(2,2),(1,3)] size=3 > k
                     Dequeue min -> removes 3
                     heap = [(2,2),(1,3)]     size=2

Result: [1, 2]
```

```csharp
int[] TopKFrequent(int[] nums, int k)
{
    // Step 1: count frequencies
    var frequency = new Dictionary<int, int>();
    foreach (var number in nums)
    {
        frequency[number] = frequency.TryGetValue(number, out var count)
            ? count + 1
            : 1;
    }

    // Step 2: maintain a min-heap of size k (priority = frequency)
    var heap = new PriorityQueue<int, int>();
    foreach (var entry in frequency)
    {
        heap.Enqueue(entry.Key, entry.Value);

        // If heap exceeds size k, remove the least frequent
        if (heap.Count > k)
            heap.Dequeue();
    }

    // Step 3: extract results
    var result = new List<int>();
    while (heap.Count > 0)
        result.Add(heap.Dequeue());

    result.Reverse();   // most frequent first
    return result.ToArray();
}
```
