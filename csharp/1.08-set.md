# Set

> `HashSet<T>` stores unique elements with O(1) average add, remove, and lookup.
> Useful for deduplication, membership tests, and set operations (union, intersection).

## Basic Operations

```csharp
var seen = new HashSet<int>();

seen.Add(42);                      // add an element (returns false if already present)
seen.Remove(42);                   // remove an element (returns false if not found)
bool exists = seen.Contains(42);   // membership test
int anyElement = seen.First();     // grab an arbitrary element
```

## Set Operations

```csharp
var setA = new HashSet<int> { 1, 2, 3, 4 };
var setB = new HashSet<int> { 3, 4, 5, 6 };

// Union: {1, 2, 3, 4, 5, 6} -- modifies setA in place
setA.UnionWith(setB);

// Intersection: {3, 4} -- modifies setA in place
setA.IntersectWith(setB);

// Difference: elements in setA but not in setB
setA.ExceptWith(setB);
```

## Initialize from a Collection

```csharp
int[] nums = { 1, 2, 2, 3, 3, 3 };
var unique = new HashSet<int>(nums);   // {1, 2, 3}
```
