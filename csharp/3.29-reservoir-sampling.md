# Reservoir Sampling

> Randomly pick an index of a target value with equal probability, without knowing
> the total count in advance. For the kth occurrence, replace the current pick with
> probability 1/k. This guarantees uniform distribution. Time: O(n), Space: O(1).

```
  Example: nums = [1, 2, 3, 3, 3], target = 3

  i=0: nums[0]=1, skip (not target)
  i=1: nums[1]=2, skip (not target)
  i=2: nums[2]=3, count=1, random(1,1)=1 --> pick index 2   (prob 1/1)
  i=3: nums[3]=3, count=2, random(1,2):
       if 1 --> pick index 3  (prob 1/2)
       if 2 --> keep index 2  (prob 1/2)
  i=4: nums[4]=3, count=3, random(1,3):
       if 1 --> pick index 4  (prob 1/3)
       if 2 or 3 --> keep     (prob 2/3)

  Each of indices 2, 3, 4 has exactly 1/3 probability.
```

## Random Pick with Reservoir Sampling

```csharp
class Picker
{
    int[] nums;
    Random random = new();

    public Picker(int[] nums)
    {
        this.nums = nums;
    }

    public int Pick(int target)
    {
        int? result = null;
        int count = 0;

        for (int i = 0; i < nums.Length; i++)
        {
            if (nums[i] == target)
            {
                count++;
                int chance = random.Next(1, count + 1);  // Random in [1, count]

                if (chance == 1)     // Replace with probability 1/count
                    result = i;
            }
        }

        return result ?? -1;
    }
}
```
