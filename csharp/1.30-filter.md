# Filter

> LINQ's `Where` is C#'s equivalent of `filter` -- it selects elements that match a predicate.
> Returns a new sequence containing only the matching elements.

## Basic Where (Filter)

```
Input:  [90, 60, 80, 50, 95]
                  ↓ Where(x => x > 75)
Output: [90, 80, 95]
```

```csharp
var highScores = scores.Where(score => score > 75).ToList();
```

## Chaining Where with Select

```csharp
// Filter then transform
var result = numbers
    .Where(n => n % 2 == 0)           // keep even numbers
    .Select(n => n * n)                // square them
    .ToList();
```

## Filter with Index

```csharp
// Where overload that provides the index
var everyOther = items
    .Where((item, index) => index % 2 == 0)   // keep items at even indices
    .ToList();
```

## Remove Nulls

```csharp
var nonNull = items.Where(item => item != null).ToList();
```
