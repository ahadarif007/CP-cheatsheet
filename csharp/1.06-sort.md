# Sort

```csharp
Array.Sort(nums);
Array.Sort(words, (x,y) => x.Length.CompareTo(y.Length));
list.Sort();
list.Sort((x,y) => x[1].CompareTo(y[1]));

var m = new Dictionary<string,int>();
var e = m.ToList();
e.Sort((a,b) => a.Value.CompareTo(b.Value));
```
