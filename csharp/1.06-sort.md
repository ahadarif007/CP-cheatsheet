# Sort

> C# provides `Array.Sort` for arrays and `List.Sort` for lists (both in-place).
> LINQ's `OrderBy` returns a new sequence without mutating the original.

## Sort an Array

```csharp
// Default ascending sort (in-place)
Array.Sort(nums);

// Custom comparator -- sort strings by length
Array.Sort(words, (a, b) => a.Length.CompareTo(b.Length));
```

## Sort a List

```csharp
// Default ascending sort (in-place)
items.Sort();

// Custom comparator -- sort by second element of inner list
items.Sort((a, b) => a[1].CompareTo(b[1]));
```

## Sort a Dictionary by Value

```csharp
var frequency = new Dictionary<string, int>();

// Convert to list of key-value pairs and sort by value
var sorted = frequency.ToList();
sorted.Sort((a, b) => a.Value.CompareTo(b.Value));
```

## Keep Original Indices After Sorting

Useful when you need to know where each element was before sorting.

```
Original: values = [40, 10, 30, 20]
                     0   1   2   3    <-- original indices

Sorted:   [(1,10), (3,20), (2,30), (0,40)]
            idx=1   idx=3   idx=2   idx=0
```

```csharp
// Pair each value with its original index, then sort by value
var indexedValues = nums
    .Select((value, index) => (index, value))
    .OrderBy(pair => pair.value)
    .ToArray();
```
