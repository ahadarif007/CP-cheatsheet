# Events

> Line-sweep technique for finding gaps between intervals. Convert each interval into
> a +1 (start) and -1 (end) event, sort by time, then sweep while tracking the active
> count. When the count drops to zero, a gap begins. Time: O(n log n), Space: O(n).

```
  Example: schedule = [[[1,3],[6,7]], [[2,4]], [[2,5],[9,12]]]

  Events (sorted by time):
    (1,+1) (2,+1) (2,+1) (3,-1) (4,-1) (5,-1) (6,+1) (7,-1) (9,+1) (12,-1)

  Sweep:
    t=1: balance=1
    t=2: balance=3
    t=3: balance=2
    t=4: balance=1
    t=5: balance=0  --> gap starts
    t=6: balance=1  --> gap [5,6] recorded
    t=7: balance=0  --> gap starts
    t=9: balance=1  --> gap [7,9] recorded
    t=12: balance=0

  Free time: [[5,6], [7,9]]
```

## Employee Free Time (Line Sweep)

```csharp
List<int[]> EmployeeFreeTime(List<List<int[]>> schedule)
{
    // Create +1/-1 events for each interval
    var events = new List<int[]>();
    foreach (var employee in schedule)
    {
        foreach (var meeting in employee)
        {
            events.Add(new[] { meeting[0], 1 });   // interval start
            events.Add(new[] { meeting[1], -1 });   // interval end
        }
    }

    events.Sort((a, b) => a[0].CompareTo(b[0]));

    var freeIntervals = new List<int[]>();
    int? previous = null;
    int balance = 0;

    foreach (var ev in events)
    {
        int time = ev[0];
        int change = ev[1];

        // If balance is 0 and there's a gap since last event, record free time
        if (balance == 0 && previous != null && time != previous)
            freeIntervals.Add(new[] { previous.Value, time });

        balance += change;
        previous = time;
    }

    return freeIntervals;
}
```
