# Constants

> C# exposes min/max values for numeric types via static fields on the type struct.
> Commonly used as sentinel values in algorithms (e.g., initializing min/max searches).

## Integer Limits

```csharp
int maxInt  = int.MaxValue;    //  2,147,483,647  (~2.1 billion)
int minInt  = int.MinValue;    // -2,147,483,648

long maxLong = long.MaxValue;  //  9,223,372,036,854,775,807
long minLong = long.MinValue;  // -9,223,372,036,854,775,808
```

## Typical Usage in Algorithms

```csharp
// Initialize to extreme values when searching for min / max
int smallest = int.MaxValue;
int largest  = int.MinValue;

foreach (var value in nums)
{
    smallest = Math.Min(smallest, value);
    largest  = Math.Max(largest, value);
}
```

## Other Numeric Limits

```csharp
double posInf = double.PositiveInfinity;
double negInf = double.NegativeInfinity;
double nan    = double.NaN;

bool isInf = double.IsInfinity(posInf);   // true
bool isNan = double.IsNaN(nan);           // true
```
