# Ternary

> The ternary operator `condition ? valueIfTrue : valueIfFalse` is a concise inline conditional.
> It is an expression (returns a value), unlike `if` which is a statement.

## Basic Ternary Expression

```csharp
int result = condition ? valueA : valueB;
```

## Common Patterns

```csharp
// Absolute value
int abs = x >= 0 ? x : -x;

// Clamp to non-negative
int clamped = value < 0 ? 0 : value;

// Min of two values
int smaller = a < b ? a : b;
```

## Null-Coalescing Operator (Related)

```csharp
// ?? returns the left operand if non-null, otherwise the right
string name = input ?? "default";

// ??= assigns only if the variable is null
name ??= "fallback";
```
