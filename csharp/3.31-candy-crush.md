# Candy Crush

> Remove groups of k adjacent duplicate characters using a stack. Each stack entry tracks
> a character and its consecutive count. When the count reaches k, pop the entry (crush it).
> Time: O(n), Space: O(n).

```
  Example: s = "deeedbbcccbdaa", k = 3

  Stack trace (char, count):
    'd' -> (d,1)
    'e' -> (d,1)(e,1)
    'e' -> (d,1)(e,2)
    'e' -> (d,1)(e,3) --> count=3, CRUSH! -> (d,1)
    'd' -> (d,2)
    'b' -> (d,2)(b,1)
    'b' -> (d,2)(b,2)
    'c' -> (d,2)(b,2)(c,1)
    'c' -> (d,2)(b,2)(c,2)
    'c' -> (d,2)(b,2)(c,3) --> CRUSH! -> (d,2)(b,2)
    'b' -> (d,2)(b,3) --> CRUSH! -> (d,2)
    'd' -> (d,3) --> CRUSH! -> empty
    'a' -> (a,1)
    'a' -> (a,2)

  Result: "aa"
```

## Remove K Adjacent Duplicates

```csharp
string RemoveDuplicates(string s, int k)
{
    var stack = new Stack<(char character, int count)>();

    foreach (var ch in s)
    {
        if (stack.Count > 0 && stack.Peek().character == ch)
        {
            var top = stack.Pop();
            top.count++;

            if (top.count < k)       // Not yet k in a row; keep it
                stack.Push(top);
            // else: count == k, crushed (don't push back)
        }
        else
        {
            stack.Push((ch, 1));      // New character group
        }
    }

    // Rebuild the string from stack entries
    var result = stack.Reverse()
        .SelectMany(entry => Enumerable.Repeat(entry.character, entry.count))
        .ToArray();

    return new string(result);
}
```
