# Random

> `System.Random` generates pseudo-random numbers.
> For thread-safe usage in .NET 6+, use `Random.Shared`.

## Generate Random Numbers

```csharp
var rng = new Random();

// Random integer in [lo, hi) -- hi is exclusive
int randomInt = rng.Next(lo, hi);

// Random integer in [0, max) 
int randomBelow = rng.Next(100);    // 0 to 99

// Random double in [0.0, 1.0)
double randomDouble = rng.NextDouble();
```

## Fisher-Yates Shuffle

Shuffles an array uniformly at random in O(n) time.

```
Input:  [1, 2, 3, 4, 5]

i=4: swap a[4] with a[rand(0..4)]  -> e.g. [1, 2, 5, 4, 3]
i=3: swap a[3] with a[rand(0..3)]  -> e.g. [1, 4, 5, 2, 3]
i=2: swap a[2] with a[rand(0..2)]  -> e.g. [5, 4, 1, 2, 3]
i=1: swap a[1] with a[rand(0..1)]  -> e.g. [4, 5, 1, 2, 3]
```

```csharp
int[] Shuffle(int[] array)
{
    var rng = new Random();

    for (int i = array.Length - 1; i > 0; i--)
    {
        // Pick a random index from [0, i]
        int swapIndex = rng.Next(i + 1);

        // Swap current element with the randomly chosen one
        (array[i], array[swapIndex]) = (array[swapIndex], array[i]);
    }

    return array;
}
```
