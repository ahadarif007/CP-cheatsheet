# Math

> The `Math` class provides common mathematical functions.
> For competitive programming, modular exponentiation and division with remainder are frequently needed.

## Power and Modular Arithmetic

```csharp
// Power: a^b (returns double)
double power = Math.Pow(base, exponent);

// Modular power (cast to long to avoid overflow)
long modPower = (long)Math.Pow(base, exponent) % mod;
```

## Division with Remainder

```csharp
// DivRem returns quotient and outputs remainder
long quotient = Math.DivRem(dividend, divisor, out var remainder);
```

## Common Math Functions

```csharp
int absValue  = Math.Abs(-5);          // 5
int maxValue  = Math.Max(3, 7);        // 7
int minValue  = Math.Min(3, 7);        // 3
double sqrt   = Math.Sqrt(16);         // 4.0
double ceil   = Math.Ceiling(2.3);     // 3.0
double floor  = Math.Floor(2.7);       // 2.0
double log    = Math.Log(8, 2);        // 3.0  (log base 2 of 8)
```

## GCD (Greatest Common Divisor)

```csharp
// Available in .NET 7+ via int/long
// For older versions, implement manually:
int Gcd(int a, int b)
{
    while (b != 0)
    {
        (a, b) = (b, a % b);
    }
    return a;
}

int Lcm(int a, int b)
{
    return a / Gcd(a, b) * b;   // divide first to avoid overflow
}
```
