# Greedy

> A **greedy** algorithm makes the locally optimal choice at each step, hoping to reach a global optimum. Greedy works when the problem has **optimal substructure** and a **greedy choice property** -- choosing the best local option never blocks a better global solution.

## Increasing Triplet Subsequence

Find if there exist indices `i < j < k` such that `nums[i] < nums[j] < nums[k]`. Track the **two smallest values seen so far**. Time: **O(n)**, Space: **O(1)**.

```
  nums = [2, 1, 5, 0, 4, 6]

  Step through:
    x=2:  first=2, second=MAX
    x=1:  first=1, second=MAX        (1 < 2, update first)
    x=5:  first=1, second=5          (5 > first, update second)
    x=0:  first=0, second=5          (0 < 1, update first)
    x=4:  first=0, second=4          (4 > first but < 5, update second)
    x=6:  6 > second(4)  ->  return true!
```

```csharp
bool IncreasingTriplet(int[] nums)
{
    int first = int.MaxValue;
    int second = int.MaxValue;

    foreach (var num in nums)
    {
        if (num <= first)
            first = num;          // Smallest so far
        else if (num <= second)
            second = num;         // Second smallest so far
        else
            return true;          // Found a third value > second > first
    }

    return false;
}
```
