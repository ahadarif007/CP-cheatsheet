# Slicing

> C# supports substring extraction via `Substring()` and array slicing via range operators (`..`).
> Range syntax uses `start..end` where `start` is inclusive and `end` is exclusive.

## String Substring

```
Index:   0   1   2   3   4   5
Char:   'h' 'e' 'l' 'l' 'o' '!'
              start=2, length=2
              |-------|
Result:       "ll"
```

```csharp
// Substring(startIndex, length) -- extracts "length" characters starting at "startIndex"
string sub = text.Substring(2, 2);    // e.g. "hello!" -> "ll"
```

## Array Slicing with Range Operator

```
Index:   0   1   2   3   4   5
Value:  [10, 20, 30, 40, 50, 60]
                  ^----------^
                 arr[2..5] -> [30, 40, 50]
```

```csharp
// Range operator: arr[start..end] -- start inclusive, end exclusive
int[] slice = arr[2..5];              // elements at indices 2, 3, 4

// From the end using ^ (hat) operator
int[] lastTwo  = arr[^2..];           // last two elements
int[] allButLast = arr[..^1];         // everything except the last
```
