# Default Dict

> C# lacks Python's `defaultdict`, but you can emulate it with `Dictionary<TKey, TValue>` and a check-then-create pattern using `TryGetValue`. This is common when building **adjacency lists**, **grouping** elements, or collecting values by key.

```
  Goal: Group edges by node
  Input pairs: (1,2), (1,3), (2,4)

  Step by step:
    key=1 not found -> create list -> add 2  =>  {1:[2]}
    key=1 found     -> add 3                 =>  {1:[2,3]}
    key=2 not found -> create list -> add 4  =>  {1:[2,3], 2:[4]}
```

## Pattern: Dictionary with List Values

Use this pattern whenever you need to collect multiple values per key.

```csharp
var adjacencyList = new Dictionary<int, List<int>>();

// Safe insert: create list if key is missing, then add value
if (!adjacencyList.TryGetValue(1, out var neighbors))
{
    neighbors = new List<int>();
    adjacencyList[1] = neighbors;
}
neighbors.Add(2);
```

## Helper Extension Method

To avoid repeating the pattern, use a reusable extension method.

```csharp
static class DictExtensions
{
    public static TValue GetOrCreate<TKey, TValue>(
        this Dictionary<TKey, TValue> dict,
        TKey key) where TValue : new()
    {
        if (!dict.TryGetValue(key, out var value))
        {
            value = new TValue();
            dict[key] = value;
        }
        return value;
    }
}

// Usage:
var groups = new Dictionary<int, List<int>>();
groups.GetOrCreate(1).Add(2);
groups.GetOrCreate(1).Add(3);
// Result: {1: [2, 3]}
```
