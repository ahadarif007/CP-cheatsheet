# All

> `All()` returns `true` only if every element in the sequence satisfies the predicate.
> Short-circuits on the first failure -- stops scanning once a non-matching element is found.

## Basic Usage

```csharp
bool allNonNegative = nums.All(x => x >= 0);   // true if every element >= 0
```

## Practical Examples

```csharp
// Check if all strings are non-empty
bool allFilled = words.All(word => word.Length > 0);

// Validate all scores are in range
bool allValid = scores.All(score => score >= 0 && score <= 100);
```

## Relationship with Any

```
All(condition)  ==  !Any(x => !condition(x))
Any(condition)  ==  !All(x => !condition(x))
```

```csharp
// These are equivalent:
bool noneNegative = nums.All(x => x >= 0);
bool noneNegative2 = !nums.Any(x => x < 0);
```
