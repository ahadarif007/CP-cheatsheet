# For Else

> C# does not have a `for-else` construct like Python.
> The pattern is simulated with a boolean flag that tracks whether the loop completed without breaking.

## Pattern: Flag-Based For-Else

```
Python equivalent:
  for item in items:
      if condition:
          break
  else:
      # ran if loop completed without break

C# equivalent:
  bool broken = false;
  for (...) { if (condition) { broken = true; break; } }
  if (!broken) { /* no break occurred */ }
```

```csharp
bool didBreak = false;

for (int i = 0; i < nums.Length; i++)
{
    if (nums[i] == target)
    {
        didBreak = true;
        break;
    }
}

if (!didBreak)
{
    // This block runs only if the loop completed without breaking
    // (equivalent to Python's for-else)
}
```

## Alternative: Use a Method with Early Return

```csharp
bool Found(int[] nums, int target)
{
    foreach (var value in nums)
    {
        if (value == target)
            return true;
    }
    return false;   // "else" branch -- loop completed without match
}
```
