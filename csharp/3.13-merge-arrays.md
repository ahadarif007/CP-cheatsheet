# Merge K Sorted Arrays

> Merge **k sorted arrays** into one sorted array using a **min-heap** (priority queue). The heap always holds the smallest unprocessed element from each array. Time: **O(N log k)** where N is the total number of elements and k is the number of arrays.

```
  Input arrays:
    [1, 4, 7]
    [2, 5, 8]
    [3, 6, 9]

  Min-heap state at each step:
    Initial:  heap = [(1,arr0), (2,arr1), (3,arr2)]
    Pop 1  -> result=[1],   push 4  -> heap=[(2,arr1), (3,arr2), (4,arr0)]
    Pop 2  -> result=[1,2], push 5  -> heap=[(3,arr2), (4,arr0), (5,arr1)]
    Pop 3  -> result=[1,2,3], push 6 -> heap=[(4,arr0), (5,arr1), (6,arr2)]
    ...continues until all elements processed...
    Result: [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

## Merge K Sorted Arrays with Priority Queue

Each heap entry tracks the value, which array it came from, and the index within that array.

```csharp
int[] MergeKSortedArrays(List<int[]> arrays)
{
    // Priority queue: (value, arrayIndex, elementIndex), prioritized by value
    var priorityQueue = new PriorityQueue<(int val, int arrayIdx, int elemIdx), int>();

    // Seed the heap with the first element of each array
    for (int i = 0; i < arrays.Count; i++)
    {
        if (arrays[i].Length > 0)
            priorityQueue.Enqueue((arrays[i][0], i, 0), arrays[i][0]);
    }

    var result = new List<int>();

    while (priorityQueue.Count > 0)
    {
        var current = priorityQueue.Dequeue();
        result.Add(current.val);

        // Advance to next element in the same array
        int arrayIdx = current.arrayIdx;
        int nextElemIdx = current.elemIdx + 1;

        if (nextElemIdx < arrays[arrayIdx].Length)
        {
            int nextVal = arrays[arrayIdx][nextElemIdx];
            priorityQueue.Enqueue((nextVal, arrayIdx, nextElemIdx), nextVal);
        }
    }

    return result.ToArray();
}
```
