# Reduce

> LINQ's `Aggregate` is C#'s equivalent of `reduce` -- it combines all elements into a single value.
> Takes a seed (initial value) and a function that accumulates each element.

## Basic Aggregate (Reduce)

```
Input: [1, 2, 3, 4, 5], seed = 0, func = (acc, x) => acc + x

Step 1:  0 + 1 = 1
Step 2:  1 + 2 = 3
Step 3:  3 + 3 = 6
Step 4:  6 + 4 = 10
Step 5: 10 + 5 = 15

Result: 15
```

```csharp
int sum = nums.Aggregate(0, (accumulator, value) => accumulator + value);
```

## Product of All Elements

```csharp
int product = nums.Aggregate(1, (accumulator, value) => accumulator * value);
```

## String Concatenation

```csharp
string combined = words.Aggregate("", (accumulator, word) => accumulator + " " + word).Trim();
```

## Find Maximum (Manual)

```csharp
int max = nums.Aggregate((current, next) => current > next ? current : next);
```

## Built-In Shorthand Alternatives

```csharp
// For common reductions, LINQ has dedicated methods:
int sum     = nums.Sum();
int min     = nums.Min();
int max     = nums.Max();
double avg  = nums.Average();
```
