# Hash

> A hash map (Dictionary) counts occurrences or maps keys to values in O(1) average time.
> `TryGetValue` avoids `KeyNotFoundException` when the key might be missing.

## Frequency Counter

Build a character frequency map from a string.

```
Input:  "abracadabra"
Output: { a:5, b:2, r:2, c:1, d:1 }
```

```csharp
var frequency = new Dictionary<char, int>();

foreach (var character in text)
{
    // If key exists, increment; otherwise start at 1
    frequency[character] = frequency.TryGetValue(character, out var count)
        ? count + 1
        : 1;
}
```

## Alternative: Using ContainsKey

```csharp
var frequency = new Dictionary<char, int>();

foreach (var character in text)
{
    if (frequency.ContainsKey(character))
        frequency[character]++;
    else
        frequency[character] = 1;
}
```
