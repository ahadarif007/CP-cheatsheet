# Union Find / DSU

> Disjoint Set Union tracks connected components. Two key operations:
> **Find** (with path compression) returns the root representative of a set.
> **Union** (by size) merges two sets by attaching the smaller tree under the larger.
> Near O(1) amortized per operation due to path compression + union by size.

```
  Example: Union(0,1), Union(2,3), Union(1,3)

  Initial:    0  1  2  3     (each is its own root)

  Union(0,1): 0--1  2  3    (1's parent = 0)

  Union(2,3): 0--1  2--3    (3's parent = 2)

  Union(1,3): Find(1)=0, Find(3)=2
              size[0]=2, size[2]=2
              Attach 2 under 0:
                  0
                 / \
                1   2
                    |
                    3

  Find(3) with path compression: 3 -> 2 -> 0, then 3.parent = 0
```

## Disjoint Set Union (DSU) with Path Compression and Union by Size

```csharp
class DSU
{
    int[] parent, size;

    public DSU(int n)
    {
        parent = new int[n];
        size = new int[n];
        for (int i = 0; i < n; i++)
        {
            parent[i] = i;     // Each element is its own root
            size[i] = 1;
        }
    }

    int Find(int x)
    {
        if (parent[x] != x)
            parent[x] = Find(parent[x]);   // Path compression
        return parent[x];
    }

    public bool Union(int a, int b)
    {
        int rootA = Find(a);
        int rootB = Find(b);

        if (rootA == rootB)
            return false;                    // Already in same set

        // Union by size: attach smaller tree under larger
        if (size[rootA] < size[rootB])
            (rootA, rootB) = (rootB, rootA);

        parent[rootB] = rootA;
        size[rootA] += size[rootB];
        return true;
    }
}
```
