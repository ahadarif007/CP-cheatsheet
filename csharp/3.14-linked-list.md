# Linked List

> A **linked list** is a linear data structure where each node points to the next. Common patterns: **two pointers** (fast/slow for cycle detection), **dummy head** (simplifies edge cases for merge/insert), and **in-place reversal** (three-pointer technique).

```
  Singly linked list structure:
    head -> [1|*] -> [2|*] -> [3|*] -> [4|*] -> null
             val  next
```

## ListNode Definition

```csharp
class ListNode
{
    public int val;
    public ListNode next;

    public ListNode(int val = 0, ListNode next = null)
    {
        this.val = val;
        this.next = next;
    }
}
```

## Reverse a Linked List

Reverse the list in-place using three pointers. Time: **O(n)**, Space: **O(1)**.

```
  Before:  1 -> 2 -> 3 -> 4 -> null
  After:   null <- 1 <- 2 <- 3 <- 4

  Step by step:
    previous=null, current=1:  1.next=null,     previous=1, current=2
    previous=1,    current=2:  2.next=1,         previous=2, current=3
    previous=2,    current=3:  3.next=2,         previous=3, current=4
    previous=3,    current=4:  4.next=3,         previous=4, current=null
  Return previous (node 4) = new head
```

```csharp
ListNode Reverse(ListNode head)
{
    ListNode previous = null, current = head;

    while (current != null)
    {
        var nextNode = current.next;  // Save next before overwriting
        current.next = previous;      // Reverse the link
        previous = current;           // Advance previous
        current = nextNode;           // Advance current
    }

    return previous; // New head of reversed list
}
```

## Merge Two Sorted Lists

Use a **dummy head** to avoid special-casing the first node. Time: **O(n + m)**, Space: **O(1)**.

```
  List a: 1 -> 3 -> 5
  List b: 2 -> 4 -> 6

  Merge steps:
    dummy -> 1 (a < b)
    dummy -> 1 -> 2 (b < a)
    dummy -> 1 -> 2 -> 3 (a < b)
    dummy -> 1 -> 2 -> 3 -> 4 (b < a)
    dummy -> 1 -> 2 -> 3 -> 4 -> 5 (a < b)
    dummy -> 1 -> 2 -> 3 -> 4 -> 5 -> 6 (append remaining b)
```

```csharp
ListNode MergeTwoLists(ListNode listA, ListNode listB)
{
    var dummy = new ListNode(0); // Dummy head simplifies logic
    var pointer = dummy;

    while (listA != null && listB != null)
    {
        if (listA.val < listB.val)
        {
            pointer.next = listA;
            listA = listA.next;
        }
        else
        {
            pointer.next = listB;
            listB = listB.next;
        }
        pointer = pointer.next;
    }

    // Append whichever list has remaining nodes
    pointer.next = listA ?? listB;

    return dummy.next; // Skip dummy, return real head
}
```
