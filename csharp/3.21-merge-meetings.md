# Merge Meetings

> Insert a new interval into a list of intervals, then merge all overlapping ones.
> Sort by start time, then greedily merge: if the current interval overlaps the last
> merged one, extend it; otherwise start a new group. Time: O(n log n), Space: O(n).

```
  Example: intervals = [[1,3],[6,9]], newInterval = [2,5]

  After adding and sorting: [[1,3],[2,5],[6,9]]

  Merge step:
    Start with result = [[1,3]]
    [2,5]: 2 <= 3 (overlaps) --> extend to [1,5]
    [6,9]: 6 > 5 (no overlap) --> add [6,9]

  Result: [[1,5],[6,9]]
```

## Insert and Merge Intervals

```csharp
List<int[]> Insert(int[][] intervals, int[] newInterval)
{
    var allIntervals = intervals.ToList();
    allIntervals.Add(newInterval);
    allIntervals.Sort((a, b) => a[0].CompareTo(b[0]));  // Sort by start time

    var result = new List<int[]> { allIntervals[0] };

    foreach (var current in allIntervals)
    {
        var last = result[^1];

        if (current[0] <= last[1])
            // Overlapping: extend the end of last merged interval
            last[1] = Math.Max(last[1], current[1]);
        else
            // No overlap: start a new interval
            result.Add(current);
    }

    return result;
}
```
