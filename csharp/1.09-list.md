# List

> `List<T>` is C#'s dynamic array -- it grows automatically as elements are added.
> Backed by an array internally, it provides O(1) indexed access and amortized O(1) append.

## Initialize with a Default Value

```csharp
// Create a list of 5 zeros
var filled = Enumerable.Repeat(0, 5).ToList();   // [0, 0, 0, 0, 0]
```

## 2D List (List of Lists)

```csharp
// Create an empty grid, then add rows
var grid = new List<List<int>>();
grid.Add(new List<int>());     // add first row
grid[0].Add(1);                // add element to first row
```

## Reverse a List

```csharp
// In-place reversal
items.Reverse();
```

## Concatenate Two Lists

```csharp
// Create a new list combining both
var combined = new List<int>(listA);
combined.AddRange(listB);
```

## Common List Operations

```csharp
var numbers = new List<int> { 10, 20, 30 };

numbers.Add(40);               // append to end         -> [10, 20, 30, 40]
numbers.Insert(1, 15);         // insert at index 1     -> [10, 15, 20, 30, 40]
numbers.RemoveAt(0);           // remove at index 0     -> [15, 20, 30, 40]
numbers.Remove(30);            // remove first match    -> [15, 20, 40]
int count = numbers.Count;     // number of elements
bool has = numbers.Contains(20); // membership test
```
