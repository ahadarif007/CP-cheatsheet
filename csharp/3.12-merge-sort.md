# Merge Sort

> **Merge sort** divides the array in half, recursively sorts each half, then merges the two sorted halves. Time: **O(n log n)** guaranteed (no worst case degradation). Space: **O(n)** for the auxiliary arrays. Stable sort -- equal elements preserve their original order.

```
  Splitting phase:
    [38, 27, 43, 3, 9, 82, 10]
       /                    \
  [38, 27, 43]         [3, 9, 82, 10]
    /       \             /         \
  [38]   [27, 43]     [3, 9]    [82, 10]
          /    \       /   \     /    \
        [27]  [43]   [3]  [9] [82]  [10]

  Merging phase (bottom-up):
        [27, 43]       [3, 9]     [10, 82]
    [27, 38, 43]         [3, 9, 10, 82]
         [3, 9, 10, 27, 38, 43, 82]
```

## Merge Sort Implementation

```csharp
int[] MergeSort(int[] arr)
{
    if (arr.Length <= 1) return arr;

    int mid = arr.Length / 2;
    var leftHalf = MergeSort(arr[..mid]);     // Sort left half
    var rightHalf = MergeSort(arr[mid..]);    // Sort right half
    return Merge(leftHalf, rightHalf);
}

int[] Merge(int[] leftArr, int[] rightArr)
{
    int leftIndex = 0, rightIndex = 0;
    var result = new int[leftArr.Length + rightArr.Length];
    int writeIndex = 0;

    // Compare and merge elements from both halves
    while (leftIndex < leftArr.Length && rightIndex < rightArr.Length)
    {
        if (leftArr[leftIndex] <= rightArr[rightIndex])
            result[writeIndex++] = leftArr[leftIndex++];
        else
            result[writeIndex++] = rightArr[rightIndex++];
    }

    // Copy remaining elements from left half
    while (leftIndex < leftArr.Length)
        result[writeIndex++] = leftArr[leftIndex++];

    // Copy remaining elements from right half
    while (rightIndex < rightArr.Length)
        result[writeIndex++] = rightArr[rightIndex++];

    return result;
}
```
