# Map

> LINQ's `Select` is C#'s equivalent of `map` -- it transforms each element in a sequence.
> Returns a new sequence; the original collection is not modified.

## Basic Select (Map)

```
Input:  ["hello", "world", "foo"]
                  ↓ Select(s => s.ToUpper())
Output: ["HELLO", "WORLD", "FOO"]
```

```csharp
var uppercased = words.Select(word => word.ToUpper()).ToList();
```

## Transform with Index

```csharp
// Select overload that provides the index
var indexed = words
    .Select((word, index) => $"{index}: {word}")
    .ToList();
// ["0: hello", "1: world", "2: foo"]
```

## Convert Types

```csharp
// Parse strings to integers
string[] raw = { "1", "2", "3" };
int[] numbers = raw.Select(int.Parse).ToArray();

// Extract a field from objects
var names = people.Select(person => person.Name).ToList();
```
