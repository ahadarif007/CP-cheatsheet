# Binary Search Tree

> A **BST** keeps left children smaller and right children larger than each node. This ordering enables O(log n) search, insert, and delete on balanced trees. In-order traversal of a BST yields sorted output.

```
  Example BST:
          8
        /   \
       3     10
      / \      \
     1   6      14
        / \    /
       4   7  13

  In-order traversal: 1, 3, 4, 6, 7, 8, 10, 13, 14  (sorted)
```

## Complete Tree Check by Index

A complete binary tree has all levels full except possibly the last, which is filled left-to-right. If we number nodes starting at 1 (left child = 2*i, right child = 2*i+1), the tree is complete when the **max index equals the node count**. Time: **O(n)**.

```
  Complete:           Not complete:
       1                   1
      / \                 / \
     2   3               2   3
    / \                    \
   4   5                    5

  max_idx=5, count=5      max_idx=5, count=4
  5 == 5  -> true         5 != 4  -> false
```

```csharp
bool IsComplete(TreeNode root)
{
    long totalNodes = 0, maxIndex = 0;

    void Dfs(TreeNode node, long index)
    {
        if (node == null) return;

        totalNodes++;
        if (index > maxIndex)
            maxIndex = index;

        Dfs(node.left, index * 2);      // Left child at 2*i
        Dfs(node.right, index * 2 + 1); // Right child at 2*i+1
    }

    Dfs(root, 1);
    return totalNodes == maxIndex;
}
```

## Range Sum in BST

Sum all node values in range `[low, high]`. Use the BST property to **prune** branches that cannot contain values in range. Time: **O(n)** worst case, often much better.

```csharp
int RangeSumBST(TreeNode root, int low, int high)
{
    int sum = 0;

    void Dfs(TreeNode node)
    {
        if (node == null) return;

        // Add node value if in range
        if (low <= node.val && node.val <= high)
            sum += node.val;

        // Only go left if there could be values >= low
        if (node.val > low)
            Dfs(node.left);

        // Only go right if there could be values <= high
        if (node.val < high)
            Dfs(node.right);
    }

    Dfs(root);
    return sum;
}
```

## Validate BST (Iterative In-Order)

A valid BST has strictly increasing in-order traversal. Use an iterative in-order walk and check that each value exceeds the previous. Time: **O(n)**, Space: **O(h)**.

```
  In-order traversal:
          5
         / \
        3   7
       / \
      2   4

  Visit order: 2, 3, 4, 5, 7
  Each > previous?  3>2, 4>3, 5>4, 7>5  ->  valid BST
```

```csharp
bool IsValidBST(TreeNode root)
{
    var stack = new Stack<TreeNode>();
    TreeNode current = root;
    int? previous = null;

    while (current != null || stack.Count > 0)
    {
        // Go as far left as possible
        while (current != null)
        {
            stack.Push(current);
            current = current.left;
        }

        current = stack.Pop();

        // Check in-order property
        if (previous != null && current.val <= previous)
            return false;

        previous = current.val;
        current = current.right; // Move to right subtree
    }

    return true;
}
```
