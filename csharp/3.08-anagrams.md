# Anagrams

> Two strings are **anagrams** if they contain the same characters with the same frequencies. Use a **frequency array** of size 26 for lowercase letters. Comparing two frequency arrays takes O(1) since the array size is fixed.

```
  "listen"  vs  "silent"

  Frequency counts:
    listen:  e:1 i:1 l:1 n:1 s:1 t:1
    silent:  e:1 i:1 l:1 n:1 s:1 t:1
    -> same frequencies -> anagram!

  "hello"  vs  "world"
    hello:  e:1 h:1 l:2 o:1
    world:  d:1 l:1 o:1 r:1 w:1
    -> different -> not anagram
```

## Check if Two Strings Are Anagrams

Increment counts for the first string, decrement for the second. If any count goes negative, they differ. Time: **O(n)**, Space: **O(1)**.

```csharp
bool IsAnagram(string s, string t)
{
    if (s.Length != t.Length) return false;

    var charCount = new int[26];

    // Count characters in first string
    foreach (var ch in s)
        charCount[ch - 'a']++;

    // Subtract characters in second string
    foreach (var ch in t)
    {
        if (--charCount[ch - 'a'] < 0)
            return false; // More of this char in t than s
    }

    return true;
}
```

## Find All Anagram Indices (Sliding Window)

Given strings `s` and `p`, find all start indices in `s` where a substring is an anagram of `p`. Slide a window of size `p.Length` and compare frequency arrays. Time: **O(n)** where n = s.Length.

```
  s = "cbaebabacd", p = "abc"

  Window slides (size 3):
    [cba]ebabacd  ->  freq matches "abc"  ->  add index 0
    c[bae]babacd  ->  no match
    cb[aeb]abacd  ->  no match
    cba[eba]bacd  ->  no match
    cbae[bab]acd  ->  no match
    cbaeb[aba]cd  ->  no match
    cbaeba[bac]d  ->  freq matches "abc"  ->  add index 6
    cbaebab[acd]  ->  no match
  Answer: [0, 6]
```

```csharp
IList<int> FindAnagrams(string s, string p)
{
    var targetFreq = new int[26];
    foreach (var ch in p)
        targetFreq[ch - 'a']++;

    var windowFreq = new int[26];
    var result = new List<int>();
    int windowSize = p.Length;

    for (int i = 0; i < s.Length; i++)
    {
        // Add right character to window
        windowFreq[s[i] - 'a']++;

        // Remove leftmost character when window exceeds size
        if (i >= windowSize)
            windowFreq[s[i - windowSize] - 'a']--;

        // Compare frequency arrays
        if (targetFreq.SequenceEqual(windowFreq))
            result.Add(i - windowSize + 1);
    }

    return result;
}
```
