# Fast Power

> Compute x^n in O(log n) time using exponentiation by squaring. If the exponent's
> lowest bit is 1, multiply the result by x. Then square x and halve the exponent.
> Handles negative exponents by inverting the base.

```
  Example: x = 2, n = 10

  Binary of 10: 1010

  Step    exponent(binary)  result   x
  ----    ----------------  ------   ------
  Start   1010              1        2
  bit=0:  0101              1        4       (even, just square x)
  bit=1:  0010              4        16      (odd, result *= x)
  bit=0:  0001              4        256     (even, just square x)
  bit=1:  0000              1024     65536   (odd, result *= x)

  Answer: 1024
```

## Exponentiation by Squaring

```csharp
double MyPow(double x, long n)
{
    long exponent = n;

    // Handle negative exponent: x^(-n) = (1/x)^n
    if (exponent < 0)
    {
        x = 1 / x;
        exponent = -exponent;
    }

    double result = 1;

    while (exponent > 0)
    {
        if ((exponent & 1) == 1)    // Odd exponent: multiply result by x
            result *= x;

        x *= x;                     // Square the base
        exponent >>= 1;             // Halve the exponent
    }

    return result;
}
```
