# Trie

> A prefix tree where each node holds a map of child characters. Useful for autocomplete,
> spell checking, and prefix-based search. Insert and lookup are O(L) where L is the word length.

```
  Example: Insert "cat", "car", "cart"

         (root)
           |
          'c'
           |
          'a'
         /   \
       't'   'r'
     [cat]   /  \
           [car] 't'
                [cart]

  MatchPrefix("ca") --> ["cat", "car", "cart"]
  MatchPrefix("cart") --> ["cart"]
```

## Trie with Prefix Search

```csharp
class Trie
{
    class TrieNode
    {
        public Dictionary<char, TrieNode> children = new();
        public string word;    // Non-null if a word ends here
    }

    TrieNode root = new();

    public void Add(string word)
    {
        var current = root;
        foreach (var ch in word)
        {
            // Navigate to child, creating it if absent
            current = current.children.TryGetValue(ch, out var next)
                ? next
                : (current.children[ch] = new TrieNode());
        }
        current.word = word;   // Mark end of word
    }

    public List<string> MatchPrefix(string prefix)
    {
        var current = root;

        // Walk down the trie following the prefix
        foreach (var ch in prefix)
        {
            if (!current.children.ContainsKey(ch))
                return new();       // Prefix not in trie
            current = current.children[ch];
        }

        // Collect all words under this subtree
        var result = new List<string>();
        void Collect(TrieNode node)
        {
            if (node.word != null)
                result.Add(node.word);
            foreach (var child in node.children.Values)
                Collect(child);
        }
        Collect(current);
        return result;
    }
}
```

## Search with '.' Wildcard

The '.' character matches any single letter. Uses DFS to explore all branches
when a wildcard is encountered.

```csharp
class WordDictionary
{
    class TrieNode
    {
        public Dictionary<char, TrieNode> children = new();
        public bool isEnd;
    }

    TrieNode root = new();

    public void AddWord(string word)
    {
        var current = root;
        foreach (var ch in word)
        {
            if (!current.children.ContainsKey(ch))
                current.children[ch] = new TrieNode();
            current = current.children[ch];
        }
        current.isEnd = true;
    }

    public bool Search(string word)
    {
        bool DFS(int index, TrieNode node)
        {
            if (index == word.Length)
                return node.isEnd;

            char ch = word[index];

            if (ch != '.')
                // Exact character match
                return node.children.ContainsKey(ch) && DFS(index + 1, node.children[ch]);

            // Wildcard: try every child branch
            foreach (var child in node.children.Values)
                if (DFS(index + 1, child))
                    return true;

            return false;
        }

        return DFS(0, root);
    }
}
```
