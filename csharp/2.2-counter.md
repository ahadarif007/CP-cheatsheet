Top-K with counter

```csharp
List<int> TopK(int[] nums,int k){ var m=new Dictionary<int,int>(); foreach(var x in nums) m[x]=m.TryGetValue(x,out var v)? v+1:1; return m.OrderByDescending(e=>e.Value).Take(k).Select(e=>e.Key).ToList(); }
```
