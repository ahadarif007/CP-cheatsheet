# Counter

> A **frequency counter** maps each element to its occurrence count. C# has no built-in `Counter` class, so we use `Dictionary<T, int>` with `TryGetValue` for safe incrementing. Essential for anagram checks, top-K problems, and frequency-based grouping.

```
  Input string: "banana"

  Counting step by step:
    'b' -> {b:1}
    'a' -> {b:1, a:1}
    'n' -> {b:1, a:1, n:1}
    'a' -> {b:1, a:2, n:1}
    'n' -> {b:1, a:2, n:2}
    'a' -> {b:1, a:3, n:2}
```

## Building a Frequency Map

Time complexity: **O(n)** where n is the length of the input.

```csharp
var frequency = new Dictionary<char, int>();

foreach (var ch in s)
{
    // TryGetValue returns false if key missing, sets count to 0
    frequency.TryGetValue(ch, out var count);
    frequency[ch] = count + 1;
}
```

## Top-K Frequent Elements

Given an array, return the **k most frequent** elements. Time complexity: **O(n log n)** with sorting, or O(n log k) with a heap.

```csharp
List<int> TopKFrequent(int[] nums, int k)
{
    // Step 1: Count frequencies
    var frequency = new Dictionary<int, int>();
    foreach (var num in nums)
    {
        frequency.TryGetValue(num, out var count);
        frequency[num] = count + 1;
    }

    // Step 2: Sort by frequency descending and take top k
    return frequency
        .OrderByDescending(entry => entry.Value)
        .Take(k)
        .Select(entry => entry.Key)
        .ToList();
}
```
