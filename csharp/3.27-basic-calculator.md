# Basic Calculator

> Evaluate an expression string with `+`, `-`, `*`, `/`, and parentheses using recursive descent.
> A stack accumulates terms: `+` pushes, `-` pushes negated, `*`/`/` modify the top.
> Parentheses trigger a recursive call. A sentinel `$` marks the end. Time: O(n), Space: O(n).

```
  Example: "2*(3+4)-5"

  Append sentinel: "2*(3+4)-5$"

  Outer call: sign='+', num=2
    Hit '(' --> recurse at index 2
      Inner call: sign='+', num=3
        Hit '+' --> push 3, sign='+', num=4
        Hit ')' --> push 4, return sum=7, index=7
    num=7, hit '*' --> push nothing yet (sign='+')
    Actually sign='+' so push 2, then sign='*', num=7
    Correction: push 2 first, then '*' pops 2 and pushes 2*7=14
    Hit '-' --> sign='-', num=5
    Hit '$' --> push -5
    Stack: [14, -5] --> sum = 9
```

## Recursive Descent Calculator

Handles `+`, `-`, `*`, `/` with parentheses. Appends `$` as an end sentinel.

```csharp
int Calculate(string s)
{
    s += "$";   // Sentinel to flush the last number

    (long sum, int i) Eval(int i)
    {
        var stack = new Stack<long>();
        char sign = '+';
        long num = 0;

        while (i < s.Length)
        {
            char ch = s[i];

            if (ch == ' ')                          // Skip whitespace
            {
                i++;
                continue;
            }

            if (char.IsDigit(ch))                   // Build multi-digit number
            {
                num = num * 10 + (ch - '0');
                i++;
                continue;
            }

            if (ch == '(')                          // Recurse into subexpression
            {
                var result = Eval(i + 1);
                num = result.sum;
                i = result.i;
                ch = s[i];
            }

            if (!char.IsDigit(ch))
            {
                // Apply the pending operator
                if (sign == '+') stack.Push(num);
                else if (sign == '-') stack.Push(-num);
                else if (sign == '*') stack.Push(stack.Pop() * num);
                else if (sign == '/') stack.Push(stack.Pop() / num);

                if (ch == ')')                      // End of subexpression
                    return (stack.Sum(x => x), i + 1);

                sign = ch;
                num = 0;
                i++;
            }
        }

        return ((int)stack.Sum(x => x), i);
    }

    return (int)Eval(0).sum;
}
```
