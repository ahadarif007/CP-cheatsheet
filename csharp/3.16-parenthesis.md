# Parenthesis

> Parenthesis problems check whether brackets are **balanced** -- every open bracket has a matching close in the correct order. Two approaches: a **counter** for single-type parens (O(1) space), and a **stack** for mixed brackets `(){}[]`. The stack also enables **removal** of invalid parentheses.

## Count-Only Validation (Single Type)

For strings with only `(` and `)`, track a running count. Increment for `(`, decrement for `)`. If count ever goes negative, there is an unmatched `)`. Time: **O(n)**, Space: **O(1)**.

```
  "(())"  -> count: 0,1,2,1,0 -> valid (ends at 0, never negative)
  "())("  -> count: 0,1,0,-1  -> invalid (went negative at index 2)
```

```csharp
bool IsValidParentheses(string s)
{
    int openCount = 0;

    foreach (var ch in s)
    {
        if (ch == '(')
            openCount++;
        else if (ch == ')')
        {
            openCount--;
            if (openCount < 0)
                return false; // More closing than opening
        }
    }

    return openCount == 0; // All opened brackets must be closed
}
```

## Stack-Based Validation (Multiple Bracket Types)

Handle `()`, `{}`, `[]` by pushing open brackets onto a stack and popping when a matching close bracket arrives. Time: **O(n)**, Space: **O(n)**.

```
  "{[()]}"  ->  stack: {, [, (  then match ), ], }  -> valid
  "{[(])}"  ->  stack: {, [, (  then ) matches (, ] doesn't match {  -> invalid
```

```csharp
bool IsValid(string s)
{
    var stack = new Stack<char>();
    var matchingBracket = new Dictionary<char, char>
    {
        { ')', '(' },
        { '}', '{' },
        { ']', '[' }
    };

    foreach (var ch in s)
    {
        if (matchingBracket.ContainsValue(ch))
        {
            stack.Push(ch); // Open bracket: push onto stack
        }
        else if (matchingBracket.ContainsKey(ch))
        {
            // Close bracket: must match top of stack
            if (stack.Count == 0 || stack.Pop() != matchingBracket[ch])
                return false;
        }
    }

    return stack.Count == 0; // Stack must be empty for valid input
}
```

## Remove Minimum Parentheses to Make Valid

Mark indices of unmatched parentheses for deletion, then rebuild the string without them. Time: **O(n)**, Space: **O(n)**.

```
  Input:  "lee(t(c)o)de)"

  Pass 1 - find unmatched:
    index 3:  '(' -> push 3
    index 5:  '(' -> push 5
    index 7:  ')' -> matches, pop 5
    index 9:  ')' -> matches, pop 3
    index 12: ')' -> no match, mark for deletion

  Stack still has: []  (empty)
  Delete set: {12}
  Result: "lee(t(c)o)de"
```

```csharp
string MinRemoveToMakeValid(string s)
{
    var markedForDeletion = new bool[s.Length];
    var stack = new Stack<int>(); // Stores indices of unmatched '('

    for (int i = 0; i < s.Length; i++)
    {
        char ch = s[i];

        if (ch == '(')
        {
            stack.Push(i); // Track index of open paren
        }
        else if (ch == ')')
        {
            if (stack.Count > 0)
                stack.Pop();             // Matched with an open paren
            else
                markedForDeletion[i] = true; // Unmatched close paren
        }
    }

    // Any remaining open parens in stack are also unmatched
    while (stack.Count > 0)
        markedForDeletion[stack.Pop()] = true;

    // Build result string, skipping deleted indices
    var builder = new StringBuilder();
    for (int i = 0; i < s.Length; i++)
    {
        if (!markedForDeletion[i])
            builder.Append(s[i]);
    }

    return builder.ToString();
}
```
