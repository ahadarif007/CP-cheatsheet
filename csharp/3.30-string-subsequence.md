# String Subsequence

> Find the minimum number of copies of `source` needed to form `target` as a subsequence.
> Precompute character positions in source, then greedily match target characters using
> binary search. When no valid position remains, start a new copy. Time: O(T log S), Space: O(S).

```
  Example: source = "abc", target = "abcbc"

  Position map: a->[0], b->[1], c->[2]

  Match target characters (idx = next required position in source):
    'a': idx=-1, search pos[a] for >=0 --> found 0, idx=1
    'b': idx=1,  search pos[b] for >=1 --> found 1, idx=2
    'c': idx=2,  search pos[c] for >=2 --> found 2, idx=3
    'b': idx=3,  search pos[b] for >=3 --> not found! New copy (copies=2), idx=pos[b][0]+1=2
    'c': idx=2,  search pos[c] for >=2 --> found 2, idx=3

  Answer: 2 copies of "abc" needed
```

## Shortest Way to Form String

```csharp
int ShortestWay(string source, string target)
{
    // Precompute positions for each character in source
    var positions = new Dictionary<char, List<int>>();
    for (int i = 0; i < source.Length; i++)
    {
        char ch = source[i];
        if (!positions.ContainsKey(ch))
            positions[ch] = new List<int>();
        positions[ch].Add(i);
    }

    int copies = 1;
    int sourceIndex = -1;   // Next required position in source (exclusive lower bound)

    foreach (var ch in target)
    {
        if (!positions.ContainsKey(ch))
            return -1;              // Character not in source at all

        var posList = positions[ch];
        int j = posList.BinarySearch(sourceIndex);
        j = j < 0 ? ~j : j;       // Find first position >= sourceIndex

        if (j == posList.Count)
        {
            // No valid position left in current copy; start a new one
            copies++;
            sourceIndex = posList[0] + 1;
        }
        else
        {
            sourceIndex = posList[j] + 1;
        }
    }

    return copies;
}
```
