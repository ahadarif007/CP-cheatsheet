# Graph

> Graphs are represented as adjacency lists -- an array/list where `graph[u]` holds all neighbors of node `u`.
> DFS uses recursion (or a stack); BFS uses a queue and a visited set.

## Adjacency List Representation

```
Nodes: 0, 1, 2, 3
Edges: (0,1), (0,2), (1,3), (2,3)

Adjacency list (undirected):
  0 -> [1, 2]
  1 -> [0, 3]
  2 -> [0, 3]
  3 -> [1, 2]
```

```csharp
// Build adjacency list for N nodes
var graph = Enumerable.Range(0, N)
    .Select(_ => new List<int>())
    .ToArray();

// Add undirected edges
foreach (var edge in edges)
{
    graph[edge[0]].Add(edge[1]);
    graph[edge[1]].Add(edge[0]);
}
```

## Depth-First Search (DFS) -- Recursive

```
DFS from node 0 (tree-like, parent tracking):

Visit 0 -> Visit 1 -> Visit 3 (backtrack)
                                -> Visit 2 (backtrack)
```

```csharp
void Dfs(int node, int parent)
{
    // Process node here
    foreach (var neighbor in graph[node])
    {
        if (neighbor != parent)  // avoid revisiting parent
            Dfs(neighbor, node);
    }
}
```

## Breadth-First Search (BFS)

```
BFS from node 0:

Queue: [0]          Visit 0, enqueue 1, 2
Queue: [1, 2]       Visit 1, enqueue 3
Queue: [2, 3]       Visit 2 (3 already visited)
Queue: [3]          Visit 3

Order: [0, 1, 2, 3]
```

```csharp
List<int> Bfs(int start)
{
    var order   = new List<int>();
    var visited = new bool[N];
    var queue   = new Queue<int>();

    queue.Enqueue(start);
    visited[start] = true;

    while (queue.Count > 0)
    {
        var node = queue.Dequeue();
        order.Add(node);

        foreach (var neighbor in graph[node])
        {
            if (!visited[neighbor])
            {
                visited[neighbor] = true;
                queue.Enqueue(neighbor);
            }
        }
    }

    return order;
}
```
