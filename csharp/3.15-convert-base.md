# Convert Base

> Base conversion extracts digits by repeated **modulo** and **division**. To convert an integer to base B: take `num % B` for the least-significant digit, then `num /= B`, and repeat. Reverse the collected digits at the end. For negative numbers in hex, use unsigned interpretation (two's complement).

```
  Convert 255 to hexadecimal:

  Step 1:  255 % 16 = 15 -> 'f',  255 / 16 = 15
  Step 2:   15 % 16 = 15 -> 'f',   15 / 16 = 0
  Digits collected: ['f', 'f']
  Reverse: "ff"

  Convert -1 to hexadecimal (32-bit two's complement):
  -1 as uint32 = 4294967295 = 0xFFFFFFFF
  Result: "ffffffff"
```

## Convert Integer to Hexadecimal

Handles negative numbers via unsigned cast (two's complement). Time: **O(log n)**, Space: **O(log n)**.

```csharp
string ToHex(int num)
{
    if (num == 0) return "0";

    // Cast to uint to handle negative numbers (two's complement)
    uint unsigned = (uint)num;
    const string hexDigits = "0123456789abcdef";
    var builder = new StringBuilder();

    while (unsigned > 0)
    {
        int digit = (int)(unsigned % 16);  // Extract lowest hex digit
        builder.Append(hexDigits[digit]);
        unsigned /= 16;                    // Shift right by one hex digit
    }

    // Digits were collected in reverse order
    var chars = builder.ToString().ToCharArray();
    Array.Reverse(chars);
    return new string(chars);
}
```
