# Any

> `Any()` returns `true` if at least one element in the sequence satisfies the predicate.
> Short-circuits on the first match -- does not scan the entire collection.

## Basic Usage

```csharp
bool hasPositive = nums.Any(x => x > 0);   // true if any element > 0
```

## Without a Predicate

```csharp
// Check if collection is non-empty
bool isNonEmpty = items.Any();   // true if at least one element exists
```

## Practical Examples

```csharp
// Check if any string contains a substring
bool hasMatch = words.Any(word => word.Contains("error"));

// Check if any value is null
bool hasNull = values.Any(value => value == null);
```
