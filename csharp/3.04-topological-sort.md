# Topological Sort

> Topological sort orders vertices of a **directed acyclic graph (DAG)** so that for every edge u->v, u comes before v. Uses **Kahn's algorithm** (BFS with in-degree tracking). If the sort cannot process all nodes, the graph has a cycle. Time: **O(V + E)**.

```
  Prerequisites: [[1,0], [2,0], [3,1], [3,2]]
  Meaning: 0->1, 0->2, 1->3, 2->3

  Graph:
    0 --> 1 --> 3
    |         ^
    +-> 2 ---+

  In-degrees: [0:0, 1:1, 2:1, 3:2]
  Start with in-degree 0:  queue = [0]
  Process 0 -> decrement 1,2  ->  queue = [1, 2]
  Process 1 -> decrement 3    ->  queue = [2]
  Process 2 -> decrement 3    ->  queue = [3]
  Process 3                   ->  queue = []
  Seen = 4 == n  ->  can finish!
```

## Course Schedule (Cycle Detection)

Determine if all courses can be finished given prerequisite pairs. Returns `true` if no cycle exists.

```csharp
bool CanFinish(int numCourses, int[][] prerequisites)
{
    // Build adjacency list and in-degree array
    var adjacencyList = Enumerable.Range(0, numCourses)
        .Select(_ => new List<int>())
        .ToArray();
    var inDegree = new int[numCourses];

    foreach (var edge in prerequisites)
    {
        adjacencyList[edge[1]].Add(edge[0]); // edge[1] -> edge[0]
        inDegree[edge[0]]++;
    }

    // Start BFS with all zero-in-degree nodes
    var queue = new Queue<int>();
    for (int i = 0; i < numCourses; i++)
    {
        if (inDegree[i] == 0)
            queue.Enqueue(i);
    }

    int processedCount = 0;

    while (queue.Count > 0)
    {
        var course = queue.Dequeue();
        processedCount++;

        foreach (var neighbor in adjacencyList[course])
        {
            // Decrement in-degree; enqueue if it reaches 0
            if (--inDegree[neighbor] == 0)
                queue.Enqueue(neighbor);
        }
    }

    // If we processed all courses, no cycle exists
    return processedCount == numCourses;
}
```
