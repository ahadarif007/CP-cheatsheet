# Binary Search

> Binary search finds a target in **O(log n)** by halving the search space each step. The key is identifying what condition splits the space into two halves. Works on sorted arrays and on any monotonic predicate.

```
  Array: [1, 3, 5, 7, 9, 11]    target = 7

  Step 1:  lo=0, hi=5, mid=2  ->  arr[2]=5 < 7  ->  lo=3
  Step 2:  lo=3, hi=5, mid=4  ->  arr[4]=9 > 7  ->  hi=3
  Step 3:  lo=3, hi=3, mid=3  ->  arr[3]=7 = 7  ->  found!
```

## First Bad Version

Given n versions `[1..n]`, find the **first bad** one. Classic binary search on a boolean predicate. Time: **O(log n)**.

```csharp
int FirstBadVersion(int n, Func<int, bool> IsBad)
{
    int lo = 1, hi = n;

    while (lo < hi)
    {
        int mid = lo + (hi - lo) / 2;  // Avoid overflow
        if (IsBad(mid))
            hi = mid;       // Bad: answer is mid or earlier
        else
            lo = mid + 1;   // Good: answer is after mid
    }

    return lo; // lo == hi == first bad version
}
```

## Integer Square Root

Find the largest integer whose square is <= x. Time: **O(log x)**.

```
  x = 8
  Step 1:  lo=1, hi=8, mid=4  ->  4*4=16 > 8  ->  hi=4
  Step 2:  lo=1, hi=4, mid=2  ->  2*2=4 <= 8   ->  lo=3
  Step 3:  lo=3, hi=4, mid=3  ->  3*3=9 > 8    ->  hi=3
  Step 4:  lo=3, hi=3  ->  exit loop
  Answer:  lo - 1 = 2
```

```csharp
int MySqrt(int x)
{
    if (x <= 1) return x;

    int lo = 1, hi = x;

    while (lo < hi)
    {
        int mid = lo + (hi - lo) / 2;

        if ((long)mid * mid > x)
            hi = mid;       // mid is too large
        else
            lo = mid + 1;   // mid might be the answer
    }

    return lo - 1; // lo overshoots by 1
}
```
