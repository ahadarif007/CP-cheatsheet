# Dynamic Programming

> DP solves problems by breaking them into **overlapping subproblems** and storing results to avoid recomputation. Two approaches: **top-down** (recursion + memoization) and **bottom-up** (iterative table filling). Key: define the state, the transition, and the base case.

## Coin Change (Unbounded Knapsack)

Find the **minimum number of coins** to make a given amount. Each coin can be used unlimited times. Time: **O(amount * coins)**, Space: **O(amount)**.

```
  coins = [1, 3, 4], amount = 6
  dp[i] = minimum coins to make amount i

  dp: [0, 1, 2, 1, 1, 2, 2]
       ^              ^   ^
       |              |   +-- dp[6] = min(dp[5]+1, dp[3]+1, dp[2]+1) = 2
       |              +-- dp[4] = 1 (use coin 4)
       +-- base case

  Answer: dp[6] = 2  (coins: 3 + 3)
```

```csharp
int CoinChange(int[] coins, int amount)
{
    int INFINITY = 1_000_000_000;
    var minCoins = Enumerable.Repeat(INFINITY, amount + 1).ToArray();
    minCoins[0] = 0; // Base case: 0 coins for amount 0

    foreach (var coin in coins)
    {
        for (int currentAmount = coin; currentAmount <= amount; currentAmount++)
        {
            // Either skip this coin, or use it (add 1 to dp[currentAmount - coin])
            minCoins[currentAmount] = Math.Min(
                minCoins[currentAmount],
                minCoins[currentAmount - coin] + 1
            );
        }
    }

    return minCoins[amount] >= INFINITY ? -1 : minCoins[amount];
}
```

## Longest Common Subsequence (LCS)

Find the length of the longest subsequence common to two strings. Time: **O(n * m)**, Space: **O(n * m)**.

```
  a = "abcde", b = "ace"

  DP table (dp[i][j] = LCS of a[0..i-1] and b[0..j-1]):

        ""  a  c  e
    ""   0  0  0  0
    a    0  1  1  1    a==a -> dp[0][0]+1 = 1
    b    0  1  1  1    b!=a,c,e -> carry max
    c    0  1  2  2    c==c -> dp[1][1]+1 = 2
    d    0  1  2  2    d!=a,c,e -> carry max
    e    0  1  2  3    e==e -> dp[3][2]+1 = 3

  Answer: 3  (subsequence "ace")
```

```csharp
int LongestCommonSubsequence(string a, string b)
{
    int rows = a.Length, cols = b.Length;
    var dp = new int[rows + 1, cols + 1];

    for (int i = 0; i < rows; i++)
    {
        for (int j = 0; j < cols; j++)
        {
            if (a[i] == b[j])
                dp[i + 1, j + 1] = dp[i, j] + 1;        // Characters match
            else
                dp[i + 1, j + 1] = Math.Max(
                    dp[i, j + 1],   // Skip char from a
                    dp[i + 1, j]    // Skip char from b
                );
        }
    }

    return dp[rows, cols];
}
```
