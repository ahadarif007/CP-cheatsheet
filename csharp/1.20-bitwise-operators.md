# Bitwise Operators

> Bitwise operators work on individual bits of integer values.
> Essential for bit manipulation problems, bitmasks, and low-level optimizations.

## Operator Reference

```
Operator   Name    Example            Binary                Result
--------   ----    -------            ------                ------
  &        AND     0b1100 & 0b1010    1100 & 1010  =  1000    8
  |        OR      0b1100 | 0b1010    1100 | 1010  =  1110   14
  ^        XOR     0b1100 ^ 0b1010    1100 ^ 1010  =  0110    6
  >>       SHR     0b1100 >> 2        1100 >> 2    =  0011    3
  <<       SHL     0b0011 << 2        0011 << 2    =  1100   12
  ~        NOT     ~0b1100            ~1100        =  0011   (inverts all bits)
```

## Basic Operations

```csharp
int andResult = 0b1100 & 0b1010;    // 0b1000 = 8   (bits both set)
int orResult  = 0b1100 | 0b1010;    // 0b1110 = 14  (bits either set)
int xorResult = 0b1100 ^ 0b1010;    // 0b0110 = 6   (bits differ)
int shrResult = 0b1100 >> 2;        // 0b0011 = 3   (shift right by 2)
int shlResult = 0b0011 << 2;        // 0b1100 = 12  (shift left by 2)
```

## Unsigned Right Shift

```csharp
// Unsigned shift fills with zeros (no sign extension)
uint unsignedShift = uint.MaxValue >> 1;   // 0x7FFFFFFF
```

## Common Bit Tricks

```csharp
// Check if n-th bit is set
bool isSet = (value & (1 << n)) != 0;

// Set the n-th bit
value |= (1 << n);

// Clear the n-th bit
value &= ~(1 << n);

// Toggle the n-th bit
value ^= (1 << n);

// Check if power of two
bool isPowerOfTwo = value > 0 && (value & (value - 1)) == 0;

// Count set bits
int bitCount = System.Numerics.BitOperations.PopCount((uint)value);
```
