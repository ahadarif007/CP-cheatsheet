# Kadane's Algorithm

> Find the contiguous subarray with the largest sum. At each index, decide whether to
> extend the previous subarray or start fresh. If the running sum before this element
> is positive, extending helps; otherwise start a new subarray here. Time: O(n), Space: O(1).

```
  Example: nums = [-2, 1, -3, 4, -1, 2, 1, -5, 4]

  Index 0: nums[0]=-2, best=-2
  Index 1: nums[0]=-2 < 0, skip.  nums[1]=1,  best=1
  Index 2: nums[1]=1 > 0, add.    nums[2]=-2, best=1
  Index 3: nums[2]=-2 < 0, skip.  nums[3]=4,  best=4
  Index 4: nums[3]=4 > 0, add.    nums[4]=3,  best=4
  Index 5: nums[4]=3 > 0, add.    nums[5]=5,  best=5
  Index 6: nums[5]=5 > 0, add.    nums[6]=6,  best=6
  Index 7: nums[6]=6 > 0, add.    nums[7]=1,  best=6
  Index 8: nums[7]=1 > 0, add.    nums[8]=5,  best=6

  Answer: 6 (subarray [4, -1, 2, 1])
```

## Maximum Subarray Sum

```csharp
int MaxSubArray(int[] nums)
{
    int best = nums[0];

    for (int i = 1; i < nums.Length; i++)
    {
        // If previous running sum is positive, extend the subarray
        if (nums[i - 1] > 0)
            nums[i] += nums[i - 1];

        best = Math.Max(best, nums[i]);
    }

    return best;
}
```
