# Strings

> C# strings are immutable reference types. Use `StringBuilder` for efficient concatenation.
> Character inspection methods live on the `char` struct (`char.IsUpper`, `char.IsDigit`, etc.).

## Searching

```csharp
// Find first / last occurrence of a character
int firstIndex = text.IndexOf('x');       // -1 if not found
int lastIndex  = text.LastIndexOf('x');   // -1 if not found
```

## Splitting and Joining

```csharp
// Split a line by delimiter
string[] tokens = line.Split(':');

// Join a collection with a separator
string joined = string.Join(" ", words);
```

## StringBuilder (Mutable String Building)

```csharp
var builder = new StringBuilder();
builder.Append("a");    // append a string
builder.Append(1);      // append an int (auto-converted)
string result = builder.ToString();
```

## Character Inspection

```csharp
char first = text[0];

bool isUpper = char.IsUpper(first);   // 'A'-'Z'
bool isLower = char.IsLower(first);   // 'a'-'z'
bool isDigit = char.IsDigit(first);   // '0'-'9'
```

## Replace, StartsWith, EndsWith

```csharp
string cleaned = text.Replace("()", "");                // remove all "()"
bool hasPrefix = text.StartsWith("pre");                // prefix check
bool hasSuffix = text.EndsWith("suf");                  // suffix check
```

## String Interpolation

```csharp
string formatted = $"My name is {name}, I'm {age}";
```

## Manual Split by Letter Characters

Walks through the string, collecting runs of letters into a list.

```
Input:  "hello--world!!foo"
         ^^^^^  ^^^^^  ^^^
         word1  word2  word3

Output: ["hello", "world", "foo"]
```

```csharp
List<string> SplitWords(string input)
{
    var result = new List<string>();
    int start = -1;

    for (int i = 0; i < input.Length; i++)
    {
        char character = input[i];

        if (char.IsLetter(character))
        {
            // Mark start of a new word
            if (start == -1) start = i;
        }
        else
        {
            // End of a word -- extract and reset
            if (start != -1)
            {
                result.Add(input.Substring(start, i - start));
                start = -1;
            }
        }
    }

    // Capture trailing word if string ends with letters
    if (start != -1)
        result.Add(input.Substring(start));

    return result;
}
```
