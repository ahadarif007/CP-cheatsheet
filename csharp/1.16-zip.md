# Zip

> `Zip` pairs elements from two sequences by position, stopping at the shorter one.
> Useful for iterating two arrays in parallel without manual index management.

## Basic Zip with Deconstruction

```
A: [1, 2, 3]
B: [a, b, c]

Zip: (1,a), (2,b), (3,c)
```

```csharp
int[] numbers = { 1, 2, 3 };
string[] letters = { "a", "b", "c" };

foreach (var (number, letter) in numbers.Zip(letters))
{
    // number=1, letter="a"
    // number=2, letter="b"
    // number=3, letter="c"
}
```

## Zip with a Projection

```csharp
// Combine two arrays into a single result
var sums = numbersA.Zip(numbersB, (a, b) => a + b).ToArray();
```

## Zip with Index (Enumerate)

```csharp
// Pair each element with its index
foreach (var (value, index) in items.Zip(Enumerable.Range(0, items.Count)))
{
    // process value at index
}

// Or more idiomatically with Select
foreach (var (value, index) in items.Select((v, i) => (v, i)))
{
    // process value at index
}
```
