# Grids

> 2D grids are common in BFS/DFS problems (islands, mazes, flood fill).
> Use a direction array to iterate over the 4 (or 8) neighbors of a cell.

## Grid Structure and Directions

```
       col 0   col 1   col 2   col 3
row 0 [  1  ] [  1  ] [  0  ] [  0  ]
row 1 [  1  ] [  1  ] [  0  ] [  1  ]
row 2 [  0  ] [  0  ] [  1  ] [  1  ]

4-directional neighbors of (1,1):
         (0,1)            up    (-1, 0)
          |
  (1,0)--[1,1]--(1,2)    left  ( 0,-1)   right (0,+1)
          |
         (2,1)            down  (+1, 0)
```

## Grid DFS (Flood Fill / Island Counting)

```csharp
int rows = grid.Count;
int cols = grid[0].Count;

// 4-directional movement: down, up, right, left
int[][] directions = new[]
{
    new[] { 1, 0 },    // down
    new[] { -1, 0 },   // up
    new[] { 0, 1 },    // right
    new[] { 0, -1 }    // left
};

void Dfs(int row, int col)
{
    // Mark current cell as visited
    grid[row][col] = 2;

    foreach (var direction in directions)
    {
        int newRow = row + direction[0];
        int newCol = col + direction[1];

        // Check bounds and whether the neighbor is unvisited land
        bool inBounds = newRow >= 0 && newRow < rows
                     && newCol >= 0 && newCol < cols;

        if (inBounds && grid[newRow][newCol] == 1)
        {
            Dfs(newRow, newCol);
        }
    }
}
```

## Grid BFS

```csharp
void Bfs(int startRow, int startCol)
{
    var queue = new Queue<(int row, int col)>();
    queue.Enqueue((startRow, startCol));
    grid[startRow][startCol] = 2;   // mark visited

    while (queue.Count > 0)
    {
        var (row, col) = queue.Dequeue();

        foreach (var direction in directions)
        {
            int newRow = row + direction[0];
            int newCol = col + direction[1];

            bool inBounds = newRow >= 0 && newRow < rows
                         && newCol >= 0 && newCol < cols;

            if (inBounds && grid[newRow][newCol] == 1)
            {
                grid[newRow][newCol] = 2;   // mark visited before enqueue
                queue.Enqueue((newRow, newCol));
            }
        }
    }
}
```

## 8-Directional Movement

```csharp
// Include diagonals
int[][] eightDirections = new[]
{
    new[] { 1, 0 }, new[] { -1, 0 },   // vertical
    new[] { 0, 1 }, new[] { 0, -1 },   // horizontal
    new[] { 1, 1 }, new[] { 1, -1 },   // diagonals
    new[] { -1, 1 }, new[] { -1, -1 }
};
```
