# Bisect

> Binary search finds a target or insertion point in a sorted array in O(log n).
> `Array.BinarySearch` is built-in; custom lower/upper bound functions handle edge cases.

## Built-In Binary Search

```csharp
// Returns index of target, or a negative value if not found
// The negative value is the bitwise complement (~) of the insertion point
int index = Array.BinarySearch(sortedArray, target);

if (index >= 0)
{
    // Found at position index
}
else
{
    int insertionPoint = ~index;   // where to insert to keep sorted
}
```

## Lower Bound (First Index >= Target)

Finds the leftmost position where `target` could be inserted.

```
Array:  [1, 3, 3, 3, 5, 7]
Target: 3
         ^
LowerBound = 1  (first index where value >= 3)
```

```csharp
int LowerBound(int[] array, int target)
{
    int left = 0;
    int right = array.Length;

    while (left < right)
    {
        int mid = (left + right) / 2;

        if (array[mid] < target)
            left = mid + 1;    // search right half
        else
            right = mid;       // search left half (mid could be answer)
    }

    return left;   // first index where array[index] >= target
}
```

## Upper Bound (First Index > Target)

Finds the leftmost position where a value strictly greater than `target` begins.

```
Array:  [1, 3, 3, 3, 5, 7]
Target: 3
                     ^
UpperBound = 4  (first index where value > 3)
```

```csharp
int UpperBound(int[] array, int target)
{
    int left = 0;
    int right = array.Length;

    while (left < right)
    {
        int mid = (left + right) / 2;

        if (array[mid] <= target)
            left = mid + 1;    // search right half
        else
            right = mid;       // search left half (mid could be answer)
    }

    return left;   // first index where array[index] > target
}
```

## Count Occurrences of a Value

```csharp
// Number of times target appears = UpperBound - LowerBound
int count = UpperBound(array, target) - LowerBound(array, target);
```
