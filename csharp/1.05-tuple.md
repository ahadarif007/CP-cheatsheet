# Tuple

> C# value tuples are lightweight structs for grouping related values without defining a class.
> Fields can be named for readability or accessed positionally as `Item1`, `Item2`, etc.

## Creating a Named Tuple

```csharp
// Named tuple -- access via .first and .second
(int first, int second) pair = (2, 3);
int sum = pair.first + pair.second;    // 5
```

## Unnamed Tuple

```csharp
// Unnamed -- access via .Item1, .Item2
var point = (10, 20);
int x = point.Item1;   // 10
int y = point.Item2;   // 20
```

## Tuple Deconstruction

```csharp
// Deconstruct into separate variables
var (width, height) = (1920, 1080);
```

## Returning Multiple Values from a Method

```csharp
(int min, int max) FindRange(int[] nums)
{
    int minVal = nums.Min();
    int maxVal = nums.Max();
    return (minVal, maxVal);
}

// Caller can deconstruct
var (lo, hi) = FindRange(new[] { 3, 1, 4, 1, 5 });
```

## Tuple as Dictionary Key

```csharp
// Tuples implement equality, so they work as dictionary keys
var memo = new Dictionary<(int, int), int>();
memo[(2, 3)] = 5;
```
