# Continuous Subarrays with Sum k

> Count subarrays whose elements sum to exactly k using a prefix-sum map.
> For each index, if `prefixSum - k` was seen before, those occurrences mark subarray start points.
> Time: O(n), Space: O(n).

```
  Example: nums = [1, 1, 1], k = 2

  Index 0: prefixSum=1, check map for 1-2=-1 (not found)
           map: {0:1, 1:1}

  Index 1: prefixSum=2, check map for 2-2=0 (found, count=1)
           result=1, map: {0:1, 1:1, 2:1}

  Index 2: prefixSum=3, check map for 3-2=1 (found, count=1)
           result=2, map: {0:1, 1:1, 2:1, 3:1}

  Answer: 2 (subarrays [1,1] at indices 0-1 and 1-2)
```

## Prefix Sum with Hash Map

```csharp
int SubarraySum(int[] nums, int k)
{
    // Map from prefix sum to its occurrence count; base case: empty prefix = 0
    var prefixCounts = new Dictionary<int, int> { { 0, 1 } };
    int result = 0;
    int prefixSum = 0;

    foreach (var num in nums)
    {
        prefixSum += num;

        // If (prefixSum - k) was a previous prefix sum, we found subarrays summing to k
        if (prefixCounts.TryGetValue(prefixSum - k, out var count))
            result += count;

        // Record current prefix sum
        prefixCounts[prefixSum] = prefixCounts.TryGetValue(prefixSum, out var existing)
            ? existing + 1
            : 1;
    }

    return result;
}
```
