# Reverse Polish Notation

> Evaluate an expression in Reverse Polish Notation (postfix). Numbers are pushed onto
> a stack; when an operator is encountered, pop two operands, apply the operator, and
> push the result. Time: O(n), Space: O(n).

```
  Example: tokens = ["2", "1", "+", "3", "*"]

  Step 1: push 2          stack: [2]
  Step 2: push 1          stack: [2, 1]
  Step 3: "+" -> pop 1,2  stack: [3]       (2 + 1 = 3)
  Step 4: push 3          stack: [3, 3]
  Step 5: "*" -> pop 3,3  stack: [9]       (3 * 3 = 9)

  Answer: 9

  Note: second popped value is the LEFT operand (b op a).
```

## Evaluate RPN Expression

```csharp
int EvalRPN(IList<string> tokens)
{
    var stack = new Stack<int>();

    foreach (var token in tokens)
    {
        if (!"+-/*".Contains(token))
        {
            stack.Push(int.Parse(token));    // Push number
        }
        else
        {
            int right = stack.Pop();         // Right operand (popped first)
            int left = stack.Pop();          // Left operand

            stack.Push(token switch
            {
                "+" => left + right,
                "-" => left - right,
                "*" => left * right,
                _   => left / right          // Integer division
            });
        }
    }

    return stack.Pop();
}
```
