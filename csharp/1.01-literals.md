# Literals

> C# is statically typed -- every variable must declare its type (or use `var` for type inference).
> Value types (`int`, `long`, `double`, `bool`, `char`) are stored on the stack.
> Reference types (`string`, `object`, arrays, collections) are stored on the heap.

## Numeric Types

```csharp
// Integer literals in different bases
int decimal    = 255;           // base 10
int binary     = 0b11111111;    // base 2 (prefix 0b)
int hex        = 0xFF;          // base 16 (prefix 0x)

// Larger / precise numeric types
long bigNumber      = 1234567890123L;   // 64-bit integer (suffix L)
double floatingPt   = 1.23;            // 64-bit floating point
decimal precise     = 1.23m;           // 128-bit decimal (suffix m)
```

## Boolean, Character, and String

```csharp
bool isReady    = true;
char letter     = 'a';              // single character (single quotes)
string greeting = "hello\n";        // string with escape sequence
object empty    = null;             // null reference
```

## Common Collection Initializers

```
Type                        Ordered?    Duplicates?    Lookup
---------------------------------------------------------------
int[]                       yes         yes            O(1) index
List<T>                     yes         yes            O(1) index
Dictionary<K,V>             no          keys unique    O(1) key
HashSet<T>                  no          no             O(1) contains
```

```csharp
// Array -- fixed size
int[] numbers = new[] { 1, 2, 3 };

// List -- dynamic array
var items = new List<int>();

// Dictionary -- key-value store
var lookup = new Dictionary<string, int>();

// HashSet -- unique elements
var seen = new HashSet<char>();
```
