# Lambda

> Lambda expressions (`=>`) define inline anonymous functions.
> Combined with LINQ, they enable concise functional-style operations on collections.

## Lambda Syntax

```csharp
// Single parameter
Func<int, int> square = x => x * x;

// Multiple parameters
Func<int, int, int> add = (a, b) => a + b;

// Statement body (multi-line)
Func<int, bool> isEvenPositive = x =>
{
    bool even = x % 2 == 0;
    return even && x > 0;
};
```

## Find Closest Value to a Target

Uses `OrderBy` with a lambda to sort by distance to the target.

```csharp
// Find the value in the array closest to target
int closest = new[] { valueA, valueB }
    .OrderBy(x => Math.Abs(target - x))
    .First();
```

## Common LINQ + Lambda Patterns

```csharp
int[] numbers = { 5, 3, 8, 1, 9 };

int max    = numbers.Max();                          // 9
int minVal = numbers.Min();                          // 1
int first  = numbers.First(x => x > 4);             // 5
int count  = numbers.Count(x => x % 2 == 0);        // 1 (only 8)
bool exists = numbers.Any(x => x > 8);              // true
```
