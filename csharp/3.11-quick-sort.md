# Quick Sort

> **Quick sort** picks a pivot, partitions the array so all elements less than the pivot come before it and all greater come after, then recurses on each side. Average time: **O(n log n)**, worst case: **O(n^2)** (rare with good pivot choice). Sorts **in-place** with O(log n) stack space.

```
  Array: [3, 6, 8, 10, 1, 2, 1]    pivot = 1 (last element)

  Partition step (Lomuto scheme):
    Compare each element to pivot=1:
    [3, 6, 8, 10, 1, 2, 1]
                         ^pivot
    storeIndex=0
    3>1, 6>1, 8>1, 10>1 -> skip
    1<=1 -> swap arr[4] with arr[0] -> [1, 6, 8, 10, 3, 2, 1], storeIndex=1
    2>1 -> skip
    Place pivot: swap arr[6] with arr[1] -> [1, 1, 8, 10, 3, 2, 6]
    Pivot index = 1

  Recurse on [1] and [8, 10, 3, 2, 6]
```

## Quick Sort Implementation

Uses **Lomuto partition** scheme with the last element as pivot.

```csharp
void QuickSort(int[] arr)
{
    Sort(arr, 0, arr.Length - 1);
}

void Sort(int[] arr, int left, int right)
{
    if (left >= right) return;

    int pivotIndex = Partition(arr, left, right);
    Sort(arr, left, pivotIndex - 1);   // Sort left half
    Sort(arr, pivotIndex + 1, right);  // Sort right half
}

int Partition(int[] arr, int left, int right)
{
    int pivot = arr[right];    // Choose last element as pivot
    int storeIndex = left;     // Where the next smaller element goes

    for (int k = left; k < right; k++)
    {
        if (arr[k] < pivot)
        {
            (arr[k], arr[storeIndex]) = (arr[storeIndex], arr[k]);
            storeIndex++;
        }
    }

    // Place pivot in its final position
    (arr[right], arr[storeIndex]) = (arr[storeIndex], arr[right]);
    return storeIndex;
}
```
