# Loops

```csharp
for (int i = 0; i < n; i++) {}
for (int i = n - 1; i >= 0; i--) {}
foreach (var x in nums) {}
int i = 0; while (i < n) { i++; }
```

Reverse in-place

```csharp
int l = 0, r = nums.Length - 1;
while (l < r) { (nums[l], nums[r]) = (nums[r], nums[l]); l++; r--; }
```

Ranges

```csharp
for (int i = 0; i < 3; i++) {}
for (int i = 3; i >= 0; i--) {}
```
