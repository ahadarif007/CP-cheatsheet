# Loops

> C# provides `for`, `foreach`, and `while` loops.
> `for` is best when you need the index; `foreach` is cleaner for iterating collections.

## Standard For Loop

```csharp
// Forward iteration: 0, 1, 2, ..., length-1
for (int index = 0; index < nums.Length; index++)
{
    // process nums[index]
}

// Backward iteration: length-1, length-2, ..., 0
for (int index = nums.Length - 1; index >= 0; index--)
{
    // process nums[index]
}
```

## For-Each Loop

```csharp
// Iterate without needing an index
foreach (var value in nums)
{
    // process value
}
```

## While Loop

```csharp
int index = 0;
while (index < nums.Length)
{
    // process nums[index]
    index++;
}
```

## Reverse an Array In-Place (Two-Pointer Swap)

```
Before: [1, 2, 3, 4, 5]
         ^           ^
        left       right    --> swap 1 <-> 5

Step 1: [5, 2, 3, 4, 1]
            ^       ^
           left   right     --> swap 2 <-> 4

Step 2: [5, 4, 3, 2, 1]
               ^
          left == right     --> done (pointers met)
```

```csharp
int left = 0;
int right = nums.Length - 1;

while (left < right)
{
    // Swap elements at left and right pointers
    (nums[left], nums[right]) = (nums[right], nums[left]);
    left++;
    right--;
}
```

## Common Range Patterns

```csharp
// Range [0, 3) -- prints 0, 1, 2
for (int i = 0; i < 3; i++) { }

// Range [3, 0] descending -- prints 3, 2, 1, 0
for (int i = 3; i >= 0; i--) { }
```
