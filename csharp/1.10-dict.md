# Dict

> `Dictionary<TKey, TValue>` is a hash-based key-value store with O(1) average lookup, insert, and delete.
> Keys must be unique. Use `TryGetValue` for safe lookups that avoid exceptions.

## Basic Operations

```csharp
var map = new Dictionary<string, int>();

// Insert or update
map["key"] = 1;

// Safe lookup with default value
int value = map.TryGetValue("key", out var found) ? found : 0;

// Add only if key does not exist
map.TryAdd("newKey", 0);

// Remove a key
map.Remove("key");
```

## Iterating Over a Dictionary

```csharp
// Iterate over keys only
foreach (var key in map.Keys)
{
    // process key
}

// Iterate over key-value pairs with deconstruction
foreach (var (key, value) in map)
{
    // process key and value
}
```

## Check if Key Exists

```csharp
if (map.ContainsKey("target"))
{
    int val = map["target"];
}
```

## Initialize with Values

```csharp
var scores = new Dictionary<string, int>
{
    ["Alice"] = 95,
    ["Bob"]   = 87,
    ["Carol"] = 92
};
```
