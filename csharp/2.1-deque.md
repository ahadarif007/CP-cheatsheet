# Deque

> A **double-ended queue** allows O(1) insertion and removal at both ends. In C#, `LinkedList<T>` serves as the deque — use it for sliding-window problems, BFS with priority layers, or any scenario needing fast access to both front and back.

```
  Deque operations:

  AddFirst(5)   ->  [5]
  AddLast(6)    ->  [5, 6]
  AddFirst(3)   ->  [3, 5, 6]
  RemoveFirst() ->  [5, 6]       returns 3
  RemoveLast()  ->  [5]          returns 6
```

## Basic Deque Operations

All four operations run in **O(1)** time.

```csharp
var deque = new LinkedList<int>();

// Add to both ends
deque.AddFirst(5);   // Push to front
deque.AddLast(6);    // Push to back

// Remove from both ends
deque.RemoveFirst(); // Pop from front
deque.RemoveLast();  // Pop from back

// Peek without removing
int front = deque.First.Value;
int back = deque.Last.Value;

// Check size
int size = deque.Count;
```
