# Modulo

> C#'s `%` operator can return negative results for negative dividends (unlike Python).
> Use a safe modulo function to always get a non-negative remainder.

## The Problem with Negative Modulo

```
C# default:    -7 % 3 = -1   (sign follows dividend)
Python:        -7 % 3 =  2   (always non-negative)
Safe mod:      -7 % 3 =  2   (matches Python behavior)
```

## Safe Modulo Function

```csharp
int SafeMod(int a, int modulus)
{
    int remainder = a % modulus;
    // If remainder is negative, shift it into the positive range
    return remainder < 0 ? remainder + modulus : remainder;
}
```

## Common Use Cases

```csharp
// Circular array indexing (wrap around)
int nextIndex = SafeMod(currentIndex + step, array.Length);

// Modular arithmetic in competitive programming
long result = ((a % MOD) + (b % MOD)) % MOD;
```
