# Tree Tricks

> Common tree patterns that use **DFS with state passing** -- carry min/max values, accumulate results, or return computed depths up the recursion. These patterns avoid global variables by threading state through parameters or return values.

```
  Example tree for both problems below:
          8
        /   \
       3     10
      / \      \
     1   6      14
```

## Maximum Ancestor Difference

Find the maximum `|ancestor.val - descendant.val|` in a binary tree. Pass the **running min and max** down each path. Time: **O(n)**.

```
  Path 8 -> 3 -> 1:  min=1, max=8  ->  diff = 7
  Path 8 -> 3 -> 6:  min=3, max=8  ->  diff = 5
  Path 8 -> 10 -> 14: min=8, max=14 -> diff = 6
  Answer: 7
```

```csharp
int MaxAncestorDiff(TreeNode root)
{
    int result = 0;

    void Dfs(TreeNode node, int minVal, int maxVal)
    {
        if (node == null)
        {
            // At a leaf: compute difference from this path's min/max
            result = Math.Max(result, Math.Abs(maxVal - minVal));
            return;
        }

        // Update running min/max with current node
        Dfs(node.left,
            Math.Min(minVal, node.val),
            Math.Max(maxVal, node.val));
        Dfs(node.right,
            Math.Min(minVal, node.val),
            Math.Max(maxVal, node.val));
    }

    Dfs(root, int.MaxValue, int.MinValue);
    return result;
}
```

## Diameter of Binary Tree

The **diameter** is the longest path between any two nodes (measured in edges). At each node, the longest path *through* that node is `leftDepth + rightDepth`. Time: **O(n)**.

```
  Computing depths bottom-up:
          8           depth=3
        /   \
       3     10       depth=2, depth=1
      / \      \
     1   6      14    depth=1, depth=1, depth=1

  At node 3: left=1, right=1, path_through=2
  At node 8: left=2, right=1, path_through=3  <- diameter
```

```csharp
int DiameterOfBinaryTree(TreeNode root)
{
    int best = 0;

    int Depth(TreeNode node)
    {
        if (node == null) return 0;

        int leftDepth = Depth(node.left);
        int rightDepth = Depth(node.right);

        // Path through this node = left + right depths
        best = Math.Max(best, leftDepth + rightDepth);

        // Return height of this subtree
        return Math.Max(leftDepth, rightDepth) + 1;
    }

    Depth(root);
    return best;
}
```
