# Cyclic Sort

> **Cyclic sort** places each number at its correct index in O(n) time by swapping elements to their "home" positions. Works when the input contains numbers in a known range (e.g., 1 to n). Perfect for finding **missing/duplicate numbers** in a range.

```
  Input:  [3, 1, 5, 4, 2]
  Goal:   each nums[i] should equal i+1

  Step by step:
    i=0: nums[0]=3, should be at index 2  ->  swap(0,2)  ->  [5, 1, 3, 4, 2]
    i=0: nums[0]=5, should be at index 4  ->  swap(0,4)  ->  [2, 1, 3, 4, 5]
    i=0: nums[0]=2, should be at index 1  ->  swap(0,1)  ->  [1, 2, 3, 4, 5]
    i=0: nums[0]=1, correct position      ->  i++
    i=1: nums[1]=2, correct position      ->  i++
    i=2: nums[2]=3, correct position      ->  i++
    i=3: nums[3]=4, correct position      ->  i++
    i=4: nums[4]=5, correct position      ->  i++

  Result: [1, 2, 3, 4, 5]
```

## Cyclic Sort for Range [1..n]

Each element is swapped to `nums[i] - 1` until it is in the right place or a duplicate is detected. Time: **O(n)**, Space: **O(1)**.

```csharp
int[] CyclicSort(int[] nums)
{
    int index = 0;

    while (index < nums.Length)
    {
        int correctPosition = nums[index] - 1; // Where this value belongs

        if (nums[index] != nums[correctPosition])
        {
            // Swap current element to its correct position
            (nums[index], nums[correctPosition]) =
                (nums[correctPosition], nums[index]);
            // Don't increment -- check the swapped-in value
        }
        else
        {
            index++; // Already in place, move on
        }
    }

    return nums;
}
```
