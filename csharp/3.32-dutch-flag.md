# Dutch Flag

> Sort an array of 0s, 1s, and 2s in-place using three pointers. Partition the array into
> three regions: [0..low-1] = 0s, [low..current-1] = 1s, [current..high] = unsorted, [high+1..n-1] = 2s.
> Single pass, Time: O(n), Space: O(1).

```
  Example: nums = [2, 0, 2, 1, 1, 0]

  Initial: low=0, current=0, high=5
           [2, 0, 2, 1, 1, 0]

  current=0, nums[0]=2: swap with high
           [0, 0, 2, 1, 1, 2]  high=4

  current=0, nums[0]=0: swap with low
           [0, 0, 2, 1, 1, 2]  low=1, current=1

  current=1, nums[1]=0: swap with low
           [0, 0, 2, 1, 1, 2]  low=2, current=2

  current=2, nums[2]=2: swap with high
           [0, 0, 1, 1, 2, 2]  high=3

  current=2, nums[2]=1: skip
           current=3

  current=3, nums[3]=1: skip
           current=4

  current=4 > high=3: done!
  Result: [0, 0, 1, 1, 2, 2]
```

## Three-Way Partition (Sort Colors)

```csharp
void SortColors(int[] nums)
{
    int low = 0;                     // Boundary for 0s region
    int current = 0;                 // Current element being examined
    int high = nums.Length - 1;      // Boundary for 2s region

    while (current <= high)
    {
        if (nums[current] == 0)
        {
            // Swap to the 0s region
            (nums[low], nums[current]) = (nums[current], nums[low]);
            low++;
            current++;
        }
        else if (nums[current] == 2)
        {
            // Swap to the 2s region; don't advance current (swapped value unknown)
            (nums[current], nums[high]) = (nums[high], nums[current]);
            high--;
        }
        else
        {
            // It's a 1, already in the right region
            current++;
        }
    }
}
```
