# Fibonacci Golden

> Compute the Nth Fibonacci number in O(1) time using Binet's formula (the golden ratio).
> The golden ratio phi = (1 + sqrt(5)) / 2 gives F(n) = round(phi^n / sqrt(5)).
> Accurate for moderate N; floating-point limits apply for very large values.

```
  Golden ratio: phi = (1 + sqrt(5)) / 2 = 1.6180339...

  Binet's formula: F(n) = round(phi^n / sqrt(5))

  Example calculations:
    F(0) = round(1.0000 / 2.2360) = round(0.4472) = 0
    F(1) = round(1.6180 / 2.2360) = round(0.7236) = 1
    F(5) = round(11.090 / 2.2360) = round(4.9598) = 5
    F(10) = round(122.99 / 2.2360) = round(55.003) = 55
```

## Closed-Form Fibonacci

```csharp
int Fib(int N)
{
    double goldenRatio = (1 + Math.Sqrt(5)) / 2.0;
    return (int)Math.Round(Math.Pow(goldenRatio, N) / Math.Sqrt(5));
}
```
