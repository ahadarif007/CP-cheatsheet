# Graph

> Graphs are represented as **adjacency lists**: each node has a list of its neighbors.
> **DFS** explores as deep as possible before backtracking (uses recursion/stack).
> **BFS** explores all neighbors first before going deeper (uses queue).

## Adjacency List Construction

```
Edges: [[0,1], [0,2], [1,3], [2,3]]
Nodes: 0, 1, 2, 3

Adjacency list (undirected):
  0 → [1, 2]
  1 → [0, 3]
  2 → [0, 3]
  3 → [1, 2]
```

```java
int nodeCount = 4;
int[][] edges = {{0, 1}, {0, 2}, {1, 3}, {2, 3}};

// Build adjacency list
List<List<Integer>> graph = new ArrayList<>();
for (int i = 0; i < nodeCount; i++) {
    graph.add(new ArrayList<>());
}
for (int[] edge : edges) {
    graph.get(edge[0]).add(edge[1]);  // add edge u → v
    graph.get(edge[1]).add(edge[0]);  // add edge v → u (undirected)
}
```

## DFS (Depth-First Search)

```
DFS from node 0:
  Visit 0 → go to neighbor 1
    Visit 1 → go to neighbor 3
      Visit 3 → neighbor 1 (seen), neighbor 2
        Visit 2 → done

Order: [0, 1, 3, 2]
```

```java
// DFS with parent tracking (for trees/undirected graphs without visited array)
void dfs(int current, int parent) {
    for (int neighbor : graph.get(current)) {
        if (neighbor != parent) {
            dfs(neighbor, current);
        }
    }
}
```

## BFS (Breadth-First Search)

```
BFS from node 0:
  Queue: [0]     → visit 0, enqueue neighbors 1, 2
  Queue: [1, 2]  → visit 1, enqueue 3
  Queue: [2, 3]  → visit 2 (3 already queued)
  Queue: [3]     → visit 3

Order: [0, 1, 2, 3]
```

```java
List<Integer> bfs(int startNode) {
    List<Integer> visitOrder = new ArrayList<>();
    boolean[] visited = new boolean[nodeCount];
    Deque<Integer> queue = new ArrayDeque<>();

    queue.add(startNode);
    visited[startNode] = true;

    while (!queue.isEmpty()) {
        int current = queue.poll();
        visitOrder.add(current);

        for (int neighbor : graph.get(current)) {
            if (!visited[neighbor]) {
                visited[neighbor] = true;
                queue.add(neighbor);
            }
        }
    }

    return visitOrder;
}
```
