# Linked List

## Reverse a Linked List

Reverse by redirecting each node's `next` pointer to point backward.

```
  Before:  1 -> 2 -> 3 -> 4 -> null
  After:   null <- 1 <- 2 <- 3 <- 4

  Step by step:
    prev=null, current=1:  1.next=null,   prev=1, current=2
    prev=1,    current=2:  2.next=1,      prev=2, current=3
    prev=2,    current=3:  3.next=2,      prev=3, current=4
    prev=3,    current=4:  4.next=3,      prev=4, current=null
  Return prev (4)
```

```java
ListNode reverse(ListNode head) {
    ListNode previous = null, current = head;

    while (current != null) {
        ListNode nextNode = current.next;  // Save next before overwriting
        current.next = previous;           // Reverse the link
        previous = current;                // Advance previous
        current = nextNode;                // Advance current
    }
    return previous; // New head
}
```

## Merge Two Sorted Lists

Use a **dummy head** to simplify list construction. Compare heads of both lists,
append the smaller one, advance that pointer.

```
  a: 1 -> 3 -> 5
  b: 2 -> 4 -> 6

  dummy -> 1 -> 2 -> 3 -> 4 -> 5 -> 6
```

```java
ListNode mergeTwoLists(ListNode listA, ListNode listB) {
    ListNode dummy = new ListNode(0);
    ListNode tail = dummy;

    while (listA != null && listB != null) {
        if (listA.val < listB.val) {
            tail.next = listA;
            listA = listA.next;
        } else {
            tail.next = listB;
            listB = listB.next;
        }
        tail = tail.next;
    }
    // Attach remaining nodes
    tail.next = (listA != null ? listA : listB);
    return dummy.next;
}
```
