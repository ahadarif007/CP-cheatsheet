# Constants

> Java provides min/max constants for numeric types via wrapper classes.
> Useful as initial values for finding min/max in algorithms.
> Be careful with `Integer.MAX_VALUE + 1` — it overflows to `Integer.MIN_VALUE`.

```java
// Integer bounds: -2^31 to 2^31 - 1
int maxInt = Integer.MAX_VALUE;    //  2,147,483,647
int minInt = Integer.MIN_VALUE;    // -2,147,483,648

// Long bounds: -2^63 to 2^63 - 1
long maxLong = Long.MAX_VALUE;     //  9,223,372,036,854,775,807
long minLong = Long.MIN_VALUE;     // -9,223,372,036,854,775,808
```

```
Common usage patterns:
  Finding minimum → initialize with Integer.MAX_VALUE
  Finding maximum → initialize with Integer.MIN_VALUE

  int minVal = Integer.MAX_VALUE;
  for (int x : nums) minVal = Math.min(minVal, x);
```
