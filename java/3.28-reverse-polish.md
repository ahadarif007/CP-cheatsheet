# Reverse Polish Notation (RPN)

Evaluate an expression in **postfix notation** using a stack.
Operands are pushed onto the stack. When an operator appears, pop two operands, apply the operator, push the result.

```
  Tokens: ["2", "1", "+", "3", "*"]

  Equivalent infix: (2 + 1) * 3 = 9

  Stack trace:
    "2"  -> push 2        stack: [2]
    "1"  -> push 1        stack: [2, 1]
    "+"  -> pop 1,2 -> 3  stack: [3]
    "3"  -> push 3        stack: [3, 3]
    "*"  -> pop 3,3 -> 9  stack: [9]

  Result: 9
```

Note: The second popped value is the **left operand** (order matters for `-` and `/`).

```java
int evalRPN(List<String> tokens) {
    Deque<Integer> stack = new ArrayDeque<>();

    for (String token : tokens) {
        if (!"+-/*".contains(token)) {
            stack.push(Integer.parseInt(token)); // Operand
        } else {
            int rightOperand = stack.pop();
            int leftOperand = stack.pop();
            switch (token) {
                case "+": stack.push(leftOperand + rightOperand); break;
                case "-": stack.push(leftOperand - rightOperand); break;
                case "*": stack.push(leftOperand * rightOperand); break;
                default:  stack.push(leftOperand / rightOperand); break;
            }
        }
    }
    return stack.pop();
}
```
