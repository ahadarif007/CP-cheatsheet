# Ternary Operator

> Shorthand for `if-else` that returns a value.
> Syntax: `condition ? valueIfTrue : valueIfFalse`
> Use for simple assignments; avoid nesting for readability.

```java
int age = 20;

// Instead of:
//   int category;
//   if (age >= 18) category = 1;
//   else category = 0;

// Use ternary:
int category = (age >= 18) ? 1 : 0;  // category = 1

// Works with any type
String status = (age >= 18) ? "adult" : "minor";  // "adult"
```
