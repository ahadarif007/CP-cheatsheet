# Basic Calculator

Evaluate a string expression with `+`, `-`, `*`, `/`, and parentheses `()`.
Uses **recursive descent** with a stack for operator precedence.

```
  Expression: "2 * (3 + 4) - 5"

  Step 1: Parse "2", sign='+'
  Step 2: Hit '*', push 2, sign='*'
  Step 3: Hit '(', recurse into sub-expression "3 + 4"
          -> Returns 7, resume at ')'
  Step 4: sign='*', stack.pop() * 7 = 14, push 14
  Step 5: Hit '-', sign='-'
  Step 6: Parse "5", push -5
  Step 7: Sum stack: 14 + (-5) = 9
```

**Operator precedence:**
- `*` and `/` apply immediately to the top of the stack
- `+` and `-` just push values (applied when stack is summed)

```java
int calculate(String expression) {
    expression = expression + "$"; // Sentinel to trigger final processing
    return (int) evaluate(new ArrayDeque<>(), expression, 0)[0];
}

long[] evaluate(Deque<Long> stack, String expression, int index) {
    long currentNumber = 0;
    char pendingSign = '+';

    while (index < expression.length()) {
        char ch = expression.charAt(index);

        // Skip whitespace
        if (ch == ' ') { index++; continue; }

        // Build multi-digit number
        if (Character.isDigit(ch)) {
            currentNumber = currentNumber * 10 + (ch - '0');
            index++;
            continue;
        }

        // Handle parenthesized sub-expression
        if (ch == '(') {
            long[] subResult = evaluate(new ArrayDeque<>(), expression, index + 1);
            currentNumber = subResult[0];
            index = (int) subResult[1];
            ch = expression.charAt(index);
        }

        // Apply the pending sign when we hit an operator or end
        if (!Character.isDigit(ch)) {
            if (pendingSign == '+') stack.push(currentNumber);
            else if (pendingSign == '-') stack.push(-currentNumber);
            else if (pendingSign == '*') stack.push(stack.pop() * currentNumber);
            else if (pendingSign == '/') stack.push(stack.pop() / currentNumber);

            // Closing paren: return sum and resume position
            if (ch == ')') {
                return new long[]{stack.stream().mapToLong(x -> x).sum(), index + 1};
            }

            pendingSign = ch;
            currentNumber = 0;
            index++;
        }
    }
    return new long[]{stack.stream().mapToLong(x -> x).sum(), index};
}
```
