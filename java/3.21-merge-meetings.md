# Merge Meetings (Insert Interval)

Insert a new interval into a list of non-overlapping intervals, merging as needed.
**Strategy:** Sort all intervals by start time, then merge overlapping ones greedily.

```
  intervals: [[1,3], [6,9]]
  newInterval: [2,5]

  After adding: [[1,3], [2,5], [6,9]]
  After sorting: [[1,3], [2,5], [6,9]]
  Merge:
    [1,3] + [2,5] -> overlap (2 <= 3), merge to [1,5]
    [1,5] + [6,9] -> no overlap (6 > 5), keep separate

  Result: [[1,5], [6,9]]
```

```java
List<int[]> insert(int[][] intervals, int[] newInterval) {
    // Add new interval and sort by start time
    List<int[]> allIntervals = new ArrayList<>(Arrays.asList(intervals));
    allIntervals.add(newInterval);
    allIntervals.sort(Comparator.comparingInt(interval -> interval[0]));

    List<int[]> merged = new ArrayList<>();
    merged.add(allIntervals.get(0));

    for (int[] current : allIntervals) {
        int[] lastMerged = merged.get(merged.size() - 1);

        if (current[0] <= lastMerged[1]) {
            // Overlapping: extend the end of last merged interval
            lastMerged[1] = Math.max(lastMerged[1], current[1]);
        } else {
            // Non-overlapping: add as a new interval
            merged.add(current);
        }
    }
    return merged;
}
```
