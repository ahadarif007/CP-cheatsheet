# Counter (Frequency Counting)

Java has no built-in `Counter` like Python. Use an **int array** for characters or a **HashMap** for general elements.

## Character Frequency (Array-based) - O(n) time, O(1) space

Best for lowercase letters (`a-z`), uppercase, or ASCII characters.

```java
// Count frequency of each lowercase letter in a string
int[] charFrequency = new int[26];
for (char ch : text.toCharArray()) {
    charFrequency[ch - 'a']++;
}
// charFrequency[0] = count of 'a', charFrequency[1] = count of 'b', etc.
```

## Top-K Frequent Elements

Given an array, return the `k` most frequent elements.

```
Input:  nums = [1,1,1,2,2,3], k = 2
Output: [1, 2]

Frequency map:  {1:3, 2:2, 3:1}
                 ^^^  ^^^
                 top 2 by frequency
```

```java
List<Integer> topKFrequent(int[] nums, int k) {
    // Step 1: Build frequency map
    Map<Integer, Integer> frequencyMap = new HashMap<>();
    for (int num : nums) {
        frequencyMap.put(num, frequencyMap.getOrDefault(num, 0) + 1);
    }

    // Step 2: Sort by frequency descending and take top k
    return frequencyMap.entrySet().stream()
        .sorted((a, b) -> b.getValue() - a.getValue())
        .limit(k)
        .map(Map.Entry::getKey)
        .toList();
}
```
