# Strings

> Strings in Java are **immutable** — every modification creates a new String.
> Use `StringBuilder` for efficient string building in loops.
> Characters are primitives (`char`), accessed with `charAt()`.

## Common String Operations

```java
String text = "hello:world:java";

// Find first/last occurrence of a character
int firstIndex = text.indexOf('o');       // 4
int lastIndex  = text.lastIndexOf('o');   // 7

// Split string by delimiter → returns array
String[] tokens = text.split(":");       // ["hello", "world", "java"]

// Join list of strings with delimiter
List<String> words = List.of("a", "b", "c");
String joined = String.join(" ", words);  // "a b c"

// StringBuilder — mutable, efficient for building strings
StringBuilder builder = new StringBuilder();
builder.append("hello");
builder.append(" ");
builder.append("world");
String result = builder.toString();       // "hello world"

// Access individual character
char firstChar = text.charAt(0);          // 'h'

// Character type checks
boolean isUpper = Character.isUpperCase('A');  // true
boolean isLower = Character.isLowerCase('a');  // true
boolean isDigit = Character.isDigit('5');      // true

// Replace all occurrences of a substring
String cleaned = "a()b()c".replace("()", "");  // "abc"

// Prefix and suffix checks
boolean hasPrefix = text.startsWith("hello");  // true
boolean hasSuffix = text.endsWith("java");     // true

// Formatted string (like printf)
String formatted = String.format("Name: %s, Age: %d", "Ali", 25);
```

## Manual Word Splitting (by letter characters only)

> Splits a string into words, treating any non-letter as a delimiter.
> Useful when delimiters vary (spaces, punctuation, digits, etc.).

```
Input:  "hello42world! java"
         ^^^^^         ^^^^
         word1  word2  word3

Non-letter chars (4, 2, !, space) act as separators.
```

```java
List<String> splitByLetters(String input) {
    List<String> words = new ArrayList<>();
    int wordStart = -1;  // -1 means we're not inside a word

    for (int i = 0; i < input.length(); i++) {
        char ch = input.charAt(i);

        if (Character.isLetter(ch)) {
            // Start of a new word
            if (wordStart == -1) wordStart = i;
        } else {
            // End of current word (if we were inside one)
            if (wordStart != -1) {
                words.add(input.substring(wordStart, i));
                wordStart = -1;
            }
        }
    }

    // Don't forget the last word if string ends with a letter
    if (wordStart != -1) {
        words.add(input.substring(wordStart));
    }

    return words;
}
```
