# Reservoir Sampling

Select a **random element** from a stream where multiple elements match a target,
giving each match an **equal probability** of being chosen.

```
  nums = [1, 2, 3, 3, 3], target = 3

  Indices with value 3: [2, 3, 4]

  As we scan:
    i=2: count=1, pick with probability 1/1 -> always pick (result=2)
    i=3: count=2, pick with probability 1/2 -> maybe replace (result=2 or 3)
    i=4: count=3, pick with probability 1/3 -> maybe replace (result=2, 3, or 4)

  Each index has exactly 1/3 chance of being the final result.
```

**Why it works:** Each new match replaces the current pick with probability `1/count`, so all matches end up equally likely.

```java
class RandomPicker {
    int[] nums;

    RandomPicker(int[] nums) {
        this.nums = nums;
    }

    int pick(int target) {
        Integer chosenIndex = null;
        int matchCount = 0;

        for (int i = 0; i < nums.length; i++) {
            if (nums[i] == target) {
                matchCount++;
                // Replace with probability 1/matchCount
                int random = ThreadLocalRandom.current().nextInt(1, matchCount + 1);
                if (random == 1) {
                    chosenIndex = i;
                }
            }
        }
        return chosenIndex;
    }
}
```
