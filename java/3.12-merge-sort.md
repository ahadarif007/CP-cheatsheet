# Merge Sort

Merge sort **divides** the array in half, recursively sorts both halves, then **merges** them.
**Time:** O(n log n) always. **Space:** O(n) for temporary arrays.

```
  Split phase:          Merge phase:
  [38, 27, 43, 3]      [3, 27, 38, 43]
     /        \            ^
  [38, 27]  [43, 3]     merge
   /    \    /    \        |
 [38] [27] [43]  [3]   [27,38] + [3,43]
   \    /    \    /        ^        ^
  [27,38]   [3,43]      merge    merge
```

```java
int[] mergeSort(int[] array) {
    if (array.length <= 1) return array;

    int mid = array.length / 2;
    int[] leftHalf = mergeSort(Arrays.copyOfRange(array, 0, mid));
    int[] rightHalf = mergeSort(Arrays.copyOfRange(array, mid, array.length));
    return mergeSorted(leftHalf, rightHalf);
}

int[] mergeSorted(int[] left, int[] right) {
    int[] result = new int[left.length + right.length];
    int leftIdx = 0, rightIdx = 0, resultIdx = 0;

    // Merge by picking the smaller element each time
    while (leftIdx < left.length && rightIdx < right.length) {
        result[resultIdx++] = left[leftIdx] <= right[rightIdx]
            ? left[leftIdx++]
            : right[rightIdx++];
    }

    // Copy remaining elements
    while (leftIdx < left.length) result[resultIdx++] = left[leftIdx++];
    while (rightIdx < right.length) result[resultIdx++] = right[rightIdx++];
    return result;
}
```
