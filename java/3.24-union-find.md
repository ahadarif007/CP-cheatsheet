# Union Find / Disjoint Set Union (DSU)

Efficiently tracks **connected components**. Supports two operations:
- **find(x):** Which group does x belong to? (with path compression)
- **union(a, b):** Merge the groups of a and b (with union by size)

```
  Initially: {0} {1} {2} {3} {4}   (5 separate components)

  union(0, 1):  {0,1} {2} {3} {4}
  union(2, 3):  {0,1} {2,3} {4}
  union(1, 3):  {0,1,2,3} {4}

  find(2) == find(0)?  -> true  (same component)
  find(2) == find(4)?  -> false (different components)
```

**Time:** Nearly O(1) per operation (amortized with path compression + union by size).

```java
class DisjointSetUnion {
    int[] parent;
    int[] size;

    DisjointSetUnion(int numNodes) {
        parent = new int[numNodes];
        size = new int[numNodes];
        for (int i = 0; i < numNodes; i++) {
            parent[i] = i;  // Each node is its own root
            size[i] = 1;
        }
    }

    int find(int node) {
        // Path compression: point directly to root
        if (parent[node] != node) {
            parent[node] = find(parent[node]);
        }
        return parent[node];
    }

    boolean union(int nodeA, int nodeB) {
        int rootA = find(nodeA), rootB = find(nodeB);
        if (rootA == rootB) return false; // Already in same component

        // Union by size: attach smaller tree under larger tree
        if (size[rootA] < size[rootB]) {
            int temp = rootA;
            rootA = rootB;
            rootB = temp;
        }
        parent[rootB] = rootA;
        size[rootA] += size[rootB];
        return true;
    }
}
```
