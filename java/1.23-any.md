# Any (anyMatch)

> `anyMatch()` returns `true` if **at least one** element satisfies the condition.
> Short-circuits — stops checking as soon as a match is found.
> Equivalent to Python's `any()`.

```java
int[] nums = {-3, -1, 0, 2, 5};

// Check if any element is positive
boolean hasPositive = Arrays.stream(nums)
    .anyMatch(x -> x > 0);
// true (2 and 5 are positive, stops at 2)

// Check if any element equals a target
boolean containsZero = Arrays.stream(nums)
    .anyMatch(x -> x == 0);
// true
```
