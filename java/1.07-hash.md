# Hash (Frequency Map)

> A HashMap counts how many times each element appears.
> `getOrDefault(key, 0)` avoids null when key is missing.
> Common pattern: character/word frequency counting.

```
Input string: "aabbc"

Building frequency map step by step:
  'a' → map = {a:1}
  'a' → map = {a:2}
  'b' → map = {a:2, b:1}
  'b' → map = {a:2, b:2}
  'c' → map = {a:2, b:2, c:1}
```

```java
String text = "aabbc";
Map<Character, Integer> frequency = new HashMap<>();

for (char ch : text.toCharArray()) {
    // If key exists, get its count; otherwise default to 0, then add 1
    frequency.put(ch, frequency.getOrDefault(ch, 0) + 1);
}
// frequency = {a=2, b=2, c=1}
```

> **Tip**: For counting elements in an int array, the same pattern works:
> ```java
> Map<Integer, Integer> count = new HashMap<>();
> for (int num : nums) {
>     count.put(num, count.getOrDefault(num, 0) + 1);
> }
> ```
