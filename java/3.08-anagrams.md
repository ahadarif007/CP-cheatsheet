# Anagrams

Two strings are **anagrams** if they contain the same characters with the same frequencies.

## Check if Two Strings are Anagrams - O(n) time, O(1) space

**Trick:** Increment for first string, decrement for second. If any count goes negative, not an anagram.

```
  s = "listen"    t = "silent"
  After counting s: l:1 i:1 s:1 t:1 e:1 n:1
  Decrementing t:   s:0 i:0 l:0 e:0 n:0 t:0  -> all zero -> anagram!
```

```java
boolean isAnagram(String first, String second) {
    if (first.length() != second.length()) return false;

    int[] charCount = new int[26];
    for (char ch : first.toCharArray()) charCount[ch - 'a']++;
    for (char ch : second.toCharArray()) {
        if (--charCount[ch - 'a'] < 0) return false;
    }
    return true;
}
```

## Find All Anagram Indices (Sliding Window)

Find all starting indices in `text` where an anagram of `pattern` begins.
Use a **fixed-size sliding window** of length `pattern.length()`.

```
  text:    "c b a e b a b a c d"
  pattern: "abc"

  Window slides:
    [c b a] -> matches!  index 0
     [b a e] -> no
      [a e b] -> no
          ...
           [b a c] -> matches!  index 6
```

```java
List<Integer> findAnagrams(String text, String pattern) {
    int[] targetFreq = new int[26];
    for (char ch : pattern.toCharArray()) targetFreq[ch - 'a']++;

    int[] windowFreq = new int[26];
    List<Integer> result = new ArrayList<>();
    int windowSize = pattern.length();

    for (int i = 0; i < text.length(); i++) {
        // Add right character to window
        windowFreq[text.charAt(i) - 'a']++;

        // Remove leftmost character when window exceeds pattern size
        if (i >= windowSize) {
            windowFreq[text.charAt(i - windowSize) - 'a']--;
        }

        // Check if current window matches the pattern frequency
        if (Arrays.equals(targetFreq, windowFreq)) {
            result.add(i - windowSize + 1);
        }
    }
    return result;
}
```
