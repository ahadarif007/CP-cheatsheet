# Max Profit Stock

Stock buy/sell problems follow a **state machine** pattern:
- `cashHeld` (t0): max profit when NOT holding stock
- `stockHeld` (t1): max profit when holding stock

```
  State Machine:

  [Not Holding] --buy--> [Holding]
       ^                     |
       |                     |
       +------sell-----------+
```

## Infinite Transactions (Buy/Sell Any Number of Times)

```
  prices: [7, 1, 5, 3, 6, 4]

  Buy at 1, sell at 5 (profit 4)
  Buy at 3, sell at 6 (profit 3)
  Total profit: 7
```

```java
int maxProfit(int[] prices) {
    int cashHeld = 0;                    // Max profit without stock
    int stockHeld = Integer.MIN_VALUE;   // Max profit while holding stock

    for (int price : prices) {
        int previousCash = cashHeld;
        cashHeld = Math.max(cashHeld, stockHeld + price);       // Sell
        stockHeld = Math.max(stockHeld, previousCash - price);  // Buy
    }
    return cashHeld;
}
```

## Single Transaction Only (Buy Once, Sell Once)

The only difference: when buying, we start from 0 profit (not previous cash).

```java
int maxProfitSingleTransaction(int[] prices) {
    int cashHeld = 0;
    int stockHeld = Integer.MIN_VALUE;

    for (int price : prices) {
        cashHeld = Math.max(cashHeld, stockHeld + price);  // Sell
        stockHeld = Math.max(stockHeld, -price);           // Buy from 0 (only 1 buy allowed)
    }
    return cashHeld;
}
```

## At Most K Transactions

Generalize to K transactions. Each transaction `i` has its own `cashHeld[i]` and `stockHeld[i]`.

```
  Transition for transaction i:
    cashHeld[i]  = max(cashHeld[i],  stockHeld[i] + price)    // Sell in transaction i
    stockHeld[i] = max(stockHeld[i], cashHeld[i-1] - price)   // Buy using profit from transaction i-1
```

```java
int maxProfitKTransactions(int maxTransactions, int[] prices) {
    int[] cashHeld = new int[maxTransactions + 1];
    int[] stockHeld = new int[maxTransactions + 1];
    Arrays.fill(stockHeld, Integer.MIN_VALUE / 4); // Avoid overflow

    for (int price : prices) {
        // Process in reverse to avoid using updated values
        for (int txn = maxTransactions; txn >= 1; txn--) {
            cashHeld[txn] = Math.max(cashHeld[txn], stockHeld[txn] + price);
            stockHeld[txn] = Math.max(stockHeld[txn], cashHeld[txn - 1] - price);
        }
    }
    return cashHeld[maxTransactions];
}
```
