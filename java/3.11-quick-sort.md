# Quick Sort

Quick sort picks a **pivot**, partitions the array so all elements less than the pivot go left
and all greater go right, then recursively sorts both halves.

**Time:** O(n log n) average, O(n^2) worst case. **Space:** O(log n) for recursion stack.

```
  [3, 6, 8, 10, 1, 2, 1]    pivot = 1 (last element)

  Partition:
  [1, 1, | 8, 10, 6, 2, 3]   elements < pivot moved left
         ^
         pivot placed here

  Recurse on [1] and [8, 10, 6, 2, 3]
```

```java
void quickSort(int[] array) {
    quickSortHelper(array, 0, array.length - 1);
}

void quickSortHelper(int[] array, int left, int right) {
    if (left >= right) return;

    int pivotIndex = partition(array, left, right);
    quickSortHelper(array, left, pivotIndex - 1);  // Sort left half
    quickSortHelper(array, pivotIndex + 1, right);  // Sort right half
}

int partition(int[] array, int left, int right) {
    int pivotValue = array[right];  // Choose last element as pivot
    int insertPos = left;           // Where to place next small element

    for (int scan = left; scan < right; scan++) {
        if (array[scan] < pivotValue) {
            // Swap smaller element to the left partition
            int temp = array[scan];
            array[scan] = array[insertPos];
            array[insertPos] = temp;
            insertPos++;
        }
    }

    // Place pivot at its correct position
    int temp = array[right];
    array[right] = array[insertPos];
    array[insertPos] = temp;
    return insertPos;
}
```
