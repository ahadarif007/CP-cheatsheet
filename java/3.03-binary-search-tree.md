# Binary Search Tree (BST)

A BST maintains the invariant: `left.val < node.val < right.val`.
**In-order traversal** of a BST gives values in **sorted order**.

```
        8
       / \
      3   10
     / \    \
    1   6    14
       / \   /
      4   7 13
```

## Complete Binary Tree Check (by Index)

A complete binary tree fills every level fully except possibly the last, which fills left to right.
**Trick:** Number nodes by position (root=1, left child=2i, right child=2i+1).
If tree is complete, the **max index == total node count**.

```
  Complete:          NOT Complete:
      1                  1
     / \                / \
    2   3              2   3
   / \                  \
  4   5                  5

  max_idx=5, count=5    max_idx=5, count=4
  5 == 5 -> true        5 != 4 -> false
```

```java
boolean isComplete(TreeNode root) {
    long[] nodeCount = {0};
    long[] maxIndex = {Long.MIN_VALUE};
    assignIndices(root, 1, nodeCount, maxIndex);
    return nodeCount[0] == maxIndex[0];
}

void assignIndices(TreeNode node, long index, long[] nodeCount, long[] maxIndex) {
    if (node == null) return;
    nodeCount[0]++;
    maxIndex[0] = Math.max(maxIndex[0], index);
    assignIndices(node.left, index * 2, nodeCount, maxIndex);
    assignIndices(node.right, index * 2 + 1, nodeCount, maxIndex);
}
```

## Range Sum in BST

Sum all node values in range `[low, high]`. Use BST property to **prune** branches.

```
        10                  Range [7, 15]
       /  \                 Include: 10, 7, 15
      5    15               Skip left subtree of 5 (all < 7)
     / \     \              Skip right of 15 (all > 15)
    3   7     18
```

```java
int rangeSumBST(TreeNode root, int low, int high) {
    if (root == null) return 0;

    int sum = 0;
    // Include current node if in range
    if (low <= root.val && root.val <= high) {
        sum += root.val;
    }
    // Only go left if there might be values >= low
    if (root.val > low) {
        sum += rangeSumBST(root.left, low, high);
    }
    // Only go right if there might be values <= high
    if (root.val < high) {
        sum += rangeSumBST(root.right, low, high);
    }
    return sum;
}
```

## Validate BST (Iterative In-Order)

In-order traversal must produce strictly increasing values. If any value <= previous, it's not a valid BST.

```
In-order of valid BST:   1 -> 3 -> 4 -> 6 -> 7 -> 8 -> 10  (sorted)
In-order of invalid BST: 1 -> 3 -> 6 -> 4 -> ...           (6 > 4, invalid!)
```

```java
boolean isValidBST(TreeNode root) {
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode current = root;
    Long previousValue = null;

    while (current != null || !stack.isEmpty()) {
        // Go as far left as possible
        while (current != null) {
            stack.push(current);
            current = current.left;
        }
        // Process current node
        current = stack.pop();
        if (previousValue != null && current.val <= previousValue) {
            return false; // Not strictly increasing
        }
        previousValue = (long) current.val;
        current = current.right; // Move to right subtree
    }
    return true;
}
```
