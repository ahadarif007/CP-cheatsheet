# Topological Sort (Kahn's Algorithm - BFS)

Topological sort orders nodes so that for every edge `u -> v`, `u` comes before `v`.
Works only on **DAGs** (Directed Acyclic Graphs). If a cycle exists, not all nodes can be processed.

**Algorithm (Kahn's BFS):**
1. Compute in-degree for every node
2. Add all nodes with in-degree 0 to the queue
3. Process queue: for each node, decrement neighbors' in-degrees
4. If a neighbor's in-degree hits 0, add it to queue
5. If processed count == total nodes, no cycle exists

```
Prerequisites: [[1,0], [2,0], [3,1], [3,2]]
(meaning: to take course 1, you must first take course 0)

  0 --> 1 --> 3      In-degrees: 0:0, 1:1, 2:1, 3:2
  |         ^
  +-> 2 ---+         Valid order: 0 -> 1 -> 2 -> 3
                                  0 -> 2 -> 1 -> 3
```

## Course Schedule (Cycle Detection)

```java
boolean canFinish(int numCourses, int[][] prerequisites) {
    // Build adjacency list and compute in-degrees
    List<List<Integer>> adjacencyList = new ArrayList<>();
    for (int i = 0; i < numCourses; i++) {
        adjacencyList.add(new ArrayList<>());
    }
    int[] inDegree = new int[numCourses];

    for (int[] edge : prerequisites) {
        adjacencyList.get(edge[1]).add(edge[0]); // edge[1] -> edge[0]
        inDegree[edge[0]]++;
    }

    // Start with all nodes that have no prerequisites
    Deque<Integer> queue = new ArrayDeque<>();
    for (int i = 0; i < numCourses; i++) {
        if (inDegree[i] == 0) queue.add(i);
    }

    // Process nodes in topological order
    int processedCount = 0;
    while (!queue.isEmpty()) {
        int course = queue.poll();
        processedCount++;
        for (int neighbor : adjacencyList.get(course)) {
            if (--inDegree[neighbor] == 0) {
                queue.add(neighbor); // All prerequisites met
            }
        }
    }

    // If we processed all courses, no cycle exists
    return processedCount == numCourses;
}
```

## Alien Dictionary

Given a sorted list of words in an alien language, derive the character ordering.
Each adjacent word pair gives one ordering rule (first differing character).

```
Words: ["wrt", "wrf", "er", "ett", "rftt"]

Compare pairs:
  "wrt" vs "wrf" -> t < f
  "wrf" vs "er"  -> w < e
  "er"  vs "ett" -> r < t
  "ett" vs "rftt"-> e < r

Graph: w -> e -> r -> t -> f
Order: "wertf"
```

```java
String alienOrder(String[] words) {
    // Collect all unique characters
    Set<Character> allChars = new HashSet<>();
    for (String word : words) {
        for (char ch : word.toCharArray()) allChars.add(ch);
    }

    // Build graph and in-degree map
    Map<Character, List<Character>> adjacencyList = new HashMap<>();
    Map<Character, Integer> inDegree = new HashMap<>();
    for (char ch : allChars) {
        adjacencyList.put(ch, new ArrayList<>());
        inDegree.put(ch, 0);
    }

    // Extract ordering rules from adjacent word pairs
    for (int i = 0; i + 1 < words.length; i++) {
        String firstWord = words[i], secondWord = words[i + 1];
        int minLength = Math.min(firstWord.length(), secondWord.length());
        boolean foundDifference = false;

        for (int j = 0; j < minLength; j++) {
            if (firstWord.charAt(j) != secondWord.charAt(j)) {
                char from = firstWord.charAt(j), to = secondWord.charAt(j);
                adjacencyList.get(from).add(to);
                inDegree.put(to, inDegree.get(to) + 1);
                foundDifference = true;
                break;
            }
        }
        // Invalid: longer word comes before its prefix (e.g., "abc" before "ab")
        if (!foundDifference && firstWord.length() > secondWord.length()) return "";
    }

    // Kahn's BFS topological sort
    Deque<Character> queue = new ArrayDeque<>();
    for (var entry : inDegree.entrySet()) {
        if (entry.getValue() == 0) queue.add(entry.getKey());
    }

    StringBuilder result = new StringBuilder();
    while (!queue.isEmpty()) {
        char ch = queue.poll();
        result.append(ch);
        for (char neighbor : adjacencyList.get(ch)) {
            inDegree.put(neighbor, inDegree.get(neighbor) - 1);
            if (inDegree.get(neighbor) == 0) queue.add(neighbor);
        }
    }

    // If not all characters are in result, there's a cycle
    return result.length() == allChars.size() ? result.toString() : "";
}
```
