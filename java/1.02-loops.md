# Loops

> Java has `for`, enhanced `for-each`, and `while` loops.
> Use `for` when you need the index, `for-each` when you just need values.

## Standard Loops

```java
// Forward loop: 0, 1, 2, ..., n-1
for (int i = 0; i < n; i++) {
    // process element at index i
}

// Backward loop: n-1, n-2, ..., 0
for (int i = n - 1; i >= 0; i--) {
    // process in reverse order
}

// For-each loop — no index, just values
for (int value : nums) {
    // process each value directly
}

// While loop
int i = 0;
while (i < n) {
    // process
    i++;
}
```

## Reverse an Array In-Place (Two-Pointer Swap)

```
Before: [1, 2, 3, 4, 5]
         ↑           ↑
        left       right    → swap 1 ↔ 5

Step 1: [5, 2, 3, 4, 1]
            ↑     ↑
           left  right      → swap 2 ↔ 4

Step 2: [5, 4, 3, 2, 1]
               ↑
          left = right      → done (pointers met)
```

```java
int left = 0;
int right = nums.length - 1;
while (left < right) {
    // Swap elements at left and right pointers
    int temp = nums[left];
    nums[left] = nums[right];
    nums[right] = temp;
    left++;
    right--;
}
```

## Range Equivalents

```java
// range(3) → 0, 1, 2
for (int i = 0; i < 3; i++) {}

// range(3, -1, -1) → 3, 2, 1, 0
for (int i = 3; i >= 0; i--) {}
```
