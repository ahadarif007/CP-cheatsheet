# Loops

```java
for (int i = 0; i < n; i++) {}
for (int i = n - 1; i >= 0; i--) {}
for (int x : nums) {}
int i = 0; while (i < n) { i++; }
```

Reverse in-place

```java
int l = 0, r = nums.length - 1;
while (l < r) { int tmp = nums[l]; nums[l] = nums[r]; nums[r] = tmp; l++; r--; }
```

Ranges

```java
for (int i = 0; i < 3; i++) {}
for (int i = 3; i >= 0; i--) {}
```
