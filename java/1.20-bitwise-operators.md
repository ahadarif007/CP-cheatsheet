# Bitwise Operators

> Operate on individual bits of integer values.
> Essential for bit manipulation problems, flags, and low-level optimizations.

```
Operator overview with examples:

  AND (&):  keeps bits that are 1 in BOTH operands
    1100 & 1010 = 1000  (decimal: 12 & 10 = 8)

  OR  (|):  keeps bits that are 1 in EITHER operand
    1100 | 1010 = 1110  (decimal: 12 | 10 = 14)

  XOR (^):  keeps bits that DIFFER between operands
    1100 ^ 1010 = 0110  (decimal: 12 ^ 10 = 6)

  Right Shift (>>):   divides by 2^n (keeps sign bit)
    1100 >> 2 = 0011   (decimal: 12 >> 2 = 3)

  Left Shift  (<<):   multiplies by 2^n
    0011 << 2 = 1100   (decimal: 3 << 2 = 12)

  Unsigned Right Shift (>>>): fills with 0s (ignores sign)
    -1 >>> 1 = 2147483647  (Integer.MAX_VALUE)
```

```java
int andResult = (0b1100 & 0b1010);   // 0b1000 = 8
int orResult  = (0b1100 | 0b1010);   // 0b1110 = 14
int xorResult = (0b1100 ^ 0b1010);   // 0b0110 = 6

int rightShift = (0b1100 >> 2);      // 0b0011 = 3
int leftShift  = (0b0011 << 2);      // 0b1100 = 12

// Unsigned right shift — treats bits as unsigned
int unsignedShift = (-1 >>> 1);      // Integer.MAX_VALUE
```

> **Common bit tricks:**
> - Check if even: `(n & 1) == 0`
> - Check if power of 2: `n > 0 && (n & (n - 1)) == 0`
> - Swap without temp: `a ^= b; b ^= a; a ^= b;`
