# Zip (Parallel Iteration)

> Java has no built-in `zip()` like Python.
> Use a shared index loop to iterate two collections in parallel.

```java
List<String> names = List.of("Alice", "Bob", "Charlie");
List<Integer> scores = List.of(95, 87, 92);

// Iterate both lists together using a shared index
int limit = Math.min(names.size(), scores.size());
for (int i = 0; i < limit; i++) {
    String name = names.get(i);
    int score = scores.get(i);
    // name="Alice", score=95 → name="Bob", score=87 → ...
}
```

## Rotate Matrix 90 Degrees Clockwise

> Two-step approach: **transpose** (swap rows/columns), then **reverse each row**.

```
Original:       Transpose:       Reverse rows:
1  2  3         1  4  7          7  4  1
4  5  6    →    2  5  8    →     8  5  2
7  8  9         3  6  9          9  6  3

Step 1: Swap matrix[i][j] with matrix[j][i] (above diagonal only)
Step 2: Reverse each row
```

```java
void rotateMatrix(int[][] matrix) {
    int size = matrix.length;

    // Step 1: Transpose — swap across the main diagonal
    for (int row = 0; row < size; row++) {
        for (int col = row + 1; col < size; col++) {
            int temp = matrix[row][col];
            matrix[row][col] = matrix[col][row];
            matrix[col][row] = temp;
        }
    }

    // Step 2: Reverse each row
    for (int row = 0; row < size; row++) {
        int left = 0, right = size - 1;
        while (left < right) {
            int temp = matrix[row][left];
            matrix[row][left] = matrix[row][right];
            matrix[row][right] = temp;
            left++;
            right--;
        }
    }
}
```
