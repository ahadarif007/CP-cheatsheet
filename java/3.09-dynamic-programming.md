# Dynamic Programming

DP solves problems by breaking them into **overlapping subproblems** and storing results to avoid recomputation.

## Coin Change (Unbounded Knapsack)

Find the **minimum number of coins** to make a given amount. Each coin can be used unlimited times.

```
  coins = [1, 3, 4], amount = 6

  dp[i] = minimum coins to make amount i

  dp: [0, 1, 2, 1, 1, 2, 2]
       ^  ^  ^  ^  ^  ^  ^
       0  1  1+1 3 4 3+1+1 3+3

  Answer: dp[6] = 2 (coins: 3 + 3)
```

```java
int coinChange(int[] coins, int amount) {
    int INFINITY = 1_000_000_000;
    int[] minCoins = new int[amount + 1];
    Arrays.fill(minCoins, INFINITY);
    minCoins[0] = 0; // Base case: 0 coins needed for amount 0

    for (int coin : coins) {
        for (int currentAmount = coin; currentAmount <= amount; currentAmount++) {
            // Either skip this coin, or use it and add 1 to the count
            minCoins[currentAmount] = Math.min(
                minCoins[currentAmount],
                minCoins[currentAmount - coin] + 1
            );
        }
    }
    return minCoins[amount] >= INFINITY ? -1 : minCoins[amount];
}
```

## Longest Common Subsequence (LCS)

Find the length of the longest subsequence common to two strings.
A **subsequence** can skip characters but must maintain relative order.

```
  a = "abcde"
  b = "ace"

  LCS = "ace" (length 3)

  DP table (dp[i][j] = LCS of a[0..i-1] and b[0..j-1]):

      ""  a  c  e
  ""   0  0  0  0
  a    0  1  1  1     <- 'a' matches 'a'
  b    0  1  1  1
  c    0  1  2  2     <- 'c' matches 'c'
  d    0  1  2  2
  e    0  1  2  3     <- 'e' matches 'e'
```

```java
int longestCommonSubsequence(String textA, String textB) {
    int lengthA = textA.length(), lengthB = textB.length();
    int[][] dp = new int[lengthA + 1][lengthB + 1];

    for (int i = 0; i < lengthA; i++) {
        for (int j = 0; j < lengthB; j++) {
            if (textA.charAt(i) == textB.charAt(j)) {
                dp[i + 1][j + 1] = dp[i][j] + 1; // Characters match
            } else {
                // Take the best from excluding either character
                dp[i + 1][j + 1] = Math.max(dp[i][j + 1], dp[i + 1][j]);
            }
        }
    }
    return dp[lengthA][lengthB];
}
```
