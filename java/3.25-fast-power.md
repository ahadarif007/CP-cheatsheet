# Fast Power (Binary Exponentiation)

Compute `x^n` in **O(log n)** time by squaring the base and halving the exponent.

```
  x = 2, n = 10

  Binary of 10 = 1010

  Iteration:  exponent   bit   action            result
  1           10 (1010)   0    square x           result=1,   x=4
  2           5  (101)    1    result *= x        result=4,   x=16
  3           2  (10)     0    square x           result=4,   x=256
  4           1  (1)      1    result *= x        result=1024

  2^10 = 1024
```

**Key idea:** If exponent bit is 1, multiply result by current base. Always square the base.

```java
double myPow(double base, int n) {
    long exponent = n;
    // Handle negative exponent: x^(-n) = (1/x)^n
    if (exponent < 0) {
        base = 1 / base;
        exponent = -exponent;
    }

    double result = 1;
    while (exponent > 0) {
        if ((exponent & 1) == 1) {
            result *= base;    // Odd exponent: multiply into result
        }
        base *= base;          // Square the base
        exponent >>= 1;        // Halve the exponent
    }
    return result;
}
```
