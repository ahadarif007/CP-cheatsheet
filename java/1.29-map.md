# Map (Stream Transform)

> `stream().map()` transforms each element and collects the results.
> Does **not** modify the original collection — returns a new list.
> Equivalent to Python's `map()` or list comprehension `[f(x) for x in list]`.

```java
List<String> words = List.of("hello", "world", "java");

// Transform each string to uppercase
List<String> uppercased = words.stream()
    .map(String::toUpperCase)
    .toList();
// ["HELLO", "WORLD", "JAVA"]

// Transform integers: square each number
List<Integer> nums = List.of(1, 2, 3, 4);
List<Integer> squares = nums.stream()
    .map(n -> n * n)
    .toList();
// [1, 4, 9, 16]
```
