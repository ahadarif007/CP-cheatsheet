# Deque (Double-Ended Queue)

A **Deque** allows insertion and removal from **both ends** in O(1) time.
Use `ArrayDeque` over `LinkedList` for better cache performance and lower overhead.

```
  addFirst(5)                    addLast(6)
      |                              |
      v                              v
  +---------+---------+---------+---------+
  |    5    |   ...   |   ...   |    6    |
  +---------+---------+---------+---------+
      ^                              ^
      |                              |
  removeFirst()                 removeLast()
```

**Common use cases:** BFS queue, sliding window max/min, stack replacement.

```java
Deque<Integer> deque = new ArrayDeque<>();

// Add to front and back
deque.addFirst(5);
deque.addLast(6);

// Remove from front and back
deque.removeFirst();
deque.removeLast();

// Peek without removing
deque.peekFirst();
deque.peekLast();
```

| Operation      | Method          | Time  |
|----------------|-----------------|-------|
| Add front      | `addFirst(e)`   | O(1)  |
| Add back       | `addLast(e)`    | O(1)  |
| Remove front   | `removeFirst()` | O(1)  |
| Remove back    | `removeLast()`  | O(1)  |
| Peek front     | `peekFirst()`   | O(1)  |
| Peek back      | `peekLast()`    | O(1)  |
