# Parenthesis

## Count-Only Validation (Single Type)

For strings with only `(` and `)`, just track the **balance** (open count).
Invalid if balance ever goes negative.

```
  "(())()"
   (  (  )  )  (  )
   1  2  1  0  1  0   <- balance never negative, ends at 0 -> valid

  "())"
   (  )  )
   1  0  -1  <- negative! -> invalid
```

```java
boolean isValidParentheses(String text) {
    int balance = 0;
    for (char ch : text.toCharArray()) {
        if (ch == '(') balance++;
        else if (ch == ')') {
            balance--;
            if (balance < 0) return false; // More closing than opening
        }
    }
    return balance == 0; // All opened brackets are closed
}
```

## Stack-Based Validation (Multiple Types)

For `()`, `{}`, `[]` - use a **stack**. Push opening brackets, pop and match for closing.

```
  "{[()]}"
   { -> push {
   [ -> push [
   ( -> push (
   ) -> pop (, matches -> ok
   ] -> pop [, matches -> ok
   } -> pop {, matches -> ok
   Stack empty -> valid!
```

```java
boolean isValid(String text) {
    Deque<Character> stack = new ArrayDeque<>();
    Map<Character, Character> closingToOpening = Map.of(
        ')', '(', '}', '{', ']', '['
    );

    for (char ch : text.toCharArray()) {
        if (closingToOpening.containsValue(ch)) {
            stack.push(ch); // Opening bracket -> push
        } else if (closingToOpening.containsKey(ch)) {
            // Closing bracket -> must match top of stack
            if (stack.isEmpty() || stack.pop() != closingToOpening.get(ch)) {
                return false;
            }
        }
    }
    return stack.isEmpty();
}
```

## Remove Minimum Parentheses to Make Valid

Remove the minimum number of `(` or `)` to make the string valid.
**Two passes:** mark unmatched `)` during forward scan, then mark unmatched `(` remaining in the stack.

```
  "a)b(c)d("
   a ) b ( c ) d (
       ^           ^    <- unmatched, mark for deletion
  Result: "ab(c)d"
```

```java
String minRemoveToMakeValid(String text) {
    Deque<Integer> unmatchedOpenIndices = new ArrayDeque<>();
    boolean[] markedForDeletion = new boolean[text.length()];

    for (int i = 0; i < text.length(); i++) {
        char ch = text.charAt(i);
        if (ch == '(') {
            unmatchedOpenIndices.push(i); // Track opening bracket position
        } else if (ch == ')') {
            if (!unmatchedOpenIndices.isEmpty()) {
                unmatchedOpenIndices.pop(); // Matched with an opening bracket
            } else {
                markedForDeletion[i] = true; // Unmatched closing bracket
            }
        }
    }

    // Mark all remaining unmatched opening brackets
    while (!unmatchedOpenIndices.isEmpty()) {
        markedForDeletion[unmatchedOpenIndices.pop()] = true;
    }

    // Build result skipping deleted characters
    StringBuilder result = new StringBuilder();
    for (int i = 0; i < text.length(); i++) {
        if (!markedForDeletion[i]) result.append(text.charAt(i));
    }
    return result.toString();
}
```
