# Dutch National Flag (3-Way Partition)

Sort an array of 0s, 1s, and 2s **in-place** in a single pass using three pointers.

```
  Three regions maintained by pointers:

  [0, 0, 0, | 1, 1, 1, | ?, ?, ? | 2, 2, 2]
   ^         ^           ^         ^
   0      zeroEnd     current   twoStart
           (nextZero)             (nextTwo)

  Rules:
    nums[current] == 0 -> swap with nextZero, advance both
    nums[current] == 1 -> just advance current
    nums[current] == 2 -> swap with nextTwo, shrink nextTwo (don't advance current!)
```

Note: When swapping with `nextTwo`, we don't advance `current` because the swapped-in value hasn't been inspected yet.

```java
void sortColors(int[] nums) {
    int nextZero = 0;                  // Next position for 0
    int current = 0;                   // Current element being inspected
    int nextTwo = nums.length - 1;     // Next position for 2

    while (current <= nextTwo) {
        if (nums[current] == 0) {
            // Swap current element to the 0-region
            int temp = nums[nextZero];
            nums[nextZero] = nums[current];
            nums[current] = temp;
            nextZero++;
            current++;
        } else if (nums[current] == 2) {
            // Swap current element to the 2-region
            int temp = nums[current];
            nums[current] = nums[nextTwo];
            nums[nextTwo] = temp;
            nextTwo--;
            // Don't advance current: swapped value needs inspection
        } else {
            current++; // 1 is already in the middle
        }
    }
}
```
