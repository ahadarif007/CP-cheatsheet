# Binary Tree

> A binary tree is a hierarchical structure where each node has at most 2 children.
> Three main traversal orders: **preorder** (root-left-right), **inorder** (left-root-right), **postorder** (left-right-root).
> BFS (level order) visits nodes level by level using a queue.

## TreeNode Definition

```java
class TreeNode {
    int val;
    TreeNode left, right;

    TreeNode() {}
    TreeNode(int val) { this.val = val; }
}
```

## Recursive Traversals

```
Example tree:
        1
       / \
      2   3
     / \
    4   5

Preorder  (Root → Left → Right):  [1, 2, 4, 5, 3]
Inorder   (Left → Root → Right):  [4, 2, 5, 1, 3]
Postorder (Left → Right → Root):  [4, 5, 2, 3, 1]
```

```java
// Preorder: visit root FIRST, then left, then right
void preorder(TreeNode node) {
    if (node == null) return;
    visit(node);            // process current node
    preorder(node.left);
    preorder(node.right);
}

// Inorder: visit left, then root, then right (gives sorted order for BST)
void inorder(TreeNode node) {
    if (node == null) return;
    inorder(node.left);
    visit(node);            // process current node
    inorder(node.right);
}

// Postorder: visit left, then right, then root LAST
void postorder(TreeNode node) {
    if (node == null) return;
    postorder(node.left);
    postorder(node.right);
    visit(node);            // process current node
}
```

## Iterative Inorder Traversal (Using Stack)

> Simulates the call stack manually. Push all left children, then pop and go right.

```
Stack simulation for tree [1, 2, 3, 4, 5]:

Push lefts:  stack = [1, 2, 4]
Pop 4:       output = [4],       go right (null)
Pop 2:       output = [4, 2],    go right → push 5
Pop 5:       output = [4, 2, 5], go right (null)
Pop 1:       output = [4, 2, 5, 1], go right → push 3
Pop 3:       output = [4, 2, 5, 1, 3]
```

```java
List<Integer> inorderIterative(TreeNode root) {
    List<Integer> result = new ArrayList<>();
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode current = root;

    while (current != null || !stack.isEmpty()) {
        // Go as far left as possible, pushing nodes onto stack
        while (current != null) {
            stack.push(current);
            current = current.left;
        }

        // Pop the top node, add its value, then explore right subtree
        current = stack.pop();
        result.add(current.val);
        current = current.right;
    }

    return result;
}
```

## BFS Level Order Traversal (Using Queue)

> Visits all nodes at depth d before any node at depth d+1.
> Uses a queue. Process all nodes at current level, then enqueue their children.

```
Level order for tree [1, 2, 3, 4, 5]:

Level 0: [1]
Level 1: [2, 3]
Level 2: [4, 5]

Result: [[1], [2, 3], [4, 5]]
```

```java
List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Deque<TreeNode> queue = new ArrayDeque<>();
    queue.add(root);

    while (!queue.isEmpty()) {
        int levelSize = queue.size();
        List<Integer> currentLevel = new ArrayList<>(levelSize);

        for (int i = 0; i < levelSize; i++) {
            TreeNode node = queue.poll();
            currentLevel.add(node.val);

            // Enqueue children for next level
            if (node.left != null) queue.add(node.left);
            if (node.right != null) queue.add(node.right);
        }

        result.add(currentLevel);
    }

    return result;
}
```
