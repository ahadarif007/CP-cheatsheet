# Sliding Window

Sliding window maintains a valid **subarray/substring** by moving two pointers (`left`, `right`).
Expand `right` to include elements, shrink `left` when a constraint is violated.

```
  String: "a b c a b c b b"
  Window:     [c a b]         <- no duplicates, length = 3
          left^     ^right

  When 'c' appears again at right:
              [a b c]         <- shrink left past old 'c'
           left^     ^right
```

**Time:** O(n) - each element is added and removed at most once.

## Longest Substring Without Repeating Characters

```java
int lengthOfLongestSubstring(String text) {
    // lastSeen[ch] = last index where character ch appeared
    int[] lastSeen = new int[256];
    Arrays.fill(lastSeen, -1);

    int maxLength = 0, left = 0;
    for (int right = 0; right < text.length(); right++) {
        char ch = text.charAt(right);
        // If ch was seen within current window, shrink window
        if (lastSeen[ch] >= left) {
            left = lastSeen[ch] + 1;
        }
        lastSeen[ch] = right;
        maxLength = Math.max(maxLength, right - left + 1);
    }
    return maxLength;
}
```

## Fruits Into Baskets (At Most 2 Distinct Types)

Collect fruits from a row of trees, but you can only carry **at most 2 types**.
Find the longest contiguous subarray with at most 2 distinct values.

```
  Fruits: [1, 2, 3, 2, 2]
  Window: [1, 2]        -> 2 types, length = 2
          [2, 3, 2, 2]  -> 2 types, length = 4  (best)
```

```java
int totalFruit(int[] fruits) {
    // Track count of each fruit type in current window
    Map<Integer, Integer> fruitCount = new HashMap<>();
    int left = 0, maxLength = 0;

    for (int right = 0; right < fruits.length; right++) {
        // Add right fruit to window
        fruitCount.put(fruits[right],
            fruitCount.getOrDefault(fruits[right], 0) + 1);

        // Shrink window until we have at most 2 types
        while (fruitCount.size() > 2) {
            int leftFruit = fruits[left++];
            fruitCount.put(leftFruit, fruitCount.get(leftFruit) - 1);
            if (fruitCount.get(leftFruit) == 0) {
                fruitCount.remove(leftFruit);
            }
        }
        maxLength = Math.max(maxLength, right - left + 1);
    }
    return maxLength;
}
```
