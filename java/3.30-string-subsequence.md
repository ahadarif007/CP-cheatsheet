# String Subsequence

## Shortest Way to Form String (Minimum Subsequence Copies)

Find the minimum number of times you need to use `source` as a subsequence to build `target`.
If any character in `target` doesn't exist in `source`, return -1.

```
  source = "abc", target = "abcbc"

  Pass 1: match "abc" from source   -> covered "abc"
  Pass 2: match "bc" from source    -> covered "bc"
  Total: 2 passes

  Strategy: Use binary search to find the next occurrence of each target
  character in source, starting from the current position.
```

**Optimization:** Precompute positions of each character in `source` for O(log n) lookups.

```java
int shortestWay(String source, String target) {
    // Precompute: for each character, store sorted list of indices in source
    Map<Character, List<Integer>> charPositions = new HashMap<>();
    for (int i = 0; i < source.length(); i++) {
        charPositions.computeIfAbsent(source.charAt(i), key -> new ArrayList<>()).add(i);
    }

    int passCount = 1;
    int sourceIndex = -1; // Current position in source (-1 = before start)

    for (char ch : target.toCharArray()) {
        // Character not in source at all -> impossible
        if (!charPositions.containsKey(ch)) return -1;

        List<Integer> positions = charPositions.get(ch);

        // Binary search for next position >= sourceIndex
        int searchResult = Collections.binarySearch(positions, sourceIndex);
        int nextPos = searchResult < 0 ? -searchResult - 1 : searchResult;

        if (nextPos == positions.size()) {
            // No valid position found -> start a new pass through source
            passCount++;
            sourceIndex = positions.get(0) + 1;
        } else {
            sourceIndex = positions.get(nextPos) + 1;
        }
    }
    return passCount;
}
```
