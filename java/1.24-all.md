# All (allMatch)

> `allMatch()` returns `true` only if **every** element satisfies the condition.
> Short-circuits — stops as soon as one element fails the check.
> Equivalent to Python's `all()`.

```java
int[] nums = {0, 2, 4, 6};

// Check if all elements are non-negative
boolean allNonNegative = Arrays.stream(nums)
    .allMatch(x -> x >= 0);
// true (every element is >= 0)

int[] mixed = {0, -1, 4, 6};
boolean allPositive = Arrays.stream(mixed)
    .allMatch(x -> x > 0);
// false (0 and -1 fail the check)
```
