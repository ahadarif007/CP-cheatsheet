# Lambda

> Lambdas are anonymous functions, commonly used with streams and comparators.
> Syntax: `(parameters) -> expression` or `(parameters) -> { statements; }`
> Method references (`Class::method`) are shorthand for simple lambdas.

## Find Closest Value to Target

```java
int target = 10;
int[] candidates = {3, 7, 12, 15};

// Find the value closest to target using stream + custom comparator
int closest = Arrays.stream(candidates)
    .boxed()
    .min(Comparator.comparingInt(x -> Math.abs(target - x)))
    .get();
// closest = 12 (distance 2, vs 7 with distance 3)
```

## Common Lambda Patterns

```java
// Sort strings by length
list.sort((a, b) -> a.length() - b.length());

// Same thing using method reference
list.sort(Comparator.comparingInt(String::length));

// Filter + transform with stream
List<String> upperLong = words.stream()
    .filter(w -> w.length() > 3)            // keep words longer than 3
    .map(String::toUpperCase)               // convert to uppercase
    .toList();
```
