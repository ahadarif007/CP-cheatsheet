# Continuous Subarrays with Sum K

Count the number of **contiguous subarrays** that sum to exactly `k`.
Uses **prefix sum + HashMap** to find pairs where `prefixSum[j] - prefixSum[i] = k`.

```
  nums = [1, 1, 1], k = 2

  Prefix sums: 0, 1, 2, 3
                   ^  ^
                   |  prefixSum=2, check 2-2=0, found 1 time
                      ^  ^
                      |  prefixSum=3, check 3-2=1, found 1 time

  Subarrays: [1,1] (index 0-1), [1,1] (index 1-2) -> answer = 2
```

**Key insight:** If `currentSum - k` was seen before, then the subarray between that point and now sums to `k`.

```java
int subarraySum(int[] nums, int targetSum) {
    // prefixCount[sum] = how many times this prefix sum has occurred
    Map<Integer, Integer> prefixCount = new HashMap<>();
    prefixCount.put(0, 1); // Empty prefix has sum 0

    int count = 0, runningSum = 0;
    for (int num : nums) {
        runningSum += num;
        // If (runningSum - targetSum) was seen, those subarrays sum to targetSum
        count += prefixCount.getOrDefault(runningSum - targetSum, 0);
        prefixCount.put(runningSum, prefixCount.getOrDefault(runningSum, 0) + 1);
    }
    return count;
}
```
