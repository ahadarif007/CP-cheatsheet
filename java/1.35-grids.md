# Grids (2D Array Traversal)

> Grids are 2D arrays. Common in flood-fill, island counting, and pathfinding.
> Use a directions array to move up/down/left/right from any cell.
> Mark visited cells to avoid infinite loops (set to a sentinel value like 2).

## Grid DFS (Flood Fill)

```
Grid (1 = land, 0 = water):
  ┌───┬───┬───┬───┐
  │ 1 │ 1 │ 0 │ 0 │
  ├───┼───┼───┼───┤
  │ 1 │ 0 │ 0 │ 1 │
  ├───┼───┼───┼───┤
  │ 0 │ 0 │ 1 │ 1 │
  └───┴───┴───┴───┘

4 directions from cell (r, c):
  Up:    (r-1, c)
  Down:  (r+1, c)
  Left:  (r, c-1)
  Right: (r, c+1)

DFS from (0,0) marks all connected 1s as visited (set to 2):
  Start (0,0)=1 → mark 2, go right
  (0,1)=1 → mark 2, go down
  (1,0)=1 → mark 2, no more connected 1s
```

```java
int rows = grid.length;
int cols = grid[0].length;

// 4-directional movement: down, up, right, left
int[][] directions = {{1, 0}, {-1, 0}, {0, 1}, {0, -1}};

void dfs(int row, int col) {
    // Mark current cell as visited
    grid[row][col] = 2;

    // Explore all 4 neighbors
    for (int[] dir : directions) {
        int newRow = row + dir[0];
        int newCol = col + dir[1];

        // Check bounds and if neighbor is unvisited land
        boolean inBounds = newRow >= 0 && newRow < rows
                        && newCol >= 0 && newCol < cols;
        if (inBounds && grid[newRow][newCol] == 1) {
            dfs(newRow, newCol);
        }
    }
}
```
