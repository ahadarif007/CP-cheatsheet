# Candy Crush (Remove K Adjacent Duplicates)

Remove groups of `k` **consecutive identical characters** repeatedly until no more removals are possible.
Use a **stack of [character, count]** pairs.

```
  s = "deeedbbcccbdaa", k = 3

  Process each character:
    d(1) e(1) e(2) e(3)-> pop!  d(1) b(1) b(2) c(1) c(2) c(3)-> pop!
    b(2) b(3)-> pop!  d(1) d(2) a(1) a(2)

  Result: "aa"

  Stack trace (char, count):
  [(d,1)] -> [(d,1),(e,1)] -> [(d,1),(e,2)] -> [(d,1),(e,3)] pop -> [(d,1)]
  -> [(d,1),(b,1)] -> [(d,1),(b,2)] -> ... -> [(d,1),(d,2),(a,1),(a,2)]
```

```java
String removeDuplicates(String text, int k) {
    // Stack entries: [character, consecutiveCount]
    Deque<int[]> stack = new ArrayDeque<>();

    for (char ch : text.toCharArray()) {
        if (!stack.isEmpty() && stack.peek()[0] == ch) {
            stack.peek()[1]++;          // Same char: increment count
            if (stack.peek()[1] == k) {
                stack.pop();            // Reached k consecutive: remove group
            }
        } else {
            stack.push(new int[]{ch, 1}); // New char: start count at 1
        }
    }

    // Reconstruct string from stack (bottom to top)
    StringBuilder result = new StringBuilder();
    while (!stack.isEmpty()) {
        int[] entry = stack.removeLast(); // Remove from bottom
        for (int i = 0; i < entry[1]; i++) {
            result.append((char) entry[0]);
        }
    }
    return result.toString();
}
```
