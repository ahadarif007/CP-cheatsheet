# Shift Array Right (Rotate)

Rotate an array right by `k` positions **in-place** using the **three-reversal trick**.

```
  nums = [1, 2, 3, 4, 5, 6, 7], k = 3

  Step 1: Reverse entire array
          [7, 6, 5, 4, 3, 2, 1]

  Step 2: Reverse first k elements
          [5, 6, 7, 4, 3, 2, 1]
           ^^^^^^^

  Step 3: Reverse remaining elements
          [5, 6, 7, 1, 2, 3, 4]
                     ^^^^^^^^^
```

**Time:** O(n), **Space:** O(1)

```java
void rotate(int[] nums, int k) {
    int length = nums.length;
    k %= length; // Handle k > length
    reverseRange(nums, 0, length - 1);   // Reverse all
    reverseRange(nums, 0, k - 1);        // Reverse first k
    reverseRange(nums, k, length - 1);   // Reverse the rest
}

void reverseRange(int[] nums, int left, int right) {
    while (left < right) {
        int temp = nums[left];
        nums[left] = nums[right];
        nums[right] = temp;
        left++;
        right--;
    }
}
```
