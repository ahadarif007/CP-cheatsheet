# Math

> `Math.pow()` returns a `double` — cast to `long` or `int` when needed.
> `Math.floorDiv()` and `Math.floorMod()` round toward negative infinity (like Python).
> Java's `/` and `%` truncate toward zero (different from Python for negatives).

```java
// Power: a^b mod m (cast to long to avoid overflow)
long power = (long) Math.pow(2, 10) % 1000000007;
// 1024

// Floor division — rounds toward negative infinity
long quotient = Math.floorDiv(-7, 2);    // -4 (not -3 like -7/2)

// Floor modulo — always non-negative for positive modulus
long remainder = Math.floorMod(-7, 3);   // 2 (not -1 like -7%3)
```

```
Comparison: Java's / vs Math.floorDiv()

   -7 / 2        = -3   (truncates toward zero)
   Math.floorDiv(-7, 2) = -4   (rounds toward -infinity)

   -7 % 3        = -1   (matches Java's /)
   Math.floorMod(-7, 3) =  2   (matches floorDiv)
```

> **Common math operations:**
> ```java
> Math.abs(-5);        // 5
> Math.max(3, 7);      // 7
> Math.min(3, 7);      // 3
> Math.sqrt(16);       // 4.0
> Math.log(Math.E);    // 1.0
> Math.ceil(2.3);      // 3.0
> Math.floor(2.7);     // 2.0
> ```
