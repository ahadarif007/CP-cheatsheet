# Random

> `ThreadLocalRandom` is preferred over `Random` — thread-safe and faster.
> `nextInt(lo, hi)` returns a random integer in range **[lo, hi)** (exclusive end).
> `Collections.shuffle()` randomly reorders a list in-place.

```java
// Random integer in range [lo, hi) — hi is exclusive
int randomNum = ThreadLocalRandom.current().nextInt(1, 100);
// e.g., any value from 1 to 99

// Shuffle a list randomly (Fisher-Yates algorithm internally)
List<Integer> cards = new ArrayList<>(List.of(1, 2, 3, 4, 5));
Collections.shuffle(cards);
// e.g., [3, 1, 5, 2, 4]
```
