# Bisect (Binary Search)

> `Arrays.binarySearch()` finds an element in a **sorted** array.
> Returns the index if found, or `-(insertion point) - 1` if not found.
> For custom lower/upper bounds, write your own binary search.

## Built-in Binary Search

```java
int[] sorted = {1, 3, 5, 7, 9};

int index = Arrays.binarySearch(sorted, 5);
// index = 2 (found at position 2)

int missing = Arrays.binarySearch(sorted, 4);
// missing = -3 → insertion point would be index 2, so -(2)-1 = -3
// To get insertion point: -(missing) - 1 = 2
```

## Lower Bound (First Position >= Target)

> Returns the **leftmost** index where target could be inserted to keep the array sorted.

```
Array:   [1, 2, 2, 2, 5, 7]
Target:  2
          ↑
          index 1 (first element >= 2)
```

```java
int lowerBound(int[] array, int target) {
    int left = 0, right = array.length;
    while (left < right) {
        int mid = (left + right) / 2;
        if (array[mid] < target) {
            left = mid + 1;   // target is to the right
        } else {
            right = mid;      // target is at mid or to the left
        }
    }
    return left;
}
```

## Upper Bound (First Position > Target)

> Returns the index **after the last occurrence** of target.

```
Array:   [1, 2, 2, 2, 5, 7]
Target:  2
                      ↑
                      index 4 (first element > 2)
```

```java
int upperBound(int[] array, int target) {
    int left = 0, right = array.length;
    while (left < right) {
        int mid = (left + right) / 2;
        if (array[mid] <= target) {
            left = mid + 1;   // move right past equal elements
        } else {
            right = mid;      // first element strictly greater
        }
    }
    return left;
}
```

> **Count occurrences**: `upperBound(arr, x) - lowerBound(arr, x)` gives the count of `x` in a sorted array.
