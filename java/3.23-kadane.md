# Kadane's Algorithm

Find the **maximum sum contiguous subarray** in O(n) time.
**Core idea:** At each position, decide whether to extend the previous subarray or start fresh.

```
  nums: [-2, 1, -3, 4, -1, 2, 1, -5, 4]

  At each index, running sum (extend vs. restart):
  idx:    0   1    2   3    4   5   6    7   8
  val:   -2   1   -3   4   -1   2   1   -5   4
  sum:   -2   1   -2   4    3   5   6    1   5
                       ^              ^
                       restart        max = 6

  Maximum subarray: [4, -1, 2, 1] = 6
```

```java
int maxSubArray(int[] nums) {
    int maxSum = nums[0];

    // Modify array in-place: nums[i] becomes max subarray sum ending at i
    for (int i = 1; i < nums.length; i++) {
        // Extend previous subarray only if it adds positive value
        if (nums[i - 1] > 0) {
            nums[i] += nums[i - 1];
        }
        maxSum = Math.max(maxSum, nums[i]);
    }
    return maxSum;
}
```
