# Tuple (Pair)

> Java has no built-in tuple type like Python.
> Use a `record` (Java 16+) to create an immutable pair.
> Records auto-generate `equals()`, `hashCode()`, and `toString()`.

```java
// Define a generic Pair type using a record
record Pair<A, B>(A first, B second) {}

// Create a pair of integers
Pair<Integer, Integer> point = new Pair<>(2, 3);

// Access elements
int x = point.first();   // 2
int y = point.second();  // 3
```

> **Alternative for older Java**: Use `int[]` of size 2, or `Map.Entry`.
>
> ```java
> // Quick pair using an array
> int[] pair = {2, 3};
>
> // Using Map.entry (immutable)
> var entry = Map.entry("key", 42);
> ```
