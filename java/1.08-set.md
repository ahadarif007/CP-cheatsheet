# Set

> A `HashSet` stores **unique elements** — duplicates are ignored.
> `add`, `remove`, and `contains` are all **O(1)** average time.
> Use a set when you need fast membership checks or deduplication.

```java
Set<Integer> seen = new HashSet<>();

seen.add(10);                          // {10}
seen.add(20);                          // {10, 20}
seen.add(10);                          // {10, 20} — duplicate ignored

seen.remove(10);                       // {20}

boolean exists = seen.contains(20);    // true

// Get any one element (no guaranteed order)
Integer anyElement = seen.iterator().next();
```

> **Common use case**: Detecting duplicates in an array
> ```java
> boolean hasDuplicate(int[] nums) {
>     Set<Integer> seen = new HashSet<>();
>     for (int num : nums) {
>         if (!seen.add(num)) return true;  // add() returns false if already present
>     }
>     return false;
> }
> ```
