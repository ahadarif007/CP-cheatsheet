# For-Else Pattern

> Python has `for...else` where the `else` runs only if the loop completes without `break`.
> Java has no `for-else` syntax — simulate it with a boolean flag.

```
Python equivalent:
  for x in range(n):
      if condition:
          break
  else:
      # runs only if loop completed without break

Java pattern:
  Use a flag variable to track whether break was hit.
```

```java
boolean didBreak = false;

for (int i = 0; i < n; i++) {
    if (someCondition(i)) {
        didBreak = true;
        break;
    }
}

if (!didBreak) {
    // This block runs only if the loop completed without breaking
    // (equivalent to Python's for-else)
}
```
