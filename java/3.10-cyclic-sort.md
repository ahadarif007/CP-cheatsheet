# Cyclic Sort

Cyclic sort places each number at its **correct index** in O(n) time, O(1) space.
Works when values are in range `[1, n]` (place value `v` at index `v-1`).

```
  nums: [3, 1, 5, 4, 2]

  Step 1: nums[0]=3, should be at index 2. Swap -> [5, 1, 3, 4, 2]
  Step 2: nums[0]=5, should be at index 4. Swap -> [2, 1, 3, 4, 5]
  Step 3: nums[0]=2, should be at index 1. Swap -> [1, 2, 3, 4, 5]
  Step 4: nums[0]=1, correct position. Move on.
  Done!   [1, 2, 3, 4, 5]
```

**Use cases:** Find missing number, find duplicates, first missing positive.

```java
int[] cyclicSort(int[] nums) {
    int index = 0;
    while (index < nums.length) {
        int correctIndex = nums[index] - 1; // Where this value should go

        if (nums[index] != nums[correctIndex]) {
            // Swap current value to its correct position
            int temp = nums[index];
            nums[index] = nums[correctIndex];
            nums[correctIndex] = temp;
        } else {
            index++; // Already in correct place, move on
        }
    }
    return nums;
}
```
