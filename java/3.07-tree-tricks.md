# Tree Tricks

## Max Ancestor Difference

Find the maximum `|ancestor.val - descendant.val|` in a binary tree.
**Trick:** Pass the running `min` and `max` values down each path. At a leaf, the answer is `|max - min|`.

```
        8
       / \
      3   10
     / \    \
    1   6    14

  Path 8->3->1: min=1, max=8, diff=7
  Path 8->10->14: min=8, max=14, diff=6
  Answer: 7
```

```java
int maxAncestorDiff(TreeNode root) {
    int[] maxDiff = {0};
    dfs(root, Integer.MAX_VALUE, Integer.MIN_VALUE, maxDiff);
    return maxDiff[0];
}

void dfs(TreeNode node, int pathMin, int pathMax, int[] maxDiff) {
    if (node == null) {
        // At a leaf boundary, compute the diff along this root-to-leaf path
        maxDiff[0] = Math.max(maxDiff[0], Math.abs(pathMax - pathMin));
        return;
    }
    int newMin = Math.min(pathMin, node.val);
    int newMax = Math.max(pathMax, node.val);
    dfs(node.left, newMin, newMax, maxDiff);
    dfs(node.right, newMin, newMax, maxDiff);
}
```

## Diameter of Binary Tree

The **diameter** is the longest path between any two nodes (measured in edges).
The longest path through any node = `leftDepth + rightDepth`.

```
        1
       / \
      2   3         Diameter = 3 (path: 4 -> 2 -> 1 -> 3)
     / \
    4   5           At node 2: leftDepth=1, rightDepth=1, path=2
                    At node 1: leftDepth=2, rightDepth=1, path=3
```

```java
int diameterOfBinaryTree(TreeNode root) {
    int[] maxDiameter = {0};
    computeDepth(root, maxDiameter);
    return maxDiameter[0];
}

int computeDepth(TreeNode node, int[] maxDiameter) {
    if (node == null) return 0;

    int leftDepth = computeDepth(node.left, maxDiameter);
    int rightDepth = computeDepth(node.right, maxDiameter);

    // Path through this node = left depth + right depth
    maxDiameter[0] = Math.max(maxDiameter[0], leftDepth + rightDepth);

    // Return depth of this subtree
    return Math.max(leftDepth, rightDepth) + 1;
}
```
