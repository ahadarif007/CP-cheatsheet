# Default Dict (Auto-initializing Map)

Java's equivalent of Python's `defaultdict` is `computeIfAbsent()`.
It automatically creates a default value when a key is accessed for the first time.

```
  computeIfAbsent(key, k -> new ArrayList<>())

  Before:  map = {}
  After:   map = {1: [2]}

  If key exists   -> returns existing value
  If key missing  -> creates new ArrayList, stores it, returns it
```

## Adjacency List / Grouping Pattern

```java
// Group values by key (like defaultdict(list) in Python)
Map<Integer, List<Integer>> adjacencyList = new HashMap<>();

// Auto-creates empty list for key 1, then adds 2 to it
adjacencyList.computeIfAbsent(1, key -> new ArrayList<>()).add(2);
adjacencyList.computeIfAbsent(1, key -> new ArrayList<>()).add(3);
// Result: {1: [2, 3]}
```

## Other Default Patterns

```java
// Default to 0 for counting (like defaultdict(int))
Map<String, Integer> wordCount = new HashMap<>();
wordCount.merge("hello", 1, Integer::sum);

// Default to empty Set (like defaultdict(set))
Map<Integer, Set<Integer>> graph = new HashMap<>();
graph.computeIfAbsent(1, key -> new HashSet<>()).add(2);
```
