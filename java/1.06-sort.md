# Sort

> `Arrays.sort()` sorts primitive arrays in-place (uses dual-pivot quicksort).
> `Collections.sort()` or `list.sort()` sorts lists (uses TimSort).
> Custom ordering uses `Comparator`.

## Array Sorting

```java
int[] nums = {5, 2, 8, 1, 4};

// Sort entire array in ascending order
Arrays.sort(nums);                      // [1, 2, 4, 5, 8]

// Sort only a subrange [fromIndex, toIndex)
Arrays.sort(nums, 0, 3);               // sort first 3 elements only

// Sort String array by length
String[] words = {"banana", "fig", "apple"};
Arrays.sort(words, Comparator.comparingInt(String::length));
// ["fig", "apple", "banana"]
```

## List Sorting

```java
List<Integer> list = new ArrayList<>(List.of(5, 2, 8));

// Natural order
Collections.sort(list);                // [2, 5, 8]

// Sort list of int[] by second element (index 1)
List<int[]> intervals = new ArrayList<>();
intervals.sort((a, b) -> a[1] - b[1]);

// Multi-key sort: by second element, then by first element
intervals.sort(
    Comparator.comparing((int[] a) -> a[1])
              .thenComparing(a -> a[0])
);
```

## Sort Map Entries by Value

```java
Map<String, Integer> freqMap = new HashMap<>();
freqMap.put("apple", 3);
freqMap.put("banana", 1);
freqMap.put("cherry", 5);

// Convert to list and sort by value
List<Map.Entry<String, Integer>> sortedEntries = new ArrayList<>(freqMap.entrySet());
sortedEntries.sort(Map.Entry.comparingByValue());
// banana=1, apple=3, cherry=5
```

## Sort While Keeping Original Indices

> Useful when you need to know where each element was before sorting.

```
Original:   [40, 10, 30, 20]
Indices:    [ 0,  1,  2,  3]

After sort by value:
Values:     [10, 20, 30, 40]
Orig index: [ 1,  3,  2,  0]
```

```java
int[] nums = {40, 10, 30, 20};
int length = nums.length;

// Create [originalIndex, value] pairs
int[][] indexedValues = new int[length][2];
for (int i = 0; i < length; i++) {
    indexedValues[i][0] = i;         // original index
    indexedValues[i][1] = nums[i];   // value
}

// Sort by value (column 1)
Arrays.sort(indexedValues, Comparator.comparingInt(a -> a[1]));

// indexedValues[0] = [1, 10] → value 10 was at original index 1
```
