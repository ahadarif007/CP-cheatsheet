# Reduce

> `reduce()` combines all elements into a single value using an accumulator function.
> Takes an **identity** (initial value) and a **binary operator**.
> Equivalent to Python's `functools.reduce()`.

```
reduce(0, Integer::sum) on [1, 2, 3, 4]:

  Step 1:  0 + 1 = 1   (identity + first element)
  Step 2:  1 + 2 = 3
  Step 3:  3 + 3 = 6
  Step 4:  6 + 4 = 10  → result
```

```java
int[] nums = {1, 2, 3, 4};

// Sum all elements (identity = 0)
int sum = Arrays.stream(nums)
    .reduce(0, Integer::sum);
// 10

// Find product (identity = 1)
int product = Arrays.stream(nums)
    .reduce(1, (a, b) -> a * b);
// 24

// Find max (no identity — returns OptionalInt)
int max = Arrays.stream(nums)
    .reduce(Integer::max)
    .orElse(0);
// 4
```
