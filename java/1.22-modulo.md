# Modulo

> Java's `%` operator can return **negative values** for negative dividends.
> For example: `-7 % 3 = -1` in Java, but in Python it would be `2`.
> Use this helper to always get a non-negative remainder.

```
Java's % vs true modulo:

  -7 % 3 = -1   (Java's %)
  mod(-7, 3) = 2 (mathematical modulo, always non-negative)

  Fix: if result < 0, add the modulus back.
```

```java
// Safe modulo that always returns a non-negative result
int mod(int dividend, int modulus) {
    int remainder = dividend % modulus;
    return remainder < 0 ? remainder + modulus : remainder;
}

// Example usage:
// mod(-7, 3) → (-7 % 3) = -1 → -1 + 3 = 2
// mod(7, 3)  → (7 % 3)  = 1  → 1 (already non-negative)
```

> **Tip**: Java's `Math.floorMod(a, b)` does the same thing:
> ```java
> Math.floorMod(-7, 3);  // 2
> ```
