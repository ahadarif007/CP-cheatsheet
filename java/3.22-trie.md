# Trie (Prefix Tree)

A **Trie** stores strings character by character in a tree structure.
Each path from root to a marked node represents a stored word.
Enables **O(L)** prefix search where L = prefix length.

```
  Words: "apple", "app", "bat"

          root
         /    \
        a      b
        |      |
        p      a
        |      |
        p*     t*     (* = word ends here)
        |
        l
        |
        e*
```

## Basic Trie with Prefix Search

```java
class Trie {
    static class TrieNode {
        Map<Character, TrieNode> children = new HashMap<>();
        String word; // Non-null if a word ends here
    }

    TrieNode root = new TrieNode();

    void insert(String word) {
        TrieNode current = root;
        for (char ch : word.toCharArray()) {
            current = current.children.computeIfAbsent(ch, key -> new TrieNode());
        }
        current.word = word;
    }

    List<String> matchPrefix(String prefix) {
        // Navigate to the prefix node
        TrieNode current = root;
        for (char ch : prefix.toCharArray()) {
            if (!current.children.containsKey(ch)) return List.of();
            current = current.children.get(ch);
        }
        // Collect all words under this prefix
        List<String> results = new ArrayList<>();
        collectWords(current, results);
        return results;
    }

    void collectWords(TrieNode node, List<String> results) {
        if (node.word != null) results.add(node.word);
        for (var child : node.children.values()) {
            collectWords(child, results);
        }
    }
}
```

## Trie with Wildcard Search (`.` matches any character)

Used in problems like "Add and Search Word".

```
  search("b.t") matches "bat", "bit", "but"
  The '.' branches into ALL children at that level.
```

```java
class WordDictionary {
    static class TrieNode {
        Map<Character, TrieNode> children = new HashMap<>();
        boolean isEndOfWord;
    }

    TrieNode root = new TrieNode();

    void addWord(String word) {
        TrieNode current = root;
        for (char ch : word.toCharArray()) {
            current = current.children.computeIfAbsent(ch, key -> new TrieNode());
        }
        current.isEndOfWord = true;
    }

    boolean search(String word) {
        return searchFrom(word, 0, root);
    }

    boolean searchFrom(String word, int index, TrieNode node) {
        if (index == word.length()) return node.isEndOfWord;

        char ch = word.charAt(index);
        if (ch != '.') {
            // Exact character match
            return node.children.containsKey(ch)
                && searchFrom(word, index + 1, node.children.get(ch));
        }

        // Wildcard: try ALL children
        for (var child : node.children.values()) {
            if (searchFrom(word, index + 1, child)) return true;
        }
        return false;
    }
}
```
