# Filter

> `stream().filter()` keeps only elements that match a condition.
> Elements where the predicate returns `false` are removed.
> Equivalent to Python's `filter()` or `[x for x in list if condition]`.

```java
List<Integer> scores = List.of(50, 80, 65, 90, 72, 95);

// Keep only scores above 75
List<Integer> highScores = scores.stream()
    .filter(score -> score > 75)
    .toList();
// [80, 90, 95]

// Chain filter + map: get uppercase names of long words
List<String> words = List.of("hi", "hello", "hey", "wonderful");
List<String> result = words.stream()
    .filter(w -> w.length() > 3)       // keep words with 4+ characters
    .map(String::toUpperCase)          // convert to uppercase
    .toList();
// ["HELLO", "WONDERFUL"]
```
