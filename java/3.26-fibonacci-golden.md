# Fibonacci (Golden Ratio Formula)

Compute the Nth Fibonacci number in **O(1)** time using Binet's formula.

```
  Golden ratio: phi = (1 + sqrt(5)) / 2 = 1.618...

  F(n) = round(phi^n / sqrt(5))

  F(0) = 0
  F(1) = 1
  F(5) = round(1.618^5 / 2.236) = round(4.999) = 5
  F(10) = 55
```

Note: This formula loses precision for large N due to floating-point errors. For large values, use matrix exponentiation or iterative DP instead.

```java
int fibonacci(int n) {
    double goldenRatio = (1 + Math.sqrt(5)) / 2.0;
    return (int) Math.round(Math.pow(goldenRatio, n) / Math.sqrt(5));
}
```
