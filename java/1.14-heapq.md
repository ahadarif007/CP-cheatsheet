# PriorityQueue (Heap)

> `PriorityQueue` is a **min-heap** by default — smallest element comes out first.
> Use `Comparator.reverseOrder()` for a **max-heap**.
> `offer()` and `poll()` are both **O(log n)**, `peek()` is **O(1)**.

```
Min-Heap structure (smallest on top):
         1
        / \
       3   2
      / \
     7   4

poll() → 1 (removes and returns smallest)
```

## Basic Usage

```java
// Min-heap: smallest element first
PriorityQueue<Integer> minHeap = new PriorityQueue<>();

// Max-heap: largest element first
PriorityQueue<Integer> maxHeap = new PriorityQueue<>(Comparator.reverseOrder());

// Heap of int arrays, sorted by first element
PriorityQueue<int[]> pairHeap = new PriorityQueue<>(
    Comparator.comparingInt(a -> a[0])
);
```

## Get Top K Largest Elements from Array

```java
// Using stream + sort (simple but O(n log n))
int[] topThree = Arrays.stream(nums)
    .boxed()
    .sorted(Comparator.reverseOrder())
    .limit(3)
    .mapToInt(x -> x)
    .toArray();
```

## Top K Frequent Elements (Heap Approach)

> Keep a min-heap of size k. When heap exceeds k, remove the least frequent.
> After processing all elements, the heap contains the k most frequent.

```
nums = [1,1,1,2,2,3], k = 2

Frequency: {1:3, 2:2, 3:1}

Process entries with min-heap (size limit k=2):
  Add {3:1} → heap = [{3:1}]
  Add {2:2} → heap = [{3:1}, {2:2}]
  Add {1:3} → heap size > k, poll min → remove {3:1}
              heap = [{2:2}, {1:3}]

Result: [2, 1]  (the 2 most frequent elements)
```

```java
int[] topKFrequent(int[] nums, int k) {
    // Step 1: Count frequencies
    Map<Integer, Integer> frequency = new HashMap<>();
    for (int num : nums) {
        frequency.put(num, frequency.getOrDefault(num, 0) + 1);
    }

    // Step 2: Min-heap ordered by frequency (smallest freq on top)
    PriorityQueue<Map.Entry<Integer, Integer>> minHeap =
        new PriorityQueue<>(Map.Entry.comparingByValue());

    // Step 3: Maintain heap of size k
    for (var entry : frequency.entrySet()) {
        minHeap.offer(entry);
        if (minHeap.size() > k) {
            minHeap.poll();  // remove least frequent
        }
    }

    // Step 4: Extract results from heap
    int[] result = new int[minHeap.size()];
    int i = 0;
    while (!minHeap.isEmpty()) {
        result[i++] = minHeap.poll().getKey();
    }
    return result;
}
```
