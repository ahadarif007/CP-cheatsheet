# List (ArrayList)

> `ArrayList` is a dynamic array — grows automatically as elements are added.
> Access by index is **O(1)**, add/remove at end is **O(1)** amortized.
> Insert/remove at middle is **O(n)** due to shifting.

## Common Operations

```java
// Create a list pre-filled with 5 zeros
List<Integer> filledList = new ArrayList<>(Collections.nCopies(5, 0));
// [0, 0, 0, 0, 0]

// 2D list (list of lists) — used for adjacency lists, grids, etc.
List<List<Integer>> grid = new ArrayList<>();
grid.add(new ArrayList<>());   // add first row
grid.get(0).add(1);            // add element to first row
grid.get(0).add(2);
// grid = [[1, 2]]

// Reverse a list in-place
Collections.reverse(filledList);

// Concatenate two lists into a new list
List<Integer> list1 = List.of(1, 2, 3);
List<Integer> list2 = List.of(4, 5, 6);
List<Integer> combined = new ArrayList<>();
combined.addAll(list1);
combined.addAll(list2);
// combined = [1, 2, 3, 4, 5, 6]
```

```
ArrayList internal structure:
┌───┬───┬───┬───┬───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │   │   │   │   │
└───┴───┴───┴───┴───┴───┴───┴───┘
  0   1   2   3   ← used (size=4)
                  ← unused capacity

add(5) → O(1) amortized (appends at index 4)
add(1, 99) → O(n) (shifts elements right to insert at index 1)
```
