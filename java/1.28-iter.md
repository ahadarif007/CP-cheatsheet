# Iterator

> An `Iterator` provides a way to traverse a collection element by element.
> `hasNext()` checks if more elements exist, `next()` returns the next element.
> Useful when you need to **remove elements during iteration** (via `it.remove()`).

```java
List<Integer> numbers = new ArrayList<>(List.of(1, 2, 3, 4, 5));

// Basic iteration with Iterator
Iterator<Integer> it = numbers.iterator();
while (it.hasNext()) {
    int value = it.next();
    // process value
}

// Remove even numbers safely during iteration
Iterator<Integer> remover = numbers.iterator();
while (remover.hasNext()) {
    int value = remover.next();
    if (value % 2 == 0) {
        remover.remove();  // safe removal during iteration
    }
}
// numbers = [1, 3, 5]
```

> **Warning**: Modifying a list inside a `for-each` loop causes `ConcurrentModificationException`.
> Use an `Iterator` with `remove()` instead.
