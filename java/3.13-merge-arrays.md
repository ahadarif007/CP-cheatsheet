# Merge K Sorted Arrays / Lists

Merge K sorted collections using a **min-heap (PriorityQueue)**.
Always extract the smallest element across all collections, then push its successor.

**Time:** O(N log K) where N = total elements, K = number of arrays.

```
  Arrays: [1, 4, 7], [2, 5, 8], [3, 6, 9]

  Min-Heap state at each step:
  Heap: [1, 2, 3]  -> extract 1, push 4  -> result: [1]
  Heap: [2, 3, 4]  -> extract 2, push 5  -> result: [1, 2]
  Heap: [3, 4, 5]  -> extract 3, push 6  -> result: [1, 2, 3]
  ...continues until all elements merged...
```

## Merge K Sorted Arrays

```java
int[] mergeKSortedArrays(List<int[]> arrays) {
    // Heap entries: [value, arrayIndex, elementIndex]
    PriorityQueue<int[]> minHeap = new PriorityQueue<>(
        Comparator.comparingInt(entry -> entry[0])
    );

    // Initialize heap with the first element of each array
    for (int arrayIdx = 0; arrayIdx < arrays.size(); arrayIdx++) {
        if (arrays.get(arrayIdx).length > 0) {
            minHeap.add(new int[]{
                arrays.get(arrayIdx)[0], // value
                arrayIdx,                // which array
                0                        // position in that array
            });
        }
    }

    List<Integer> result = new ArrayList<>();
    while (!minHeap.isEmpty()) {
        int[] smallest = minHeap.poll();
        result.add(smallest[0]);

        int arrayIdx = smallest[1];
        int nextElementIdx = smallest[2] + 1;

        // Push next element from the same array
        if (nextElementIdx < arrays.get(arrayIdx).length) {
            minHeap.add(new int[]{
                arrays.get(arrayIdx)[nextElementIdx],
                arrayIdx,
                nextElementIdx
            });
        }
    }
    return result.stream().mapToInt(x -> x).toArray();
}
```

## Merge K Sorted Linked Lists

```java
ListNode mergeKLists(ListNode[] lists) {
    PriorityQueue<ListNode> minHeap = new PriorityQueue<>(
        Comparator.comparingInt(node -> node.val)
    );

    // Add head of each non-null list
    for (ListNode head : lists) {
        if (head != null) minHeap.add(head);
    }

    ListNode dummy = new ListNode(0);
    ListNode tail = dummy;

    while (!minHeap.isEmpty()) {
        ListNode smallest = minHeap.poll();
        tail.next = smallest;
        tail = tail.next;

        // Push the next node from the same list
        if (smallest.next != null) {
            minHeap.add(smallest.next);
        }
    }
    return dummy.next;
}
```
