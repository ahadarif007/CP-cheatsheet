# Regular Expression

> `replaceAll()` uses regex patterns to find and replace text.
> `matches()` checks if the entire string matches a pattern.
> For repeated use, compile with `Pattern.compile()` for better performance.

## Common Operations

```java
String text = "hello world 123";

// Remove all vowels
String noVowels = text.replaceAll("[aeiou]", "");
// "hll wrld 123"

// Remove all digits
String noDigits = text.replaceAll("\\d+", "");
// "hello world "

// Check if string is all digits
boolean isNumeric = "12345".matches("\\d+");
// true

// Split by one or more spaces
String[] words = "hello   world  java".split("\\s+");
// ["hello", "world", "java"]

// Extract all numbers from a string
import java.util.regex.*;
Matcher matcher = Pattern.compile("\\d+").matcher("abc123def456");
List<String> numbers = new ArrayList<>();
while (matcher.find()) {
    numbers.add(matcher.group());
}
// ["123", "456"]
```

```
Common regex patterns:
  \\d     → any digit [0-9]
  \\w     → word character [a-zA-Z0-9_]
  \\s     → whitespace (space, tab, newline)
  [aeiou] → any character in the set
  [^aeiou]→ any character NOT in the set
  +       → one or more
  *       → zero or more
  .       → any character
```
