# Slicing

> Java doesn't have Python-style slicing syntax.
> Use `substring()` for strings and `Arrays.copyOfRange()` for arrays.
> Both use **inclusive start, exclusive end** — `[start, end)`.

```
String:  "abcdef"
Index:    0 1 2 3 4 5

substring(2, 4) → "cd"
                    ↑ start=2 (inclusive)
                        ↑ end=4 (exclusive)

Array:   [10, 20, 30, 40, 50]
Index:     0   1   2   3   4

copyOfRange(arr, 2, 5) → [30, 40, 50]
                           ↑ start=2     ↑ end=5
```

```java
String text = "abcdef";
// Extract characters from index 2 to 3 (end is exclusive)
String sub = text.substring(2, 4);  // "cd"

int[] numbers = {10, 20, 30, 40, 50};
// Copy elements from index 2 to 4 (end is exclusive)
int[] subArray = Arrays.copyOfRange(numbers, 2, 5);  // [30, 40, 50]
```
