## Bisect

The `bisect` module provides binary search on sorted lists. It finds where to INSERT an element to maintain sorted order. This gives O(log n) search in sorted data.

- `bisect_left`: insert BEFORE existing equal elements (leftmost valid position)
- `bisect_right` (or just `bisect`): insert AFTER existing equal elements (rightmost valid position)

```python
import bisect
# Given sorted list:          [1, 2, 3, 4, 5]
# Indices:                      0  1  2  3  4

bisect.bisect_left([1,2,3,4,5], 2)   # 1 (insert BEFORE the existing 2, at index 1)
bisect.bisect_right([1,2,3,4,5], 2)  # 2 (insert AFTER the existing 2, at index 2)
bisect.bisect_left([1,2,3,4,5], 7)   # 5 (would go at end - past all elements)
bisect.bisect_right([1,2,3,4,5], 7)  # 5 (same result when element not present)
```

**insort** - insert while maintaining sorted order. Combines bisect + insert in one call:

```python
import bisect
l = [1, 3, 7, 5, 6, 4, 9, 8, 2]
result = []
for e in l:
    bisect.insort(result, e)   # insert e in sorted position
print(result)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
# Note: search is O(log n) but insert is O(n) due to shifting elements
```

```python
li1 = [1, 3, 4, 4, 4, 6, 7]
bisect.insort(li1, 5)   # inserts 5 in correct position
# Result: [1, 3, 4, 4, 4, 5, 6, 7]
```

**Find range of a target in sorted array** - very common interview pattern:

```python
# Find first and last occurrence of target in sorted array
s = bisect.bisect_left(nums, target)   # first position where target could be
e = bisect.bisect(nums, target) - 1    # last position of target (right boundary - 1)
if s <= e:       # target exists in array
    return [s, e]
else:            # target not found
    return [-1, -1]
```
