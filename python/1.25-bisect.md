## Bisect

The `bisect` module provides binary search on sorted lists. It finds where to INSERT an element to maintain sorted order. This gives O(log n) search in sorted data.

```
bisect_left(arr, x)  → insert BEFORE existing equal elements
bisect_right(arr, x) → insert AFTER  existing equal elements

Example: arr = [1, 2, 2, 2, 5],  x = 2

Index:          0  1  2  3  4
Array:         [1, 2, 2, 2, 5]
                   ↑        ↑
            bisect_left=1   bisect_right=4
            (before first 2) (after last 2)
```

```python
import bisect

bisect.bisect_left([1,2,3,4,5], 2)   # 1 (insert BEFORE the existing 2)
bisect.bisect_right([1,2,3,4,5], 2)  # 2 (insert AFTER the existing 2)
bisect.bisect_left([1,2,3,4,5], 7)   # 5 (would go at end)
bisect.bisect_left([1,2,3,4,5], 0)   # 0 (would go at start)
```

**insort** - insert while maintaining sorted order:

```python
import bisect
l = [1, 3, 7, 5, 6, 4, 9, 8, 2]
result = []
for e in l:
    bisect.insort(result, e)   # insert e in sorted position
print(result)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
# Note: search is O(log n) but insert is O(n) due to shifting elements
```

**Find range of a target in sorted array** - common interview pattern:

```
nums = [1, 3, 5, 5, 5, 7, 9],  target = 5

Index:    0  1  2  3  4  5  6
Array:   [1, 3, 5, 5, 5, 7, 9]
               ↑        ↑
          bisect_left=2  bisect_right=5
          first = 2      last = 5-1 = 4   → answer: [2, 4]
```

```python
s = bisect.bisect_left(nums, target)   # first position of target
e = bisect.bisect(nums, target) - 1    # last position of target
if s <= e:       # target exists
    return [s, e]
else:            # target not found
    return [-1, -1]
```

**Count occurrences in sorted array:**

```python
count = bisect.bisect_right(nums, x) - bisect.bisect_left(nums, x)
```

**Find leftmost value >= x  (lower bound):**

```python
i = bisect.bisect_left(nums, x)
if i < len(nums):
    return nums[i]    # smallest element >= x
```

**Find rightmost value <= x  (upper bound):**

```python
i = bisect.bisect_right(nums, x) - 1
if i >= 0:
    return nums[i]    # largest element <= x
```
