## Map

`map(function, iterable)` applies a function to every element and returns an iterator of results.

```
Input:    ['a', 'b', 'c']
              ↓     ↓     ↓       map(str.upper, ...)
Output:   ['A', 'B', 'C']
```

```python
my_pets = ['alfred', 'tabitha', 'william']
uppered = list(map(str.upper, my_pets))
# ['ALFRED', 'TABITHA', 'WILLIAM']

# With lambda and multiple iterables (pairs up elements):
strings = ['a', 'b', 'c']
numbers = [1, 2, 3]
results = list(map(lambda x, y: (x, y), strings, numbers))
# [('a', 1), ('b', 2), ('c', 3)]
```

**Most common CP uses:**

```python
# Read space-separated integers from input
a, b, c = map(int, input().split())

# Convert list of ints to string for output
nums = [1, 4, 9]
''.join(map(str, nums))       # '149'
' '.join(map(str, nums))      # '1 4 9'

# Apply a transformation to all elements
lengths = list(map(len, ["hi", "hello", "hey"]))   # [2, 5, 3]
```
