## Map

`map(function, iterable)` applies a function to every element and returns an iterator of results. It's like a loop + transformation in one line.

```python
# Apply str.upper to each element
my_pets = ['alfred', 'tabitha', 'william', 'arla']
uppered_pets = list(map(str.upper, my_pets))
# ['ALFRED', 'TABITHA', 'WILLIAM', 'ARLA']

# Map with lambda and multiple iterables - pairs up elements
my_strings = ['a', 'b', 'c', 'd', 'e']
my_numbers = [1, 2, 3, 4, 5]
results = list(map(lambda x, y: (x, y), my_strings, my_numbers))
# [('a', 1), ('b', 2), ('c', 3), ('d', 4), ('e', 5)]
```

**Convert list of ints to string** - very common for joining numbers:

```python
A1 = [1, 4, 9]
''.join(map(str, A1))   # '149' (convert each int to str, then join)
```
