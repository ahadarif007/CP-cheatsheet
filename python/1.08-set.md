## Set

A set is an unordered collection with no duplicate elements. It provides O(1) average time for add, remove, and lookup operations. Sets are perfect for "have I seen this before?" checks.

```python
a = 3
st = set()          # create empty set (note: {} creates an empty dict, not set!)
st.add(a)           # add element: st = {3}
st.remove(a)        # remove element (raises KeyError if not found!)
st.discard(a)       # remove element (does NOTHING if not found - safer!)
st.add(a)           # add again: st = {3}
next(iter(s))       # peek at one element without removing it
                    # iter() creates an iterator, next() gets first item
st.pop()            # remove and return an arbitrary element (set is unordered!)
```

**Set operations** - very useful for comparing collections:

**Set comprehension and frozenset:**

```python
evens = {x for x in range(10) if x % 2 == 0}  # {0, 2, 4, 6, 8}

# frozenset is an immutable, hashable set - can be used as dict key or in another set
fs = frozenset([1, 2, 3])
d = {fs: "value"}                # frozenset as dict key (regular set can't do this)
set_of_sets = {frozenset([1,2]), frozenset([3,4])}  # set containing sets
```

**Subset and superset checks:**

```python
{1, 2}.issubset({1, 2, 3})       # True  (all elements of left are in right)
{1, 2, 3}.issuperset({1, 2})     # True  (right is contained in left)
{1, 2} <= {1, 2, 3}              # True  (operator form of issubset)
```

```python
s = set('abc')        # {'c', 'a', 'b'} (order may vary - sets are unordered)

# UNION (|=): combine both sets - all elements from either set
s |= set('cdf')      # {'f', 'a', 'b', 'd', 'c'} (c was already there, no duplicate)

# INTERSECTION (&=): keep only elements that appear in BOTH sets
s &= set('bd')       # {'d', 'b'} (only b and d are in both)

# DIFFERENCE (-=): remove all elements that appear in the other set
s -= set('b')        # {'d'} (remove b from our set)

# SYMMETRIC DIFFERENCE (^=): keep elements in one OR the other, but NOT both
s ^= set('abd')     # {'a', 'b'} (d was in both so removed; a,b only in new set)
```
