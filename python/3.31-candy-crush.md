## Candy Crush

Remove adjacent duplicates with a stack. Each stack entry stores [character, consecutive_count]. When count reaches k, pop it (crush!). Much more efficient than string scanning.

```python
def removeDuplicates(self, s: str, k: int) -> str:
    stk = []                    # each entry: [character, consecutive_count]
    for c in s:
        if stk and stk[-1][0] == c:    # same char as top of stack?
            stk[-1][1] += 1            # increment its count
            if stk[-1][1] >= k:        # reached k consecutive? CRUSH!
                stk.pop()              # remove entirely
        else:
            stk.append([c, 1])         # new character, start counting at 1
    # Rebuild string from remaining stack entries
    ans = []
    for c in stk:
        ans.extend([c[0]] * c[1])      # repeat character by its count
    return "".join(ans)
```
