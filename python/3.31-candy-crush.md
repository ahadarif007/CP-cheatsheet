## Candy Crush

Remove groups of k adjacent duplicates using a stack. Each stack entry tracks `[character, count]`. When count reaches k, pop it (crush!).

```
Example: s = "deeedbbcccbdaa", k = 3

Process each character:
  d -> [d,1]
  e -> [d,1][e,1]
  e -> [d,1][e,2]
  e -> [d,1][e,3] -> CRUSH! -> [d,1]
  d -> [d,2]
  b -> [d,2][b,1]
  b -> [d,2][b,2]
  c -> [d,2][b,2][c,1]
  c -> [d,2][b,2][c,2]
  c -> [d,2][b,2][c,3] -> CRUSH! -> [d,2][b,2]
  b -> [d,2][b,3] -> CRUSH! -> [d,2]
  d -> [d,3] -> CRUSH! -> []
  a -> [a,1]
  a -> [a,2]

Result: "aa"
```

```python
def removeDuplicates(self, s: str, k: int) -> str:
    stk = []                           # each entry: [char, count]
    for c in s:
        if stk and stk[-1][0] == c:
            stk[-1][1] += 1
            if stk[-1][1] >= k:        # reached k consecutive -> crush!
                stk.pop()
        else:
            stk.append([c, 1])
    return "".join(c * n for c, n in stk)
```
