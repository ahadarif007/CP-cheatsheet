## Grids

Grid problems (2D arrays) are very common. This helper pattern gives you the 4 valid neighbors of a cell and a DFS function for flood-fill style problems:

```python
R = len(grid)        # number of rows
C = len(grid[0])     # number of columns

def neighbors(r, c):
    """Yield all valid 4-directional neighbors (up, down, left, right)"""
    for nr, nc in ((r,c-1), (r,c+1), (r-1,c), (r+1,c)):  # left, right, up, down
        if 0 <= nr < R and 0 <= nc < C:   # bounds check: must be inside grid
            yield nr, nc                   # yield = return one at a time (generator)

def dfs(r, c, index):
    """Flood fill: mark all connected 1s with 'index' and return area"""
    area = 0
    grid[r][c] = index   # mark as visited by changing value (avoids extra visited set!)
    for x, y in neighbors(r, c):
        if grid[x][y] == 1:        # only visit unvisited land cells
            area += dfs(x, y, index)  # recursively visit neighbor
    return area + 1        # +1 counts the current cell
```
