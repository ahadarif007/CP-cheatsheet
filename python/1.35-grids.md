## Grids

Grid problems (2D arrays) are very common. Master these patterns.

**Grid coordinate system:**

```
        col 0  col 1  col 2  col 3
row 0  [ 1,     2,     3,     4  ]
row 1  [ 5,     6,     7,     8  ]
row 2  [ 9,    10,    11,    12  ]

Access: grid[row][col]
R = len(grid)      # number of rows
C = len(grid[0])   # number of columns
```

**4-directional neighbors** (up, down, left, right):

```
            (r-1, c)
               ↑
  (r, c-1) ← (r,c) → (r, c+1)
               ↓
            (r+1, c)
```

```python
R, C = len(grid), len(grid[0])

def neighbors(r, c):
    """Yield all valid 4-directional neighbors"""
    for nr, nc in ((r-1,c), (r+1,c), (r,c-1), (r,c+1)):
        if 0 <= nr < R and 0 <= nc < C:
            yield nr, nc
```

**8-directional neighbors** (includes diagonals):

```
  (r-1,c-1)  (r-1,c)  (r-1,c+1)
          ↖     ↑     ↗
  (r,c-1) ← (r, c) → (r, c+1)
          ↙     ↓     ↘
  (r+1,c-1)  (r+1,c)  (r+1,c+1)
```

```python
def neighbors8(r, c):
    """Yield all valid 8-directional neighbors"""
    for dr in (-1, 0, 1):
        for dc in (-1, 0, 1):
            if dr == 0 and dc == 0:
                continue
            nr, nc = r + dr, c + dc
            if 0 <= nr < R and 0 <= nc < C:
                yield nr, nc
```

**DFS flood fill** - mark connected components:

```python
def dfs(r, c, index):
    """Mark all connected 1s with 'index' and return area"""
    grid[r][c] = index   # mark visited (avoids extra visited set)
    area = 1
    for nr, nc in neighbors(r, c):
        if grid[nr][nc] == 1:
            area += dfs(nr, nc, index)
    return area
```

**BFS on grid** - shortest path in unweighted grid:

```python
from collections import deque

def bfs(sr, sc):
    """Find shortest distance from (sr, sc) to all reachable cells"""
    dist = [[-1]*C for _ in range(R)]
    dist[sr][sc] = 0
    q = deque([(sr, sc)])
    while q:
        r, c = q.popleft()
        for nr, nc in neighbors(r, c):
            if dist[nr][nc] == -1 and grid[nr][nc] != '#':
                dist[nr][nc] = dist[r][c] + 1
                q.append((nr, nc))
    return dist
```

**Common grid operations:**

```python
# Create R x C grid filled with 0
grid = [[0] * C for _ in range(R)]

# Transpose a grid (swap rows and columns)
transposed = list(zip(*grid))     # returns tuples
transposed = [list(row) for row in zip(*grid)]  # returns lists

# Rotate grid 90° clockwise
rotated = [list(row) for row in zip(*grid[::-1])]

# Check bounds in one line
def in_bounds(r, c):
    return 0 <= r < R and 0 <= c < C
```
