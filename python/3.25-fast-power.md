## Fast Power

Calculate x^n in O(log n) time instead of O(n). The trick: x^n = (x^(n/2))^2. We repeatedly square the base and halve the exponent.

Example: 2^10 = (2^5)^2 = (2^2 * 2)^2 ... only ~4 multiplications instead of 10!

```python
def myPow(self, x: float, n: int) -> float:
    if n < 0:             # handle negative exponent: x^(-n) = (1/x)^n
        n *= -1
        x = 1/x
    ans = 1
    while n > 0:
        if n % 2 == 1:    # odd exponent: multiply result by current base
            ans = ans * x  # "collect" one factor of x
        x *= x            # square the base: x -> x^2 -> x^4 -> x^8 -> ...
        n = n // 2        # halve the exponent
    return ans
# Example: 2^10: x=2,n=10 -> x=4,n=5 -> ans*=4,x=16,n=2 -> x=256,n=1 -> ans*=256
```
