## Fast Power

Calculate `x^n` in O(log n) time. Idea: repeatedly square the base and halve the exponent.

```
Trace: 2^10

Step    x       n      ans    action
─────   ─────   ────   ────   ──────────────────
  1     2       10     1      n even, just square x
  2     4        5     1      n odd, ans *= 4 -> 4
  3     16       2     4      n even, just square x
  4     256      1     4      n odd, ans *= 256 -> 1024
  5     65536    0     1024   n=0, done!

Result: 2^10 = 1024
```

```python
def myPow(self, x: float, n: int) -> float:
    if n < 0:             # x^(-n) = (1/x)^n
        n *= -1
        x = 1 / x
    ans = 1
    while n > 0:
        if n % 2 == 1:    # odd exponent: collect one factor of x
            ans *= x
        x *= x            # square base: x -> x^2 -> x^4 -> x^8 ...
        n //= 2           # halve exponent
    return ans
```
