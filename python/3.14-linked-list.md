## Linked List

A linked list is a sequence of nodes where each node points to the next. Unlike arrays, insertion/deletion is O(1) but access is O(n).

Key tips:
1. Solutions typically require 3 pointers: current, previous, and next
2. A **dummy head** (prehead) simplifies edge cases - create a fake first node, build list after it, return `dummy.next`

**Reverse a linked list** - the most fundamental linked list operation:

```
  Before:  1 → 2 → 3 → 4 → None
  After:   None ← 1 ← 2 ← 3 ← 4

  Step by step:
    prev=None, node=1:  1.next = None,  prev=1, node=2
    prev=1,    node=2:  2.next = 1,     prev=2, node=3
    prev=2,    node=3:  3.next = 2,     prev=3, node=4
    prev=3,    node=4:  4.next = 3,     prev=4, node=None  ← done! return prev=4
```

```python
def reverseLinkedList(head):
    prev, node = None, head
    while node:
        # Python tuple swap does all three at once:
        # 1. node.next = prev   (point backward)
        # 2. prev = node        (advance prev)
        # 3. node = node.next   (advance node, using OLD next)
        node.next, prev, node = prev, node, node.next
    return prev
```

**Merge two sorted lists** - use dummy head pattern:

```
  l1:  1 → 3 → 5
  l2:  2 → 4 → 6

  dummy → 1 → 2 → 3 → 4 → 5 → 6
          ↑ compare heads, take smaller, advance that pointer
```

```python
def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
    dummy = ListNode(-1)
    prev = dummy

    while l1 and l2:
        if l1.val < l2.val:
            prev.next = l1
            l1 = l1.next
        else:
            prev.next = l2
            l2 = l2.next
        prev = prev.next

    prev.next = l1 or l2          # attach remaining nodes

    return dummy.next
```

**Reverse sublist by values** (when allowed to modify node values):

```python
def reverse(head):
    node = head
    stk = []
    while node:
        if node.data % 2 == 0:
            stk.append(node)
        if node.data % 2 == 1 or node.next is None:
            while len(stk) > 1:
                stk[-1].data, stk[0].data = stk[0].data, stk[-1].data
                stk.pop(0)
                stk.pop(-1)
            stk.clear()
        node = node.next
    return head
```
