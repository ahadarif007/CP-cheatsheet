## Linked List

A linked list is a sequence of nodes where each node points to the next. Unlike arrays, insertion/deletion is O(1) but access is O(n).

Key tips:
1. Solutions typically require 3 pointers: current, previous, and next
2. A **dummy head** (prehead) simplifies edge cases - create a fake first node, build list after it, return `dummy.next`

**Reverse a linked list** - the most fundamental linked list operation:

```python
def reverseLinkedList(head):
    prev, node = None, head        # prev starts as None (new tail), node starts at head
    while node:
        # Python tuple swap does three things simultaneously:
        # 1. node.next = prev (point current node backward)
        # 2. prev = node (advance prev to current)
        # 3. node = node.next (advance to next node - using OLD node.next!)
        node.next, prev, node = prev, node, node.next
    return prev   # prev is now the new head (node is None, past the end)
```

**Reverse using values** (when allowed to modify node values):

```python
def reverse(head):
  node = head
  stk = []                        # collect nodes to reverse
  while node:
    if node.data % 2 == 0:        # even numbers get reversed
      stk.append(node)
    if node.data % 2 == 1 or node.next is None:  # odd number or end = boundary
      while len(stk) > 1:         # reverse values using two-pointer on stack
        stk[-1].data, stk[0].data = stk[0].data, stk[-1].data
        stk.pop(0)
        stk.pop(-1)
      stk.clear()
    node = node.next
  return head
```

**Merge two sorted lists** - use dummy head pattern:

```python
def mergeTwoLists(self, l1: ListNode, l2: ListNode) -> ListNode:
    dummy = ListNode(-1)          # dummy head: simplifies "first node" edge case
    prev = dummy

    while l1 and l2:              # while both lists have nodes
        if l1.val < l2.val:       # take smaller value
            prev.next = l1
            l1 = l1.next
        else:
            prev.next = l2
            l2 = l2.next
        prev = prev.next

    prev.next = l1 if l1 is not None else l2  # attach remaining nodes

    return dummy.next             # skip dummy, return actual head
```
