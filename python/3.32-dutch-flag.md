## Dutch Flag

[Dutch National Flag](https://en.wikipedia.org/wiki/Dutch_national_flag_problem) by Dijkstra -- sort array of 0s, 1s, 2s in one pass with three pointers.

```
Maintain three regions using pointers p0, curr, p2:

  [0, 0, 0, | 1, 1, 1, | ?, ?, ?, | 2, 2, 2]
   ─────────  ─────────  ─────────  ─────────
   all 0s     all 1s     unexamined  all 2s
  ^         ^           ^            ^       ^
  0        p0          curr         p2     end

Rules:
  nums[curr] == 0:  swap with p0, advance both p0 and curr
  nums[curr] == 1:  already in right place, advance curr
  nums[curr] == 2:  swap with p2, shrink p2 (DON'T advance curr -- swapped value is unknown)
```

```python
def sortColors(self, nums: List[int]) -> None:
    p0 = curr = 0
    p2 = len(nums) - 1

    while curr <= p2:
        if nums[curr] == 0:
            nums[p0], nums[curr] = nums[curr], nums[p0]
            p0 += 1
            curr += 1      # safe: swapped value was 0 or 1 (already processed)
        elif nums[curr] == 2:
            nums[curr], nums[p2] = nums[p2], nums[curr]
            p2 -= 1
            # don't advance curr! swapped value is unknown
        else:
            curr += 1      # 1 is in correct middle region
```
