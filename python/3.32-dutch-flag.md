## Dutch Flag

[Dutch National Flag Problem](https://en.wikipedia.org/wiki/Dutch_national_flag_problem) by [Dijkstra](https://en.wikipedia.org/wiki/Edsger_W._Dijkstra) - sort an array of 0s, 1s, and 2s in one pass using three pointers.

The idea: maintain three regions: [0, p0) are 0s, [p0, curr) are 1s, (p2, end] are 2s. Everything in [curr, p2] is unexamined.

```python
def sortColors(self, nums: List[int]) -> None:
    # p0: boundary for 0s (everything before p0 is 0)
    # curr: current element being examined
    # p2: boundary for 2s (everything after p2 is 2)
    p0 = curr = 0
    p2 = len(nums) - 1

    while curr <= p2:
        if nums[curr] == 0:
            nums[p0], nums[curr] = nums[curr], nums[p0]  # swap to 0-region
            p0 += 1        # expand 0-region
            curr += 1      # safe to advance (swapped value is 0 or 1)
        elif nums[curr] == 2:
            nums[curr], nums[p2] = nums[p2], nums[curr]  # swap to 2-region
            p2 -= 1        # expand 2-region
            # DON'T advance curr! Swapped value is unknown, need to examine it
        else:              # nums[curr] == 1
            curr += 1      # 1 is in correct middle region, just advance
```
