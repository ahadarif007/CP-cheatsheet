## Any

`any()` returns True if ANY element in the iterable is truthy. Short-circuits (stops early) on first True.

```python
def any(iterable):
    for element in iterable:
        if element:        # truthy: non-zero, non-empty, non-None
            return True    # found one! stop immediately
    return False           # nothing was truthy

# Examples:
any([0, 0, 1])      # True (1 is truthy)
any([0, '', None])   # False (all falsy)
any(x > 5 for x in [1,2,3,10])  # True (10 > 5)
```
