## Any

`any()` returns True if ANY element in the iterable is truthy. Short-circuits (stops early) on first True.

```
any([F, F, T, F, F])  →  True   (found one truthy)
any([F, F, F, F, F])  →  False  (none truthy)
     ↑  ↑  ↑               ↑
     checks each...         checks all, gives up
```

```python
def any(iterable):
    for element in iterable:
        if element:        # truthy: non-zero, non-empty, non-None
            return True    # found one! stop immediately
    return False           # nothing was truthy

# Basic examples:
any([0, 0, 1])      # True  (1 is truthy)
any([0, '', None])   # False (all falsy)
any([])              # False (empty iterable)

# With generator expression (most common in CP):
any(x > 5 for x in [1, 2, 3, 10])  # True (10 > 5, stops here)
```

**CP patterns:**

```python
# Check if any row has a zero
grid = [[1,2,3], [4,0,6], [7,8,9]]
any(0 in row for row in grid)  # True

# Check if any character is a digit
any(c.isdigit() for c in "abc3ef")  # True

# Check if any neighbor is valid
any(grid[nr][nc] == 1 for nr, nc in neighbors(r, c))
```
