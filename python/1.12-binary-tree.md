## BinaryTree

A binary tree is a tree where each node has at most 2 children (left and right). There are 4 main ways to traverse (visit all nodes):

**DFS (Depth First Search)** - goes deep before wide. Three variants differ only in WHEN you "process" the current node:

- **Preorder** (Root-Left-Right): Process node FIRST, then go left, then right
  - Use case: Create a copy of the tree, serialize tree
- **Inorder** (Left-Root-Right): Go left FIRST, then process node, then right
  - Use case: Get values in sorted order from a BST
- **Postorder** (Left-Right-Root): Go left, go right, then process node LAST
  - Use case: Delete tree (must delete children before parent)

**BFS (Breadth First Search)** - visits level by level (closest nodes first).

**Recursive Postorder** - simplest form to understand recursion:

```python
"""
     1           Visualize the tree:
    / \          - 1 is the root
   2   3         - 2 is left child of 1, 3 is right child of 1
  / \            - 4 is left child of 2, 5 is right child of 2
 4   5
"""
# PostOrder visits: 4, 5, 2, 3, 1 (leaves first, root last)
def postOrder(node):
  if node is None:           # Base case: empty node, stop recursion
    return
  postOrder(node.left)       # Step 1: fully process left subtree
  postOrder(node.right)      # Step 2: fully process right subtree
  print(node.value, end=' ') # Step 3: THEN process current node
```

**Iterative Traversals** - use a stack to simulate recursion. The trick: push nodes with a `visited` flag. When popped with `visited=True`, process it. When `visited=False`, push it back in the correct order.

The KEY insight: since a stack is LIFO (Last In, First Out), you push nodes in REVERSE order of how you want to process them!

```python
# PreOrder: Root-Left-Right (process root first)
def preOrder(tree_root):
  stack = [(tree_root, False)]     # (node, have_we_visited_it_before?)
  while stack:
    node, visited = stack.pop()    # take from top of stack
    if node:
      if visited:
        print(node.value, end=' ') # visited=True: time to process this node
      else:
        # Push in REVERSE order of desired processing: Right, Left, Root
        # (because stack is LIFO, last pushed = first processed)
        stack.append((node.right, False))  # will be processed LAST (rightmost)
        stack.append((node.left, False))   # will be processed SECOND
        stack.append((node, True))         # will be processed FIRST (root)
```

```python
# InOrder: Left-Root-Right (left first, then root, then right)
def inOrder(tree_root):
  stack = [(tree_root, False)]
  while stack:
    node, visited = stack.pop()
    if node:
      if visited:
        print(node.value, end=' ')
      else:
        # Push in REVERSE of Left-Root-Right = Right, Root, Left
        stack.append((node.right, False))  # processed LAST
        stack.append((node, True))         # processed SECOND (root)
        stack.append((node.left, False))   # processed FIRST (left subtree)
```

```python
# PostOrder: Left-Right-Root (children first, root last)
def postOrder(tree_root):
  stack = [(tree_root, False)]
  while stack:
    node, visited = stack.pop()
    if node:
      if visited:
        print(node.value, end=' ')
      else:
        # Push in REVERSE of Left-Right-Root = Root, Right, Left
        stack.append((node, True))         # processed LAST (root)
        stack.append((node.right, False))  # processed SECOND
        stack.append((node.left, False))   # processed FIRST (left subtree)
```

**BFS / Level Order Traversal** - uses a QUEUE (FIFO) instead of stack. Processes all nodes at depth d before any at depth d+1:

```python
from collections import deque

# Simple BFS - just prints all values level by level
def levelOrder(tree_root):
  queue = deque([tree_root])        # deque is efficient for popleft() - O(1)
  while queue:
    node = queue.popleft()          # take from FRONT of queue (FIFO)
    if node:
        print(node.value, end=' ')  # process current node
        queue.append(node.left)     # add children to BACK of queue
        queue.append(node.right)    # they'll be processed after all current-level nodes

# BFS with level grouping - stores values grouped by depth
def levelOrderGrouped(tree_root):
    queue = deque([(tree_root, 0)])  # (node, depth_level)
    rtn = []                         # rtn[i] will hold all values at depth i
    while queue:
        node, depth = queue.popleft()  # FIFO ensures left-to-right order per level
        if node:
            if len(rtn) < depth + 1:   # first time seeing this depth?
                rtn.append([])         # create new list for this level
            rtn[depth].append(node.value)  # add value to its level
            queue.append((node.left, depth+1))
            queue.append((node.right, depth+1))
    print(rtn)  # [[1], [2, 3], [4, 5]] for the example tree
    return rtn

# Recursive BFS (DFS that groups by level) - often cleaner code
def levelOrderRec(tree_root):
    rtn = []

    def helper(node, depth):
        if len(rtn) == depth:        # first node at this depth creates the list
            rtn.append([])
        rtn[depth].append(node.value)
        if node.left:
            helper(node.left, depth + 1)
        if node.right:
            helper(node.right, depth + 1)

    helper(tree_root, 0)
    print(rtn)
    return rtn
```

**Treating any data as a graph for BFS** - BFS isn't just for trees! You can BFS over strings, states, etc. by generating "neighbors" (next possible states):

```python
def removeInvalidParentheses(self, s: str) -> List[str]:
    rtn = []
    v = set()                      # visited set to avoid checking same string twice
    v.add(s)
    if len(s) == 0: return [""]
    while True:
        for n in v:                # check all strings at current "level"
            if self.isValid(n):
                rtn.append(n)
        if len(rtn) > 0: break    # found valid strings! (minimum removals)
        level = set()             # generate next level (remove one more char)
        for n in v:
            for i, c in enumerate(n):
                if c == '(' or c == ')':
                    sub = n[0:i] + n[i + 1:len(n)]  # remove char at index i
                    level.add(sub)                   # set prevents duplicates
        v = level                 # move to next BFS level
    return rtn
```

**Reconstructing binary trees** - important facts:

1. Binary tree could be constructed from preorder and inorder traversal
1. Inorder traversal of BST is an array sorted in the ascending order

**Convert unbalanced BST to balanced BST** - flatten to sorted array, then rebuild:

```python
def balanceBST(self, root: TreeNode) -> TreeNode:
    self.inorder = []

    def getOrder(node):
        if node is None:
            return
        getOrder(node.left)            # go left first
        self.inorder.append(node.val)  # then record value (inorder = sorted for BST!)
        getOrder(node.right)           # then go right

    getOrder(root)  # Now self.inorder is sorted: [1, 2, 3, 4]

    # Build balanced BST: always pick middle element as root
    # This guarantees height = log(n)
    def bst(listTree):
        if not listTree:
            return None
        mid = len(listTree) // 2       # middle index
        root = TreeNode(listTree[mid]) # middle becomes root (balanced!)
        root.left = bst(listTree[:mid])      # left half becomes left subtree
        root.right = bst(listTree[mid+1:])   # right half becomes right subtree
        return root

    return bst(self.inorder)
```
