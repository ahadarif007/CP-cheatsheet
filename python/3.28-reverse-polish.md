## Reverse Polish Notation (RPN)

Evaluate expressions without parentheses. Numbers go on stack; operators pop two operands, compute, push result.

```
Example: ["2", "1", "+", "3", "*"]   -> (2 + 1) * 3 = 9

Token   Stack         Action
─────   ─────         ──────
"2"     [2]           push number
"1"     [2, 1]        push number
"+"     [3]           pop 1,2 -> 2+1=3, push
"3"     [3, 3]        push number
"*"     [9]           pop 3,3 -> 3*3=9, push

Result: 9
```

```python
class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        ops = {'+', '-', '*', '/'}
        stk = []
        for token in tokens:
            if token not in ops:
                stk.append(int(token))
            else:
                b = stk.pop()          # second operand (top)
                a = stk.pop()          # first operand (below)
                if token == '+': stk.append(a + b)
                elif token == '-': stk.append(a - b)
                elif token == '*': stk.append(a * b)
                elif token == '/': stk.append(int(a / b))  # truncate toward zero
        return stk[0]
```
