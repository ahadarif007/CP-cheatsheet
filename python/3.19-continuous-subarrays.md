## Continuous Subarrays with Sum k

Find the total number of contiguous subarrays whose elements sum to k.

**Key insight (prefix sums):** If `prefix[j] - prefix[i] == k`, then subarray `(i, j]` sums to k. At each position, ask: "how many earlier prefix sums equal `current_sum - k`?"

```
Example: nums = [1, 2, 3], k = 3

index:       0    1    2
value:       1    2    3
prefix:   0  1    3    6
               ↑    ↑
               └────┘  prefix[2] - prefix[0] = 3 - 0 = 3 = k  -> subarray [1,2]
                    ↑
                    └──  prefix[2] - prefix[1] = 3 - 0... wait
                         also prefix[1] itself: value 3 = k     -> subarray [3]

Found 2 subarrays that sum to 3: [1,2] and [3]
```

```python
def subarraySum(self, nums: List[int], k: int) -> int:
    seen = {0: 1}           # prefix_sum -> count of times seen
                            # {0: 1} so that prefix_sum == k counts as a match
    count, total = 0, 0
    for n in nums:
        total += n                          # running prefix sum
        count += seen.get(total - k, 0)     # how many earlier prefixes make a valid subarray?
        seen[total] = seen.get(total, 0) + 1
    return count
```
