## Continuous Subarrays with Sum k

Find the total number of contiguous subarrays whose elements sum to k.

**Key insight (prefix sums):** If `prefix_sum[j] - prefix_sum[i] == k`, then the subarray from index `i+1` to `j` sums to k. So for each position, we ask: "how many earlier prefix sums equal `current_sum - k`?"

```python
def subarraySum(self, nums: List[int], k: int) -> int:
    mp = {0: 1}            # prefix_sum -> how many times we've seen it
                            # {0: 1} handles subarrays starting at index 0
                            # (if current sum == k, then sum - k == 0, which we need to find!)
    rtn, total = 0, 0
    for n in nums:
        total += n                       # running prefix sum
        rtn += mp.get(total - k, 0)      # count previous prefix sums that would make valid subarray
        mp[total] = mp.get(total, 0) + 1 # record this prefix sum for future elements
    return rtn
```
