## Filter

`filter(function, iterable)` keeps only elements where the function returns True. It's like a loop with an if-condition.

```python
scores = [66, 90, 68, 59, 76, 60, 88, 74, 81, 65]
over_75 = list(filter(lambda x: x > 75, scores))
# [90, 76, 88, 81] - only scores above 75 kept
```

```python
# Same thing with a named function (more readable for complex logic)
scores = [66, 90, 68, 59, 76, 60, 88, 74, 81, 65]
def is_A_student(score):
    return score > 75
over_75 = list(filter(is_A_student, scores))  # [90, 76, 88, 81]
```

**Filter for palindromes:**

```python
dromes = ("demigod", "rewire", "madam", "freer", "anutforajaroftuna", "kiosk")
palindromes = list(filter(lambda word: word == word[::-1], dromes))
# ['madam', 'anutforajaroftuna']
# word[::-1] reverses the string; palindrome = same forward and backward
```

**Filter with dictionary values:**

```python
# Get all nodes with indegree 0 (no prerequisites)
stk = list(filter(lambda x: degree[x] == 0, degree.keys()))
```
