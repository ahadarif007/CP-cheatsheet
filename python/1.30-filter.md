## Filter

`filter(function, iterable)` keeps only elements where the function returns True.

```
Input:    [66, 90, 68, 59, 76, 88]
               ✓           ✓   ✓     filter(lambda x: x > 75, ...)
Output:       [90,         76, 88]
```

```python
scores = [66, 90, 68, 59, 76, 60, 88, 74, 81, 65]
over_75 = list(filter(lambda x: x > 75, scores))
# [90, 76, 88, 81]
```

**Filter for palindromes:**

```python
words = ("demigod", "rewire", "madam", "freer", "anutforajaroftuna", "kiosk")
palindromes = list(filter(lambda w: w == w[::-1], words))
# ['madam', 'anutforajaroftuna']
```

**CP patterns:**

```python
# Get nodes with indegree 0 (topological sort starting points)
sources = list(filter(lambda x: degree[x] == 0, degree.keys()))

# Remove empty strings
tokens = list(filter(None, "a,,b,,c".split(",")))  # ['a', 'b', 'c']
# filter(None, ...) removes all falsy values (0, '', None, [], {})

# Keep only even numbers
evens = list(filter(lambda x: x % 2 == 0, range(10)))  # [0, 2, 4, 6, 8]
```

**Note:** List comprehensions are often preferred in Python for readability:

```python
# These are equivalent:
list(filter(lambda x: x > 75, scores))
[x for x in scores if x > 75]           # more Pythonic
```
