## Kadane

Kadane's algorithm finds the maximum sum contiguous subarray in O(n) time.

**Core idea:** At each index, decide: extend the previous subarray, or start fresh? If the running sum so far is negative, it can only hurt -- start fresh.

```
Input: [-2,  1, -3,  4, -1,  2,  1, -5,  4]

Step-by-step (running_sum = max(self, self + prev)):

index:    0    1    2    3    4    5    6    7    8
value:   -2    1   -3    4   -1    2    1   -5    4
run:     -2    1   -2    4    3    5    6    1    5
               ^              ←─────────→
         start   prev<0       best subarray [4,-1,2,1] = 6
         fresh   so start
                 fresh

Answer: max(run) = 6, from subarray [4, -1, 2, 1]
```

[Explanation](https://medium.com/@rsinghal757/kadanes-algorithm-dynamic-programming-how-and-why-does-it-work-3fd8849ed73d)

```python
def maxSubArray(self, nums: List[int]) -> int:
    for i in range(1, len(nums)):
        if nums[i-1] > 0:            # previous running sum helps?
            nums[i] += nums[i-1]     # extend: carry it forward
        # otherwise: start fresh (nums[i] stays as-is)
    return max(nums)
```
