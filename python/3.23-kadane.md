## Kadane

Kadane's algorithm finds the maximum sum contiguous subarray in O(n) time.

**Core idea:** At each index, decide: extend the previous subarray, or start fresh here? If the previous running sum is negative, it can only hurt us - start fresh.

Formula: `local_max[i] = max(A[i], A[i] + local_max[i-1])`

[Explanation](https://medium.com/@rsinghal757/kadanes-algorithm-dynamic-programming-how-and-why-does-it-work-3fd8849ed73d)

```python
# input: [-2, 1, -3, 4, -1, 2, 1, -5, 4]
def maxSubArray(self, nums: List[int]) -> int:
    for i in range(1, len(nums)):
        if nums[i-1] > 0:            # previous running sum is positive?
            nums[i] += nums[i-1]     # extend: add previous sum to current
        # if nums[i-1] <= 0: do nothing (start fresh at nums[i])
    # nums is now: [-2, 1, -2, 4, 3, 5, 6, 1, 5]
    # Each position holds: max subarray sum ENDING at that position
    return max(nums)  # 6 (the subarray [4, -1, 2, 1])
```
