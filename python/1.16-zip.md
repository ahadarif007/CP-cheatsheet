## Zip

`zip()` takes multiple iterables and pairs up their elements by position. It stops at the shortest iterable. Think of it like a zipper bringing two sides together.

**Basic usage** - combine two collections element-by-element:

```python
s1 = {2, 3, 1}
s2 = {'b', 'a', 'c'}
list(zip(s1, s2))  # [(1, 'a'), (2, 'c'), (3, 'b')]
# Note: sets are unordered, so pairing may vary!
```

**Parallel iteration** - loop through multiple lists simultaneously:

```python
letters = ['a', 'b', 'c']
numbers = [0, 1, 2]
for l, n in zip(letters, numbers):  # l and n advance together
  print(f'Letter: {l}')  # a, b, c
  print(f'Number: {n}')  # 0, 1, 2
```

**Zip stops at shortest** - if one list is empty, no iterations happen:

```python
letters = ['a', 'b', 'c']
numbers = []
for l, n in zip(letters, numbers):  # zero iterations! (numbers is empty)
  print(f'Letter: {l}')  # never prints
```

**Compare adjacent elements** - `zip(list, list[1:])` pairs each element with its neighbor:

```python
# words = ["hello", "help", "world"]
for a, b in zip(words, words[1:]):      # ("hello","help"), ("help","world")
    for c1, c2 in zip(a, b):            # compare char by char
        print("c1", c1, end=" ")
        print("c2", c2, end="\n")
```

**`zip(*list)` for transposing** - the `*` unpacks the list, passing each element as a separate argument to zip. This effectively transposes rows↔columns:

```
  Original:            zip(*matrix):
  ┌─────────┐          ┌─────────┐
  │ 1  2  3 │  row 0   │ 1  4  7 │  col 0 → row 0
  │ 4  5  6 │  row 1   │ 2  5  8 │  col 1 → row 1
  │ 7  8  9 │  row 2   │ 3  6  9 │  col 2 → row 2
  └─────────┘          └─────────┘
```

```python
a = [[1,2], [3,4]]
# zip(*a) is the same as zip([1,2], [3,4])
# It pairs elements at the same INDEX across all inner lists
test = zip(*a)
print(list(test))  # [(1, 3), (2, 4)]  <- transposed! rows became columns

matrix = [[1,2,3], [4,5,6], [7,8,9]]
test = zip(*matrix)  # zip([1,2,3], [4,5,6], [7,8,9])
print(*test)  # (1,4,7) (2,5,8) (3,6,9)  <- each column extracted
```

**Rotate matrix 90 degrees clockwise** - a classic trick combining reverse + transpose:

```
  Original:      Reverse rows:    Transpose (zip*):
  1  2  3        7  8  9          7  4  1
  4  5  6   →    4  5  6    →     8  5  2       ← 90° clockwise!
  7  8  9        1  2  3          9  6  3
```

```python
# matrix = [[1,2,3],[4,5,6],[7,8,9]]
matrix[:] = [list(t) for t in zip(*matrix[::-1])]
# Result: [[7,4,1],[8,5,2],[9,6,3]]
```

**Iterate through columns of a list of strings:**

```python
strs = ["cir", "car", "caa"]
for i, l in enumerate(zip(*strs)):  # zip(*strs) gives columns
    print(l)
    # ('c', 'c', 'c')  <- all first characters
    # ('i', 'a', 'a')  <- all second characters
    # ('r', 'r', 'a')  <- all third characters
# Useful for: longest common prefix, column-by-column comparison
```

**Traverse matrix diagonals** - elements on the same diagonal have the same `r+c` value:

```
  Matrix with r+c values:        Diagonals (grouped by r+c):

  ┌──────┬──────┬──────┐         r+c=0:  1
  │ 1 (0)│ 2 (1)│ 3 (2)│         r+c=1:  2, 4
  ├──────┼──────┼──────┤         r+c=2:  3, 5, 7
  │ 4 (1)│ 5 (2)│ 6 (3)│         r+c=3:  6, 8, 10
  ├──────┼──────┼──────┤         r+c=4:  9, 11
  │ 7 (2)│ 8 (3)│ 9 (4)│         r+c=5:  12
  ├──────┼──────┼──────┤
  │10 (3)│11 (4)│12 (5)│
  └──────┴──────┴──────┘
```

```python
def printDiagonalMatrix(self, matrix: List[List[int]]) -> bool:
    R = len(matrix)
    C = len(matrix[0])
    tmp = [[] for _ in range(R+C-1)]   # R+C-1 diagonals in any matrix

    for r in range(R):
        for c in range(C):
            tmp[r+c].append(matrix[r][c])  # elements with same r+c are on same diagonal!

    for t in tmp:
        print(*t)
```
