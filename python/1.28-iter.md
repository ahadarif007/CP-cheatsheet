## Iter

`iter()` creates an iterator from any iterable. `next()` gets the next value. Useful when you need manual control over iteration.

```
Iterable:   [apple, banana, cherry]
                ↓
Iterator:   → apple → banana → cherry → StopIteration
            next()   next()    next()    next() raises error
```

```python
mytuple = ("apple", "banana", "cherry")
myit = iter(mytuple)     # create iterator from tuple
print(next(myit))        # "apple"
print(next(myit))        # "banana"
print(next(myit))        # "cherry"
# next(myit)             # raises StopIteration

# Default value to avoid StopIteration error:
next(myit, "done")       # "done" (returns default instead of error)
```

**CP patterns:**

```python
# Read tokens one at a time (useful for parsing)
tokens = iter(["3", "+", "5", "*", "2"])
first = next(tokens)   # "3"

# Iterate two elements at a time using zip
data = [1, 2, 3, 4, 5, 6]
it = iter(data)
pairs = list(zip(it, it))   # [(1, 2), (3, 4), (5, 6)]
```
