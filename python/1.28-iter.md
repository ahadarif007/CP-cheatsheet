## Iter

`iter()` creates an iterator from any iterable (list, tuple, set, dict). `next()` gets the next value from an iterator. Useful when you need manual control over iteration.

```python
mytuple = ("apple", "banana", "cherry")
myit = iter(mytuple)     # create iterator from tuple
print(next(myit))        # "apple" (first element)
print(next(myit))        # "banana" (second element)
# next(myit) again would give "cherry"
# one more next() would raise StopIteration error
```
