## Reservoir Sampling

Randomly select items from a stream of unknown size with uniform probability, using O(1) space.

**How it works:** When we see the k-th matching item, keep it with probability `1/k` (replacing the current pick). Every item ends up with equal probability.

```
Stream of targets at indices: [2, 5, 8, 11]

See 1st match (i=2):  count=1, keep with 1/1 = 100%   -> pick = 2
See 2nd match (i=5):  count=2, keep with 1/2 = 50%    -> pick = 5 (or stays 2)
See 3rd match (i=8):  count=3, keep with 1/3 = 33%    -> pick = 8 (or stays)
See 4th match (i=11): count=4, keep with 1/4 = 25%    -> pick = 11 (or stays)

Final: each index has exactly 1/4 = 25% chance of being picked!
```

```python
def __init__(self, nums):
    self.nums = nums

def pick(self, target):
    result = None
    count = 0
    for i, x in enumerate(self.nums):
        if x == target:
            count += 1
            if random.randint(1, count) == 1:  # probability 1/count
                result = i
    return result
```
