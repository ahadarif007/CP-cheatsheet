## Resevior Sampling

Reservoir sampling randomly selects items from a stream of unknown size with uniform probability, using O(1) space. Each matching item has exactly 1/count chance of being the final selection.

**Why it works:** When we see the kth matching item, we select it with probability 1/k. The probability that item j (seen earlier) is still selected = (1/j) * (1 - 1/(j+1)) * (1 - 1/(j+2)) * ... * (1 - 1/k) = 1/k. All items end up with equal probability!

```python
def __init__(self, nums):
    self.nums = nums
def pick(self, target):
    res = None
    count = 0
    for i, x in enumerate(self.nums):
        if x == target:
            count += 1                          # count matching elements seen so far
            chance = random.randint(1, count)   # random number from 1 to count
            if chance == 1:                     # probability 1/count
                res = i                         # replace current selection
    return res
```
