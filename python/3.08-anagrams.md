## Anagrams

Two strings are anagrams if they contain the same characters with the same frequencies. Key insight: sorting both strings gives the same result, OR their Counters are equal.

```python
# Simple check: are s and t anagrams?
def isAnagram(self, s: str, t: str) -> bool:
    sc = collections.Counter(s)   # count characters in s
    st = collections.Counter(t)   # count characters in t
    if sc != st:                  # counters not equal = different frequencies
        return False
    return True
    # One-liner: return Counter(s) == Counter(t)
```

**Sliding window anagram finder** - find all starting indices of p's anagrams in s:

```python
def findAnagrams(self, s: str, p: str) -> List[int]:
    cntP = collections.Counter(p)    # target character frequencies
    cntS = collections.Counter()     # current window character frequencies
    P = len(p)                       # window size
    S = len(s)
    if P > S:
        return []
    ans = []
    for i, c in enumerate(s):
        cntS[c] += 1                 # add new character to window (right side)
        if i >= P:                   # window is now too big
            if cntS[s[i-P]] > 1:     # remove leftmost character
                cntS[s[i-P]] -= 1
            else:
                del cntS[s[i-P]]     # delete key if count reaches 0 (for clean comparison)
        if cntS == cntP:             # window has same character frequencies as p!
            ans.append(i-(P-1))      # start index of this window
    return ans
```
