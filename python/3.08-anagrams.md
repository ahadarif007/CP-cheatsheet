## Anagrams

Two strings are anagrams if they contain the same characters with the same frequencies. Key insight: sorting both strings gives the same result, OR their Counters are equal.

```
  "listen"  →  Counter: {l:1, i:1, s:1, t:1, e:1, n:1}
  "silent"  →  Counter: {s:1, i:1, l:1, e:1, n:1, t:1}   same! → anagram ✓

  "hello"   →  sorted: "ehllo"
  "world"   →  sorted: "dlorw"     different! → not anagram ✗
```

```python
def isAnagram(self, s: str, t: str) -> bool:
    return collections.Counter(s) == collections.Counter(t)
```

**Sliding window anagram finder** - find all starting indices of p's anagrams in s:

```
  s = "cbaebabacd",  p = "abc"    cntP = {a:1, b:1, c:1}

  window slides with fixed size = len(p) = 3:
  [c b a] e b a b a c d    cntS = {c:1, b:1, a:1} == cntP → index 0 ✓
   c [b a e] b a b a c d   cntS = {b:1, a:1, e:1} ≠ cntP
   ...
   c b a e b a [b a c] d   cntS = {b:1, a:1, c:1} == cntP → index 6 ✓
```

```python
def findAnagrams(self, s: str, p: str) -> List[int]:
    cntP = collections.Counter(p)    # target character frequencies
    cntS = collections.Counter()     # current window frequencies
    P = len(p)
    if P > len(s):
        return []
    ans = []
    for i, c in enumerate(s):
        cntS[c] += 1                         # add new char (right side)
        if i >= P:                           # window too big → remove leftmost
            left_char = s[i - P]
            cntS[left_char] -= 1
            if cntS[left_char] == 0:
                del cntS[left_char]          # delete key for clean comparison
        if cntS == cntP:                     # window matches target!
            ans.append(i - (P - 1))          # start index of this window
    return ans
```
