## Binary Search

Binary search finds a target in O(log n) time by repeatedly halving the search space. The key template: maintain invariant that answer is always in [l, r], shrink range until l == r.

```
How it works:

  [1, 3, 5, 7, 9, 11, 13, 15]     target = 9
   l           m            r      m=7 < 9 → search right half
                  l    m    r      m=11 > 9 → search left half
                  l/m  r           m=9 = 9 → found!

Each step cuts search space in half → O(log n)
```

**Template 1: Find first element satisfying condition**

```python
def firstBadVersion(self, n):
    l, r = 0, n               # search range [l, r]
    while l < r:              # loop until range is a single element
        m = l + (r-l) // 2   # find middle (avoids overflow in other languages)
        if isBadVersion(m):  # condition is True at m
            r = m            # answer could be m or left of m (keep m in range)
        else:                # condition is False at m
            l = m + 1        # answer must be right of m (exclude m)
    return l                 # l == r, converged to the answer
```

```
Condition array:  [F, F, F, F, T, T, T, T]
                               ↑
                         first True = answer

  if condition(m) is True  → r = m      (answer is m or left)
  if condition(m) is False → l = m + 1  (answer is right of m)
```

**Template 2: Custom condition function** - good for "minimize/maximize X" problems:

```python
def mySqrt(self, x: int) -> int:
    """Find largest n where n*n <= x"""
    def condition(value, x) -> bool:
        return value * value > x   # True when value is too big

    if x == 1:
        return 1

    left, right = 1, x
    while left < right:
        mid = left + (right-left) // 2
        if condition(mid, x):      # mid^2 > x, mid is too big
            right = mid
        else:                      # mid^2 <= x, mid might be answer
            left = mid + 1

    return left - 1   # left is first "too big", so left-1 is the answer
```

```
Example: sqrt(8) → answer is 2

  values:     1  2  3  4  5  6  7  8
  condition:  F  F  T  T  T  T  T  T    (value*value > 8?)
                  ↑
            first True at 3, so answer = 3 - 1 = 2
```
