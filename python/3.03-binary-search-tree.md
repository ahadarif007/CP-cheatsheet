## Binary Search Tree

A BST has the property: all left descendants < node < all right descendants. This enables O(log n) search, insert, delete.

**Check if tree is complete** - assign index values (root=1, left=2*i, right=2*i+1). A complete tree has max_index == total_nodes:

```python
def isCompleteTree(self, root: TreeNode) -> bool:
    self.total = 0
    self.mx = float('-inf')
    def dfs(node, cnt):
        if node:
            self.total += 1                # count nodes
            self.mx = max(self.mx, cnt)    # track highest index
            dfs(node.left, (cnt*2))        # left child index = 2*parent
            dfs(node.right, (cnt*2)+1)     # right child index = 2*parent+1
    dfs(root, 1)
    return self.total == self.mx   # in complete tree, max index == node count
```

**Range sum in BST** - use BST property to prune branches we don't need:

```python
def rangeSumBST(self, root: TreeNode, L: int, R: int) -> int:
    self.total = 0
    def helper(node):
        if node is None:
            return 0
        if L <= node.val <= R:       # node in range: add its value
            self.total += node.val
        if node.val > L:             # only go left if current > L (left might have valid nodes)
            left = helper(node.left)
        if node.val < R:             # only go right if current < R (right might have valid nodes)
            right = helper(node.right)
    helper(root)
    return self.total
```

**Validate BST** - each node must be within a valid range (floor, ceil). Pass these bounds down:

```python
def isValidBST(self, root: TreeNode) -> bool:
    if not root:
        return True
    # Stack holds (node, minimum_allowed, maximum_allowed)
    stk = [(root, float(-inf), float(inf))]
    while stk:
        node, floor, ceil = stk.pop()
        if node:
            if node.val >= ceil or node.val <= floor:  # violates BST property!
                return False
            # Right child must be > node.val and < ceil
            stk.append((node.right, node.val, ceil))
            # Left child must be > floor and < node.val
            stk.append((node.left, floor, node.val))
    return True
```
