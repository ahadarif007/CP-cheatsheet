## Binary Search Tree

A BST has the property: all left descendants < node < all right descendants. This enables O(log n) search, insert, delete.

```
BST property:           Indexing for completeness check:

       8                        1
      / \                      / \
     3   10                   2   3
    / \    \                 / \   \
   1   6   14               4   5   7
                     (max_index=7, nodes=6 → NOT complete)

Complete tree:          Index = count:
       8                        1
      / \                      / \
     3   10                   2   3
    / \                      / \
   1   6                    4   5
                     (max_index=5, nodes=5 → complete ✓)
```

**Check if tree is complete** - assign index values (root=1, left=2\*i, right=2\*i+1). A complete tree has max_index == total_nodes:

```python
def isCompleteTree(self, root: TreeNode) -> bool:
    self.total = 0
    self.mx = float('-inf')
    def dfs(node, cnt):
        if node:
            self.total += 1                # count nodes
            self.mx = max(self.mx, cnt)    # track highest index
            dfs(node.left, (cnt*2))        # left child index = 2*parent
            dfs(node.right, (cnt*2)+1)     # right child index = 2*parent+1
    dfs(root, 1)
    return self.total == self.mx   # in complete tree, max index == node count
```

**Range sum in BST** - use BST property to prune branches:

```
          10
         /  \
        5    15           rangeSumBST(root, L=7, R=15)
       / \     \
      3   7    18         node=10: 7 ≤ 10 ≤ 15 → add 10
                          node=5:  5 < 7  → skip value, but go RIGHT (7 might be valid)
                          node=7:  7 ≤ 7 ≤ 15 → add 7
                          node=15: 7 ≤ 15 ≤ 15 → add 15
                          node=18: 18 > 15 → skip, don't go right
                          Answer: 10 + 7 + 15 = 32
```

```python
def rangeSumBST(self, root: TreeNode, L: int, R: int) -> int:
    self.total = 0
    def helper(node):
        if node is None:
            return
        if L <= node.val <= R:       # node in range: add its value
            self.total += node.val
        if node.val > L:             # only go left if current > L
            helper(node.left)
        if node.val < R:             # only go right if current < R
            helper(node.right)
    helper(root)
    return self.total
```

**Validate BST** - each node must be within a valid range (floor, ceil). Pass bounds down:

```
         5  (floor=-∞, ceil=+∞)
        / \
       3   7  left child: (floor=-∞, ceil=5)   right child: (floor=5, ceil=+∞)
      / \
     1   4    node=4: floor=-∞, ceil=3 → 4 > 3 → INVALID!
              Wait... 4 should be ceil=5 since it's right child of 3
              Actually: right child of 3 → (floor=3, ceil=5) → 3 < 4 < 5 → valid ✓
```

```python
def isValidBST(self, root: TreeNode) -> bool:
    if not root:
        return True
    stk = [(root, float('-inf'), float('inf'))]
    while stk:
        node, floor, ceil = stk.pop()
        if node:
            if node.val >= ceil or node.val <= floor:
                return False
            stk.append((node.right, node.val, ceil))   # right: (node.val, ceil)
            stk.append((node.left, floor, node.val))   # left:  (floor, node.val)
    return True
```
