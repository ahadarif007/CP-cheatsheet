## Dict

Dictionaries (dicts) implement hash tables in Python. They provide O(1) average time for get/set/delete operations. Every key must be hashable (immutable types like strings, numbers, tuples).

```python
d = {'key': 'value'}           # Create dict with initial key-value pair
d['key'] = 'value'             # Set a key-value pair (overwrites if key exists)
{x:0 for x in {'a', 'b'}}     # Dict comprehension: {'a': 0, 'b': 0}

# ACCESSING VALUES - two ways:
d['key']                       # Direct access (raises KeyError if key missing!)
d.get('key', default)          # Safe access (returns default if missing, no error)

# USEFUL METHODS:
d.items()                      # Get all (key, value) pairs as tuples
if 'key' in d: print("found")  # Check if key exists - O(1) operation!
par = {}
par.setdefault(1, 1)          # If key 1 missing: set it to 1 AND return 1
                               # If key 1 exists: just return existing value (don't overwrite)
par = {0:True, 1:False}
par.pop(0)                    # Remove key 0 and return its value (True). par = {1: False}

# ITERATION:
for k in d: print(k)          # Iterate through KEYS only
for k, v in d.items():        # Iterate through key-value pairs (most common)
for v in d.values():          # Iterate through VALUES only
```

**Deleting keys and merging dicts:**

```python
d = {'a': 1, 'b': 2, 'c': 3}
del d['b']                       # remove key 'b': {'a': 1, 'c': 3}

# Merge dicts (Python 3.9+)
d1 = {'a': 1, 'b': 2}
d2 = {'b': 3, 'c': 4}
d3 = d1 | d2                    # {'a': 1, 'b': 3, 'c': 4} (d2 wins on conflict)
d3 = {**d1, **d2}               # same result (works in Python 3.5+)
```

**Iterating while modifying** - iterate over a snapshot of keys to avoid RuntimeError:

```python
d = {'a': 1, 'b': 2, 'c': 3}
for k in list(d.keys()):         # list() creates a copy of the keys
    if d[k] < 2:
        del d[k]                 # safe to delete during iteration
# d = {'b': 2, 'c': 3}
```

**defaultdict** - auto-creates missing keys (see 2.3-default-dict):

```python
from collections import defaultdict
graph = defaultdict(list)
graph[0].append(1)               # no need to check if key 0 exists
```

**Create dict of lists** - a pattern for initializing tracking structures:

```python
votes = ["ABC","CBD","BCA"]
# For each character in first vote string, create a list of zeros
# len(votes[0]) = 3, so each candidate gets a list of 3 zeros (one per ranking position)
rnk = {v:[0] * len(votes[0]) for v in votes[0]}
print(rnk)  # {'A': [0, 0, 0], 'B': [0, 0, 0], 'C': [0, 0, 0]}
```
