## Hash

A hash table (dictionary in Python) maps keys to values with O(1) average lookup time. This is the most-used data structure in CP after arrays.

**Frequency counting** - one of the most common patterns:

```python
ht = {}               # empty dictionary to store character counts
for c in s1:          # iterate through each character in the string
  # .get(key, default) returns the value for key if it exists,
  # otherwise returns default (0 here) WITHOUT raising an error
  ht[c] = ht.get(c, 0) + 1  # if char not seen: 0+1=1, if seen before: old_count+1
  # After processing "aab": ht = {'a': 2, 'b': 1}

# Even faster: use collections.Counter (see 2.2-counter)
from collections import Counter
ht = Counter(s1)      # same result in one line
```

**Two-sum pattern** - classic O(n) lookup for complement:

```python
def twoSum(nums, target):
    seen = {}                          # value -> index
    for i, num in enumerate(nums):
        comp = target - num            # what do we need?
        if comp in seen:               # O(1) lookup
            return [seen[comp], i]
        seen[num] = i                  # store for future lookups
```

**Prefix sum + hash map** - count subarrays with a given sum in O(n):

```
  nums = [1, 2, 1, 3],  k = 3

  index:     0    1    2    3
  value:     1    2    1    3
  prefix:    1    3    4    7      ← running total
                                  
  At index 1: prefix=3, prefix-k=0 → seen[0]=1 ✓  (subarray [1,2])
  At index 3: prefix=7, prefix-k=4 → seen[4]=1 ✓  (subarray [3])

  Key idea: if prefix[j] - prefix[i] == k, then sum(nums[i+1..j]) == k
```

```python
def subarraySum(nums, k):
    count = 0
    prefix = 0
    seen = {0: 1}                      # prefix sum 0 occurs once (empty prefix)
    for num in nums:
        prefix += num
        count += seen.get(prefix - k, 0)  # how many times have we seen prefix - k?
        seen[prefix] = seen.get(prefix, 0) + 1
    return count
```

**Grouping with defaultdict** - auto-creates missing keys (see 2.3-default-dict):

```python
from collections import defaultdict
graph = defaultdict(list)              # adjacency list
graph[u].append(v)                     # no KeyError even if u is new
```
