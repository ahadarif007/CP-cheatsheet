## itertools

`itertools` provides efficient iteration utilities. `accumulate` is especially useful - it gives running totals (prefix sums) or running operations.

```python
import itertools
data = [3, 4, 6, 2, 1, 9, 0, 7, 5, 8]

# Default: running sum (prefix sum)
list(itertools.accumulate(data))
# [3, 7, 13, 15, 16, 25, 25, 32, 37, 45]
# 3, 3+4=7, 7+6=13, 13+2=15, ...

# Custom function: running maximum
list(accumulate(data, max))
# [3, 4, 6, 6, 6, 9, 9, 9, 9, 9]
# At each position: the max value seen so far

# Financial example: amortize a loan
cashflows = [1000, -90, -90, -90, -90]
list(itertools.accumulate(cashflows, lambda bal, pmt: bal*1.05 + pmt))
# [1000, 960.0, 918.0, 873.9, 827.6]

# groupby: group consecutive equal elements
from itertools import groupby
for k, v in groupby("aabbbc"):   # groups consecutive same characters
    print(k)          # 'a', 'b', 'c'
    print(list(v))    # ['a','a'], ['b','b','b'], ['c']
# WARNING: groupby only groups CONSECUTIVE elements! Sort first if needed.
```
