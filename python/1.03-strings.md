## Strings

Strings in Python are immutable (cannot be changed in place). Every "modification" creates a new string. This is important for performance - string concatenation in a loop is O(n^2). Use lists and `join()` instead.

**Finding characters** - `.find()` returns the index of first occurrence, or -1 if not found (unlike `.index()` which throws an error):

```python
str1.find('x')    # find first occurrence of 'x', returns index or -1
str1.rfind('x')   # find LAST occurrence of 'x' (searches from right), returns index or -1
```

**Splitting strings** - `.split()` breaks a string into a list of substrings:

```python
l = "0:start:0"
tokens = l.split(":")       # split on colon delimiter
print(tokens)               # ['0', 'start', '0'] - list of 3 strings

# split() with no argument splits on ANY whitespace and removes empty strings
"the sky  is blue".split()  # ['the', 'sky', 'is', 'blue'] (extra spaces handled!)
"the sky  is blue".split(' ')  # ['the', 'sky', '', 'is', 'blue'] (empty string for double space)
```

**Reversing words** - combines split, slice reversal, and join. These three operations together are a very common pattern:

```python
# s = "the sky  is blue"
def reverseWords(self, s: str) -> str:
  wordsWithoutWhitespace = s.split()       # Step 1: split into words (handles multiple spaces)
                                            # Result: ['the', 'sky', 'is', 'blue']
  reversedWords = wordsWithoutWhitespace[::-1]  # Step 2: reverse the list with slice notation
                                                 # [::-1] means start:end:step where step=-1
                                                 # Result: ['blue', 'is', 'sky', 'the']
  final = " ".join(reversedWords)           # Step 3: join list back into string with spaces
                                            # Result: "blue is sky the"
```

**Manual split** - sometimes you need custom splitting logic (e.g., split on non-alpha characters):

```python
def splitWords(input_string) -> list:
  words = []                            # will hold our extracted words
  start = length = 0                    # track where current word starts and its length
  for i, c in enumerate(input_string):  # enumerate gives both index and character
     if c.isalpha():                    # is this character a letter?
       if length == 0:                  # starting a new word
          start = i                     # mark where it begins
          length += 1                   # count this character
     else:                              # non-alpha character (space, number, etc.)
        if length != 0:                 # we were in a word, now it ended
           words.append(input_string[start:start+length])  # slice out the word
           length = 0                   # reset for next word
  if length > 0:                        # don't forget the last word if string doesn't end with non-alpha
    words.append(input_string[start:start+length])
  return words
```

**Character type testing and Caesar cipher** - `ord()` converts char to ASCII number, `chr()` converts number back to char. This is essential for rotating/shifting characters:

```python
def rotationalCipher(input, rotation_factor):
  rtn = []                              # build result as list (faster than string concat)
  for c in input:
    if c.isupper():                     # uppercase letter A-Z
      ci = ord(c) - ord('A')           # convert to 0-25 (A=0, B=1, ..., Z=25)
      ci = (ci + rotation_factor) % 26 # shift and wrap around using modulo
      rtn.append(chr(ord('A') + ci))   # convert back to character
    elif c.islower():                   # lowercase letter a-z
      ci = ord(c) - ord('a')           # convert to 0-25
      ci = (ci + rotation_factor) % 26 # shift and wrap
      rtn.append(chr(ord('a') + ci))   # convert back
    elif c.isnumeric():                 # digit 0-9
      ci = ord(c) - ord('0')           # convert to 0-9
      ci = (ci + rotation_factor) % 10 # shift and wrap (only 10 digits)
      rtn.append(chr(ord('0') + ci))   # convert back
    else:
      rtn.append(c)                    # non-alphanumeric: keep as-is
  return "".join(rtn)                   # join list into final string
```

**Useful string methods:**

```python
isalnum()          # True if alphanumeric (letters and numbers only, no spaces/symbols)
s.count('ab')      # count non-overlapping occurrences of substring 'ab'
s.strip()          # remove leading/trailing whitespace (lstrip/rstrip for one side)
s.lower()          # 'ABC' -> 'abc' (upper() for reverse)
s.isdigit()        # True if all chars are digits: '123' -> True, '-1' -> False
s.isalpha()        # True if all chars are letters
```

**Fast character replacement with translate** - faster than chained `.replace()` calls:

```python
# Remove all vowels from a string
table = str.maketrans('', '', 'aeiou')  # 3rd arg = chars to DELETE
'hello world'.translate(table)          # 'hll wrld'

# Replace characters: a->1, b->2
table = str.maketrans('ab', '12')
'abc'.translate(table)                  # '12c'
```

**Character ↔ number conversion** - `ord()` and `chr()` are inverse functions:

```python
print(ord('A'))              # 65 (ASCII value of 'A')
print(ord('B') - ord('A') + 1)  # 2 (B is the 2nd letter)
print(chr(ord('a') + 2))    # 'c' (a + 2 positions = c)
```

**Replace characters or substrings** - useful for bracket matching by repeatedly removing valid pairs:

```python
def isValid(self, s: str) -> bool:
  # Keep removing valid pairs until none remain
  while '[]' in s or '()' in s or '{}' in s:
    s = s.replace('[]','').replace('()','').replace('{}','')
  return len(s) == 0  # if string is empty, all brackets were matched
```

**String formatting** - two common ways to insert values into strings:

```python
# .format() method - positional placeholders
txt3 = "My name is {}, I'm {}".format("John", 36)  # "My name is John, I'm 36"

# f-strings (Python 3.6+) - embed expressions directly (preferred, more readable)
name = "Eric"
profession = "comedian"
affiliation = "Monty Python"
message = (
     f"Hi {name}. "                    # f before the quote enables {expression} syntax
     f"You are a {profession}. "       # any valid Python expression works inside {}
     f"You were in {affiliation}."
)
# Result: 'Hi Eric. You are a comedian. You were in Monty Python.'
```

**String/list multiplication** - the `*` operator repeats strings and lists:

```python
'meh' * 2        # 'mehmeh' (string repeated twice)
['meh'] * 2      # ['meh', 'meh'] (list repeated twice)
['meh'] * True   # ['meh'] (True = 1, so multiply by 1)
['meh'] * False  # [] (False = 0, so multiply by 0 = empty)
# This boolean trick is useful for conditionally including elements
```

**Find substring** - returns the starting index of a substring:

```python
txt = "Hello, welcome to my world."
x = txt.find("welcome")  # 7 (substring starts at index 7)
```

**startswith / endswith** - cleaner than slicing for checking prefixes/suffixes:

```python
str = "this is string example....wow!!!"
str.endswith("!!")        # True (string ends with "!!")
str.startswith("this")   # True (string starts with "this")
str.endswith("is", 2, 4) # True (checks only str[2:4] = "is")
```

**Debug printing** - `repr()` shows the "raw" string including escape characters:

```python
print(repr("meh\n"))     # 'meh\n' (shows \n literally instead of making a newline)
```
