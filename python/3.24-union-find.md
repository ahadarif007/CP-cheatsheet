## Union Find

Union-Find (Disjoint Set Union / DSU) efficiently tracks which elements belong to the same group. Two operations:
- `find(x)`: find the "root" representative of x's group
- `union(x, y)`: merge the groups containing x and y

**Path compression** in `find`: flatten the tree so future lookups are O(1) amortized.

**DSU for integers:**

```python
class DSU:
    def __init__(self, N):
        self.par = list(range(N))  # initially, each element is its own parent (own group)

    def find(self, x):
        if self.par[x] != x:                  # x is not the root of its group
            self.par[x] = self.find(self.par[x])  # PATH COMPRESSION: point directly to root
        return self.par[x]                    # return root representative

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)  # find roots of both groups
        if xr == yr:                          # already in same group!
            return False
        self.par[yr] = xr                    # attach y's tree under x's root
        return True                           # union was performed
```

**DSU for strings:**

```python
class DSU:
    def __init__(self):
        self.par = {}                         # dict-based (for any hashable type)

    def find(self, x):
        if x != self.par.setdefault(x, x):   # setdefault: if x not in dict, add x:x
            self.par[x] = self.find(self.par[x])
        return self.par[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr: return
        self.par[yr] = xr
```

**DSU with union by rank** - always attach smaller tree under larger tree (keeps trees shallow):

```python
class DSU:
    def __init__(self, N):
        self.par = list(range(N))
        self.sz = [1] * N                    # size of each group

    def find(self, x):
        if self.par[x] != x:
            self.par[x] = self.find(self.par[x])
        return self.par[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr:
            return False
        if self.sz[xr] < self.sz[yr]:       # always attach smaller under larger
            xr, yr = yr, xr
        self.par[yr] = xr                   # attach smaller tree under larger
        self.sz[xr] += self.sz[yr]          # update size
        return True
```
