## Union Find

Union-Find (Disjoint Set Union) tracks which elements belong to the same group. Two operations:
- `find(x)`: find the root representative of x's group
- `union(x, y)`: merge the groups containing x and y

```
Path compression in find():

Before find(4):          After find(4):
    0                        0
    |                      / | \
    1                     1  2  4      (2 and 4 now point directly to root)
    |
    2
    |
    4
```

**DSU for integers:**

```python
class DSU:
    def __init__(self, N):
        self.par = list(range(N))  # each element is its own parent initially

    def find(self, x):
        if self.par[x] != x:
            self.par[x] = self.find(self.par[x])  # path compression: point to root
        return self.par[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr: return False    # already in same group
        self.par[yr] = xr
        return True
```

**DSU for strings (or any hashable type):**

```python
class DSU:
    def __init__(self):
        self.par = {}

    def find(self, x):
        if x != self.par.setdefault(x, x):  # auto-add new elements as their own parent
            self.par[x] = self.find(self.par[x])
        return self.par[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr: return
        self.par[yr] = xr
```

**DSU with union by size** (attach smaller tree under larger -- keeps trees shallow):

```python
class DSU:
    def __init__(self, N):
        self.par = list(range(N))
        self.sz = [1] * N

    def find(self, x):
        if self.par[x] != x:
            self.par[x] = self.find(self.par[x])
        return self.par[x]

    def union(self, x, y):
        xr, yr = self.find(x), self.find(y)
        if xr == yr: return False
        if self.sz[xr] < self.sz[yr]:  # always attach smaller under larger
            xr, yr = yr, xr
        self.par[yr] = xr
        self.sz[xr] += self.sz[yr]
        return True
```
