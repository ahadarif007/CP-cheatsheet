## Lambda

A lambda is an anonymous (unnamed) function written in one line. Syntax: `lambda arguments: expression`. It's just a shorthand for defining simple functions that you only need once.

Can be used with: `.sort()`, `sorted()`, `min()`, `max()`, `heapq.nlargest()`, `heapq.nsmallest()`, `map()`, `filter()`

```python
# a=3, b=8, target=10
# Which is closer to 10: a or b?
min((b, a), key=lambda x: abs(target - x))  # 8 (|10-8|=2 < |10-3|=7)
# lambda x: abs(target - x) is a function that returns "distance from target"
# min() picks the element with smallest distance
```

```python
>>> ids = ['id1', 'id2', 'id30', 'id3', 'id22', 'id100']
>>> print(sorted(ids))  # Default: lexicographic (dictionary order)
['id1', 'id100', 'id2', 'id22', 'id3', 'id30']  # '1' < '2' as strings!
>>> sorted_ids = sorted(ids, key=lambda x: int(x[2:]))  # Sort by numeric part
# x[2:] slices off "id" prefix, int() converts to number
>>> print(sorted_ids)
['id1', 'id2', 'id3', 'id22', 'id30', 'id100']  # correct numeric order!
```

**Multi-key sorting with lambda** - return a tuple from lambda for multi-level sort:

```python
sorted([('abc', 121), ('bbb', 23), ('abc', 148), ('bbb', 24)],
       key=lambda x: (x[0], x[1]))  # sort by 1st element, then 2nd
# Result: [('abc', 121), ('abc', 148), ('bbb', 23), ('bbb', 24)]
```
