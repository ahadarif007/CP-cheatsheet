## Merge Arrays

**Merge K sorted arrays** - use a heap to always get the smallest element across all arrays:

```python
# Simple one-liner using built-in merge
def mergeSortedArrays(self, arrays):
    return list(heapq.merge(*arrays))  # heapq.merge handles k-way merge efficiently
```

**Manual implementation with heap:**

```python
class Solution:
def mergeSortedArrays(self, arrays):
    pq = []
    # Initialize heap with first element from each array
    for i, arr in enumerate(arrays):
        pq.append((arr[0], i, 0))   # (value, which_array, index_in_array)
    heapify(pq)                     # build min-heap

    res = []
    while pq:
        num, i, j = heappop(pq)    # get smallest across all arrays
        res.append(num)
        if j + 1 < len(arrays[i]): # if that array has more elements
            heappush(pq, (arrays[i][j+1], i, j+1))  # push next element from same array
    return res
```

**Merge K sorted linked lists:**

```python
def mergeKLists(self, lists: List[ListNode]) -> ListNode:
    prehead = ListNode()          # dummy head for result list
    heap = []
    # Extract all values into a heap
    for i in range(len(lists)):
        node = lists[i]
        while node:
            heapq.heappush(heap, node.val)
            node = node.next
    # Build result list from heap (smallest first)
    node = prehead
    while len(heap) > 0:
        val = heapq.heappop(heap)
        node.next = ListNode()
        node = node.next
        node.val = val
    return prehead.next           # skip dummy head
```
