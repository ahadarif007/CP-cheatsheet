## Merge Arrays

**Merge K sorted arrays** - use a heap to always get the smallest element across all arrays:

```
  arrays = [[1, 4, 7],
             [2, 5, 8],
             [3, 6, 9]]

  Heap state (always pick smallest):
  heap: [(1,0,0), (2,1,0), (3,2,0)]   → pop 1, push 4 from array 0
  heap: [(2,1,0), (3,2,0), (4,0,1)]   → pop 2, push 5 from array 1
  heap: [(3,2,0), (4,0,1), (5,1,1)]   → pop 3, push 6 from array 2
  ...
  result: [1, 2, 3, 4, 5, 6, 7, 8, 9]

  Heap entry: (value, which_array, index_in_array)
```

```python
# One-liner using built-in merge
def mergeSortedArrays(self, arrays):
    return list(heapq.merge(*arrays))
```

**Manual implementation with heap:**

```python
def mergeSortedArrays(self, arrays):
    pq = []
    for i, arr in enumerate(arrays):
        pq.append((arr[0], i, 0))    # (value, which_array, index_in_array)
    heapify(pq)

    res = []
    while pq:
        num, i, j = heappop(pq)      # get smallest across all arrays
        res.append(num)
        if j + 1 < len(arrays[i]):    # if that array has more elements
            heappush(pq, (arrays[i][j + 1], i, j + 1))
    return res
```

**Merge K sorted linked lists:**

```python
def mergeKLists(self, lists: List[ListNode]) -> ListNode:
    prehead = ListNode()
    heap = []
    for i in range(len(lists)):
        node = lists[i]
        while node:
            heapq.heappush(heap, node.val)
            node = node.next
    node = prehead
    while heap:
        val = heapq.heappop(heap)
        node.next = ListNode(val)
        node = node.next
    return prehead.next
```
