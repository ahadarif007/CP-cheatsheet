## Ternary

Python's one-line if-else expression: `value_if_true if condition else value_if_false`

This is NOT a statement (no colon, no indentation) - it's an expression that returns a value.

```Python
test = stk.pop() if stk else '#'
# Equivalent to:
# if stk:            # if stack is not empty (truthy)
#     test = stk.pop()  # pop and assign
# else:
#     test = '#'     # assign default value
```

**Inside comprehensions** - filter or transform conditionally:

```python
vals = [x if x > 0 else 0 for x in [-2, 3, -1, 5]]  # [0, 3, 0, 5] — clamp negatives to 0
evens = [x for x in range(10) if x % 2 == 0]          # [0, 2, 4, 6, 8] — filter only
```
