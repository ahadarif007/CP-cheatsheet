## Tree

A tree is a fundamental data structure. Here are the key properties to memorize:

1. A [tree](https://www.geeksforgeeks.org/some-theorems-on-trees/) is an undirected [graph](https://www.cs.sfu.ca/~ggbaker/zju/math/trees.html) in which any two vertices are
   connected by exactly one path.
1. Any connected graph who has n nodes with n-1 edges is a tree.
1. The degree of a vertex is the number of edges connected to the vertex.
1. A leaf is a vertex of degree 1. An internal vertex is a vertex of degree at least 2.
1. A [path graph](https://en.wikipedia.org/wiki/Path_graph) is a tree with two or more vertices with no branches, degree of 2 except for leaves which have degree of 1

1. Any two vertices in G can be connected by a unique simple path.
1. G is acyclic, and a simple cycle is formed if any edge is added to G.
1. G is connected and has no cycles.
1. G is connected but would become disconnected if any single edge is removed from G.

**Adjacency list representation** - the standard way to represent trees in CP. Input is usually n-1 edges:

```python
from collections import defaultdict

n = 5  # number of nodes
adj = defaultdict(list)
edges = [(0,1), (0,2), (1,3), (1,4)]  # n-1 edges
for u, v in edges:
    adj[u].append(v)
    adj[v].append(u)  # undirected: add both directions
# adj = {0: [1,2], 1: [0,3,4], 2: [0], 3: [1], 4: [1]}
```

**DFS on adjacency list** - use parent tracking to avoid revisiting the node you came from (no visited set needed for trees):

```python
def dfs(node, parent):
    for neighbor in adj[node]:
        if neighbor != parent:      # don't go back to parent
            dfs(neighbor, node)

dfs(0, -1)  # start DFS from root 0, no parent (-1)
```

**Subtree size** - count nodes in each subtree (useful for many tree problems):

```python
size = [1] * n
def calc_size(node, parent):
    for neighbor in adj[node]:
        if neighbor != parent:
            calc_size(neighbor, node)
            size[node] += size[neighbor]

calc_size(0, -1)  # size[i] = number of nodes in subtree rooted at i
```

**Finding leaves** - nodes with degree 1 (only one edge):

```python
leaves = [i for i in range(n) if len(adj[i]) == 1]
```
