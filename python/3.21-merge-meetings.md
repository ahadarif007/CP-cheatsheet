## Merge Meetings

**Insert and merge a new interval into a sorted, non-overlapping list:**

```
Before: [1,3]  [6,9]       insert [2,5]
After:  [1,5]  [6,9]       (merged [1,3] and [2,5] -> [1,5])

Overlap check: current_end >= next_start
  [1,  3]  and  [2,  5]
       3  >=  2  -> overlap! merge into [1, max(3,5)] = [1,5]
  [1,  5]  and  [6,  9]
       5  <   6  -> no overlap, keep separate
```

```python
def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
    bisect.insort(intervals, newInterval)   # insert in sorted position by start
    merged = [intervals[0]]
    for s, e in intervals:
        prev_start, prev_end = merged[-1]
        if prev_end >= s:                   # overlap: current end reaches into next
            merged[-1] = [prev_start, max(prev_end, e)]  # extend
        else:
            merged.append([s, e])           # no overlap: keep separate
    return merged
```
