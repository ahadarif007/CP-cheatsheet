## Merge Meetings

**Insert and merge a new interval into a sorted, non-overlapping list:**

```python
def insert(self, intervals: List[List[int]], newInterval: List[int]) -> List[List[int]]:
    bisect.insort(intervals, newInterval)   # insert in sorted position
    merged = [intervals[0]]                 # start with first interval
    for i in intervals:
        ms, me = merged[-1]                 # current merged interval (start, end)
        s, e = i                            # next interval to consider
        if me >= s:                         # overlap! (current end >= next start)
            merged[-1] = (ms, max(me, e))   # extend current interval
        else:                               # no overlap
            merged.append(i)                # add as separate interval
    return merged
```
