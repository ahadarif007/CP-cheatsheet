## Sliding Window

A technique for problems involving contiguous subarrays/substrings. Instead of checking every possible subarray (O(n^2)), we maintain a "window" that slides across the array (O(n)).

```
How it works:

  array: [2, 1, 5, 1, 3, 2]    find max sum subarray of size 3

  step 1:  [2, 1, 5] 1  3  2    window sum = 8
  step 2:   2 [1, 5, 1] 3  2    remove 2, add 1 → sum = 7
  step 3:   2  1 [5, 1, 3] 2    remove 1, add 3 → sum = 9  ← max!
  step 4:   2  1  5 [1, 3, 2]   remove 5, add 2 → sum = 6

  Instead of recalculating sum each time: slide by removing left, adding right
```

**Pattern:**
1. Expand window to the right (add elements)
2. When window violates constraint, shrink from the left (remove elements)
3. Track the best valid window seen

**Problem clues:** min/max subarray length, subarray with sum/product = K, at most K distinct elements

**Longest substring without repeating characters:**

```python
def lengthOfLongestSubstring(self, s: str) -> str:
    hm = {}              # char → its latest index
    best = 0
    l = 0                # left pointer

    for r, c in enumerate(s):       # expand window to the right
        if c in hm and hm[c] >= l:
            l = hm[c] + 1           # move left past the duplicate
        hm[c] = r                   # record latest position of character
        best = max(best, r - l + 1) # window size = right - left + 1

    return best
```

```
  s = "abcabcbb"
       ↑     ↑
       l     r

  a b c a b c b b     hm tracks last seen index of each char
  [a]                  l=0, r=0, hm={a:0}, best=1
  [a b]                l=0, r=1, hm={a:0,b:1}, best=2
  [a b c]              l=0, r=2, hm={a:0,b:1,c:2}, best=3
   a[b c a]            l=1, r=3, saw 'a' at 0, l=0+1=1, best=3
   a  b[c a b]         l=2, r=4, saw 'b' at 1, l=1+1=2, best=3
   ...
```

**Max fruits in 2 baskets** (at most 2 distinct types in window):

```python
def fruits_into_baskets(fruits):
    maxCount, j = 0, 0    # maxCount = best answer, j = left pointer
    ht = {}                # count of each fruit type in window

    for i, c in enumerate(fruits):
        ht[c] = ht.get(c, 0) + 1  # add fruit to window

        if len(ht) <= 2:          # valid: at most 2 types
            maxCount = max(maxCount, i - j + 1)
        else:                      # invalid: more than 2 types → shrink
            jc = fruits[j]
            ht[jc] -= 1
            if ht[jc] == 0:
                del ht[jc]         # remove type when count reaches 0
            j += 1                 # shrink from left

    return maxCount
```
