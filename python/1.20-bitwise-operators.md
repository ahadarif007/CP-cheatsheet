## Bitwise Operators

Bitwise operators work on individual bits of integers. They're useful for flags, permissions, and certain algorithmic tricks.

```python
# Visualize: 0b1100 = 12, 0b1010 = 10
'0b{:04b}'.format(0b1100 & 0b1010)  # '0b1000' = 8
# AND (&): result bit is 1 only if BOTH input bits are 1
#   1100
# & 1010
# = 1000

'0b{:04b}'.format(0b1100 | 0b1010)  # '0b1110' = 14
# OR (|): result bit is 1 if EITHER input bit is 1
#   1100
# | 1010
# = 1110

'0b{:04b}'.format(0b1100 ^ 0b1010)  # '0b0110' = 6
# XOR (^): result bit is 1 if bits are DIFFERENT
# KEY TRICK: x ^ x = 0 (cancels out!) Used to find the "lone" number in a list
#   1100
# ^ 1010
# = 0110

'0b{:04b}'.format(0b1100 >> 2)      # '0b0011' = 3
# RIGHT SHIFT (>>): move all bits right by n positions (equivalent to divide by 2^n)
# 1100 >> 2 = 0011 (12 / 4 = 3)

'0b{:04b}'.format(0b0011 << 2)      # '0b1100' = 12
# LEFT SHIFT (<<): move all bits left by n positions (equivalent to multiply by 2^n)
# 0011 << 2 = 1100 (3 * 4 = 12)

~0b1010                             # -0b1011 = -11
# NOT (~): flips all bits. ~x == -(x+1) in Python (two's complement)
```

**Common bit tricks:**

```python
n & (n - 1)                 # remove lowest set bit (0 if n is power of 2)
n & (n - 1) == 0 and n > 0  # check if n is a power of 2
n & (-n)                    # isolate lowest set bit
bin(n).count('1')           # count set bits (ones)
n.bit_count()               # count set bits (Python 3.10+)
n.bit_length()              # number of bits needed to represent n (e.g. 5 -> 3)

# Get, set, clear the k-th bit (0-indexed from right):
(n >> k) & 1                # get k-th bit
n | (1 << k)                # set k-th bit to 1
n & ~(1 << k)               # clear k-th bit to 0
n ^ (1 << k)                # toggle k-th bit
```
