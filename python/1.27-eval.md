## eval

`eval()` evaluates a string as a Python expression and returns the result.

```python
x = 1
print(eval('x + 1'))   # 2 (evaluates the expression in current scope)

# Useful for parsing math expressions in problems
eval("3 + 5 * 2")      # 13 (follows operator precedence)
eval("2 ** 10")         # 1024
```

**CP use case** - evaluate expressions given as strings:

```python
# Given: tokens = ["2", "+", "3", "*", "4"]
expr = ' '.join(tokens)   # "2 + 3 * 4"
result = eval(expr)        # 14

# WARNING: never use eval() on untrusted input — security risk!
# In CP this is fine since input is controlled.
```
