## Loops

Loops let you repeat code. Python has two main loop types: `while` (repeat until condition is false) and `for` (iterate over a sequence). Understanding loops is critical because almost every algorithm uses them.

**Basic while loop** - manually control the index variable:

```python
i = 0                      # initialize counter
while i < len(str):        # keep going until i reaches end of string
  i += 1                   # manually increment (easy to forget = infinite loop!)
```

**For loop** - Python handles the counter for you (preferred when possible):

```python
for i in range(len(message)):  # range(n) generates 0, 1, 2, ..., n-1
  print(i)                     # prints index 0, 1, 2, ...
```

**enumerate()** - get both index and value (cleaner than `range(len(...))`):

```python
for i, ch in enumerate("abc"):    # i=0,ch='a' then i=1,ch='b' then i=2,ch='c'
    print(i, ch)

for i, val in enumerate(nums, start=1):  # start counting from 1 instead of 0
    print(i, val)
```

**break / continue** - control loop flow:

```python
for x in range(10):
    if x == 5: break       # exit the loop entirely when x is 5
    if x % 2 == 0: continue  # skip the rest of this iteration, go to next x
    print(x)               # prints: 1, 3
```

**Walking backwards from right** - useful in "next permutation" problems. We scan from right to left looking for the first element that is smaller than its right neighbor:

```python
# Example: nums = [1, 5, 8, 4, 7, 6, 5, 3, 1]
# We want to find where the sequence stops being non-increasing from the right
while i > 0 and nums[i-1] >= nums[i]:  # while left element >= right element
  i -= 1                                # move left
# After loop: nums[i-1] is the first "dip" - the element to swap
```

**Two-pointer reversal** - swap elements from both ends moving inward. This is an in-place O(1) space technique used in many problems:

```python
l, r = i, len(nums) - 1     # l starts at left boundary, r starts at right boundary
while l < r:                 # keep going until pointers meet in the middle
  nums[l], nums[r] = nums[r], nums[l]  # Python swap: no temp variable needed!
  l += 1                     # move left pointer right
  r -= 1                     # move right pointer left
```

**Boundary trick** - process the last element without a special case after the loop. By going one past the end, we can treat "end of string" the same as finding a delimiter:

```python
for i in range(len(message) + 1):        # +1 so i can equal len(message)
  if i == len(message) or message[i] == ' ':  # both trigger "end of word" logic
    # process the word that just ended
```

**Fun with range()** - `range(start, stop, step)` is very flexible. `stop` is always exclusive (not included):

```python
for a in range(0,3):          # produces: 0, 1, 2 (stop=3 is excluded)
for a in reversed(range(0,3)) # produces: 2, 1, 0 (reversed iterator)
for i in range(3,-1,-1)       # produces: 3, 2, 1, 0 (step=-1, stop=-1 excluded)

# TRICK: Using ~i (bitwise NOT) to access elements from the back
# ~i equals -(i+1), so ~0=-1, ~1=-2, ~2=-3
# This is useful for comparing front and back simultaneously
for i in range(len(A)//2):    # A = [0,1,2,3,4,5], iterates first half only
  print(i)     # 0, 1, 2         (index from front)
  print(A[i])  # 0, 1, 2         (element from front)
  print(~i)    # -1, -2, -3      (mirror index from back)
  print(A[~i]) # 5, 4, 3         (element from back)
  # This is perfect for palindrome checking: if A[i] != A[~i]: not palindrome
```
