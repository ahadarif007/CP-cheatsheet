## Graph

A graph is a set of nodes (vertices) connected by edges. Unlike trees, graphs can have cycles and nodes can have any number of connections.

**Build adjacency list from edge list** - the most common graph representation. `graph[i]` gives you all neighbors of node i:

```python
# N = 6 nodes, edges = [[0,1],[0,2],[2,3],[2,4],[2,5]]
graph = [[] for _ in range(N)]   # one empty list per node
for u, v in edges:
    graph[u].append(v)           # u connects to v
    graph[v].append(u)           # v connects to u (undirected = add both directions)
# Result: [[1, 2], [0], [0, 3, 4, 5], [2], [2], [2]]
# Meaning: node 0 connects to 1,2; node 1 connects to 0; node 2 connects to 0,3,4,5; etc.
```

**Convert a tree (with node objects) to an adjacency list** - useful when you need to traverse a tree in any direction (not just parent→child):

```python
adj = collections.defaultdict(list)  # auto-creates empty list for new keys
def dfs(node):
    if node.left:
        adj[node].append(node.left)      # parent -> child edge
        adj[node.left].append(node)      # child -> parent edge (bidirectional!)
        dfs(node.left)
    if node.right:
        adj[node].append(node.right)
        adj[node.right].append(node)
        dfs(node.right)
dfs(root)
```

**DFS on a graph (tree-shaped)** - use a `parent` parameter to avoid going backward when the graph has no cycles:

```python
# graph = [[1, 2], [0], [0, 3, 4, 5], [2], [2], [2]]
def dfs(node, par=-1):
    for nei in graph[node]:        # for each neighbor of current node
        if nei != par:             # don't go back to where we came from
            res = dfs(nei, node)   # recurse, passing current as parent
dfs(0)  # visits: 0 -> 1 -> 2 -> 3 -> 4 -> 5
```

**DFS with visited set** - for graphs that can have cycles, use a set to track visited nodes:

```python
def dfs(node):
    visited.add(node)
    for nei in graph[node]:
        if nei not in visited:
            dfs(nei)

visited = set()
dfs(0)
```

**BFS on a graph** - level-by-level traversal using a queue. Useful for shortest path in unweighted graphs:

```python
from collections import deque

def bfs(start):
    visited = {start}
    queue = deque([start])
    while queue:
        node = queue.popleft()          # O(1) pop from left
        for nei in graph[node]:
            if nei not in visited:
                visited.add(nei)
                queue.append(nei)

# BFS level by level (when you need the distance/depth):
def bfs_levels(start):
    visited = {start}
    queue = deque([start])
    level = 0
    while queue:
        for _ in range(len(queue)):     # process all nodes at current level
            node = queue.popleft()
            for nei in graph[node]:
                if nei not in visited:
                    visited.add(nei)
                    queue.append(nei)
        level += 1
```
