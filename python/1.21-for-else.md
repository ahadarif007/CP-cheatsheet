## For Else

Python has a unique feature: `else` on a for loop. The else block executes ONLY if the loop completed normally (WITHOUT hitting a `break`). If `break` was called, the else block is skipped.

This is useful for "search" patterns where you need to know if the search found anything:

```python
# Alien dictionary problem: determine character ordering from sorted words
for w1, w2 in zip(words, words[1:]):  # compare adjacent words: "abc" vs "ab"
    for c1, c2 in zip(w1, w2):        # compare character by character
        if c1 != c2:                  # found first difference!
            adj[c1].append(c2)        # c1 comes before c2 in alien alphabet
            degrees[c2] += 1
            break                     # only first difference matters, stop comparing
    else:
        # This runs ONLY if inner loop did NOT break
        # Meaning: all compared characters were equal (one word is prefix of other)
        if len(w1) > len(w2):         # longer word before shorter prefix = invalid!
            return ""                 # "abc" before "ab" violates dictionary rules
```
