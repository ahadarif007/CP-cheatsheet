## Max Profit Stock

Stock problems follow a state machine pattern: at each price, you're either holding stock (state t1) or not holding (state t0). Transitions: buy (t0->t1) and sell (t1->t0).

```
State Machine:

              buy (-price)
  ┌────────┐ ─────────────► ┌────────┐
  │   t0   │                │   t1   │
  │ (cash) │ ◄───────────── │ (hold) │
  └────────┘  sell (+price)  └────────┘
     │  ^                      │  ^
     └──┘                      └──┘
     rest                      rest
```

**Infinite transactions** - [base formula](https://leetcode.com/problems/best-time-to-buy-and-sell-stock-with-cooldown/discuss/75924/Most-consistent-ways-of-dealing-with-the-series-of-stock-problems):

```python
def maxProfit(self, prices: List[int]) -> int:
    t0, t1 = 0, float('-inf')     # t0 = profit when NOT holding, t1 = profit when HOLDING
    for p in prices:
        old_t0 = t0               # save before updating (both use old values)
        t0 = max(t0, t1 + p)     # sell: was holding + sell price
        t1 = max(t1, old_t0 - p) # buy: was not holding - buy price
    return t0
```

**Single transaction** - t1 can only buy from 0 profit (one buy allowed):

```python
def maxProfit(self, prices: List[int]) -> int:
    t0, t1 = 0, float('-inf')
    for p in prices:
        t0 = max(t0, t1 + p)     # sell (best sell so far)
        t1 = max(t1, -p)         # buy: 0 - price (only one buy, always starts from 0)
    return t0
```

**K transactions:**

```python
t0 = [0] * (k + 1)               # t0[i] = max profit after i sells, not holding
t1 = [float('-inf')] * (k + 1)   # t1[i] = max profit after i-1 sells, holding
for p in prices:
    for i in range(k, 0, -1):    # iterate k down to 1 (avoids overwriting needed values)
        t0[i] = max(t0[i], t1[i] + p)    # sell: complete transaction i
        t1[i] = max(t1[i], t0[i-1] - p)  # buy: start transaction i (needs i-1 completed)
return t0[k]
```

**With cooldown** (must wait 1 day after selling before buying):

```python
def maxProfit(self, prices: List[int]) -> int:
    t0, t1, t0_prev = 0, float('-inf'), 0
    for p in prices:
        old_t0 = t0
        t0 = max(t0, t1 + p)        # sell
        t1 = max(t1, t0_prev - p)   # buy from 2 days ago (cooldown!)
        t0_prev = old_t0
    return t0
```
