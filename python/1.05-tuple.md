## Tuple

A tuple is like a list but **immutable** (cannot be changed after creation). Use tuples when you need a fixed collection that won't change. They're also hashable, meaning they can be used as dictionary keys or in sets (lists cannot).

```python
thistuple = ("apple", "banana", "cherry")
print(thistuple[1])  # "banana" - access by index just like lists
# thistuple[1] = "grape"  # ERROR! Cannot modify tuples
```

**Tuple unpacking** - assign multiple variables at once:

```python
a, b, c = (1, 2, 3)           # a=1, b=2, c=3
a, b = b, a                   # Python swap idiom (creates a temp tuple under the hood)
a, *rest = (1, 2, 3, 4)       # a=1, rest=[2, 3, 4] (star captures remainder as list)
first, *mid, last = [1,2,3,4,5]  # first=1, mid=[2,3,4], last=5
```

**Tuple comparison is lexicographic** - compares element by element, left to right. This is critical for sorting and heaps:

```python
(1, 3) < (2, 1)    # True  (1 < 2, done)
(1, 3) < (1, 5)    # True  (1 == 1, then 3 < 5)
(1, 3) < (1, 3, 0) # True  (equal prefix, shorter tuple is smaller)

# Use in heapq: push (distance, node) to get min-distance first
import heapq
heap = []
heapq.heappush(heap, (5, 'B'))
heapq.heappush(heap, (2, 'A'))
heapq.heappop(heap)  # (2, 'A') - smallest distance popped first
```

**Tuples as dictionary keys** - since tuples are hashable, they make great dict keys. This is a common pattern for grouping anagrams (words with same letters):

```python
def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
    d = defaultdict(list)          # dict where missing keys auto-create empty lists
    for w in strs:
        key = tuple(sorted(w))     # sorted("eat") = ['a','e','t']
                                    # tuple(['a','e','t']) = ('a','e','t')
                                    # All anagrams produce the same tuple key!
        d[key].append(w)           # group words by their sorted-letter tuple
    return d.values()
```

**Tuples as visited-set keys** - use (row, col) or (state, position) as hashable keys:

```python
visited = set()
visited.add((r, c))            # track grid positions
visited.add((r, c, direction)) # track state in BFS with direction
```
