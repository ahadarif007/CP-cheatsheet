## Convert Base

Converting between number bases uses two operations:
1. `% base` extracts the least significant digit
2. `// base` removes that digit (shifts number right)
3. Build digits from least significant to most significant, then reverse

```
  Example: 255 → hex

  255 % 16 = 15 → 'f'     255 // 16 = 15
   15 % 16 = 15 → 'f'      15 // 16 = 0    stop!
  Built backwards: ['f', 'f'] → reverse → "ff"

  Example: 26 → binary

  26 % 2 = 0    26 // 2 = 13
  13 % 2 = 1    13 // 2 = 6
   6 % 2 = 0     6 // 2 = 3
   3 % 2 = 1     3 // 2 = 1
   1 % 2 = 1     1 // 2 = 0   stop!
  Built backwards: [0,1,0,1,1] → reverse → "11010"
```

```python
def toHex(self, num: int) -> str:
    if num == 0: return '0'
    if num < 0: num += 2 ** 32   # two's complement for negatives

    digits = "0123456789abcdef"
    rtn = []
    while num > 0:
        rtn.append(digits[num % 16])   # extract least significant digit
        num //= 16                      # remove it
    return "".join(rtn[::-1])           # reverse (built LSB-first)
```

**Generic base conversion:**

```python
def toBase(num, base):
    if num == 0: return '0'
    digits = "0123456789abcdefghijklmnopqrstuvwxyz"
    rtn = []
    while num > 0:
        rtn.append(digits[num % base])
        num //= base
    return "".join(rtn[::-1])
```
