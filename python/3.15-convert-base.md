## Convert Base

Converting between number bases uses two operations:
1. `% base` extracts the least significant digit
2. `// base` removes that digit (shifts number right)
3. Build digits from least significant to most significant, then reverse

```python
def toHex(self, num: int) -> str:
  rtn = []
  index = "0123456789abcdef"   # lookup table: digit value -> hex character
  if num == 0: return '0'
  if num < 0: num += 2 ** 32  # handle negative numbers via two's complement
                               # (this gives the unsigned 32-bit representation)
  while num > 0:
    digit = num % 16           # get least significant hex digit (0-15)
    rtn.append(index[digit])   # convert to character ('0'-'f')
    num = num // 16            # remove that digit (shift right in base 16)
  return "".join(rtn[::-1])    # reverse because we built it backwards (LSB first)
```
