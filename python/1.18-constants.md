## Constants

When searching for a maximum or minimum value, you need initial values that will always be beaten by any real number. Python's `float('inf')` and `float('-inf')` serve this purpose:

```Python
max_val = float('-inf')  # negative infinity: ANY number is greater than this
                          # use as initial value when looking for maximum
min_val = float('inf')   # positive infinity: ANY number is less than this
                          # use as initial value when looking for minimum
# Example: find max in list
# max_val starts at -inf, so first number always replaces it
```
