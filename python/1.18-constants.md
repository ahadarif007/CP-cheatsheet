## Constants

When searching for a maximum or minimum value, you need initial values that will always be beaten by any real number. Python's `float('inf')` and `float('-inf')` serve this purpose:

```Python
max_val = float('-inf')  # negative infinity: ANY number is greater than this
                          # use as initial value when looking for maximum
min_val = float('inf')   # positive infinity: ANY number is less than this
                          # use as initial value when looking for minimum

import math
math.inf                  # same as float('inf'), cleaner syntax
-math.inf                 # same as float('-inf')

import sys
sys.maxsize               # largest platform int (2^63-1 on 64-bit), useful for array bounds
```

**Competitive programming modulo** - many problems require answers modulo 10^9+7 to prevent overflow:

```Python
MOD = 10**9 + 7
# Apply mod at each step to keep numbers small:
result = (a * b) % MOD
result = (a + b) % MOD
# Python handles big ints natively, but modding keeps values small for speed
# pow(base, exp, MOD) does modular exponentiation efficiently: O(log exp)
pow(2, 100, MOD)  # 2^100 mod (10^9+7), computed in O(log 100) not O(100)
```
