## Topological Sort

Topological sort orders nodes in a directed graph so that for every edge u→v, u comes before v. Only works on DAGs (Directed Acyclic Graphs). Common use: task scheduling, course prerequisites.

[Kahn's algorithm](https://www.geeksforgeeks.org/all-topological-sorts-of-a-directed-acyclic-graph/) - BFS-based approach using "indegree" (number of incoming edges):

1. Count indegree of each node (how many prerequisites it has)
2. Start with all nodes that have indegree 0 (no prerequisites)
3. Process each node: remove it and decrease indegree of its neighbors
4. If a neighbor's indegree reaches 0, add it to the queue
5. If we process all nodes: valid ordering exists. If not: there's a cycle!

```python
def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
    # Build graph and count indegrees
    adj = collections.defaultdict(list)  # course -> list of courses that depend on it
    degree = collections.Counter()       # course -> number of prerequisites

    for dest, orig in prerequisites:     # [dest, orig] means orig must come before dest
        adj[orig].append(dest)           # orig -> dest edge
        degree[dest] += 1               # dest has one more prerequisite

    # Start with courses that have NO prerequisites (indegree = 0)
    bfs = [c for c in range(numCourses) if degree[c] == 0]

    for o in bfs:                        # process each available course
        for d in adj[o]:                 # for each course that depends on o
            degree[d] -= 1              # one less prerequisite needed
            if degree[d] == 0:          # all prerequisites satisfied!
                bfs.append(d)           # this course is now available
    # Note: bfs list GROWS during iteration - this is intentional!

    # If we were able to take all courses, no cycle exists
    return len(bfs) == numCourses
```

**Alien Dictionary** - determine character ordering from sorted words:

```python
def alienOrder(self, words: List[str]) -> str:
    nodes = set("".join(words))           # all unique characters
    adj = collections.defaultdict(list)   # ordering edges
    degree = collections.Counter(nodes)   # initialize all degrees to 1

    for w1, w2 in zip(words, words[1:]):  # compare adjacent words
        for c1, c2 in zip(w1, w2):        # character by character
            if c1 != c2:                  # first difference tells us ordering
                adj[c1].append(c2)        # c1 comes before c2
                degree[c2] += 1
                break                     # only first difference matters!
        else:                             # no break = one is prefix of other
            if len(w1) > len(w2):         # longer before shorter = invalid
                return ""

    stk = list(filter(lambda x: degree[x]==1, degree.keys()))  # degree 1 = our "zero" (initialized at 1)

    ans = []
    while stk:
        node = stk.pop()
        ans.append(node)
        for nei in adj[node]:
            degree[nei] -= 1
            if degree[nei] == 1:          # back to base = ready to process
                stk.append(nei)

    return "".join(ans) * (set(ans) == nodes)  # multiply by bool: "" if not all nodes processed
```
