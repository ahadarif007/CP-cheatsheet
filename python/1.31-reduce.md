## Reduce

`reduce(function, iterable, initial)` combines all elements into a single value by repeatedly applying the function to accumulated result and next element. Like folding a list down to one value.

```python
from functools import reduce
numbers = [3, 4, 6, 9, 34, 12]
result = reduce(lambda x, y: x + y, numbers)     # 68
# Step by step: ((((3+4)+6)+9)+34)+12 = 68

result = reduce(lambda x, y: x + y, numbers, 10) # 78
# With initial value 10: (((((10+3)+4)+6)+9)+34)+12 = 78
```
