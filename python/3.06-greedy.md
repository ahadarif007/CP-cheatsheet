## Greedy

Make the optimal [choice](https://brilliant.org/wiki/greedy-algorithm/) at each step. Greedy works when a locally optimal choice leads to globally optimal solution.

```
When to use greedy vs DP:

  Greedy:  Each local choice leads to global optimum (no backtracking needed)
  DP:      Choices have overlapping subproblems, need to explore all options

  Greedy proof intuition: "can swapping a greedy choice for a non-greedy one
                            ever improve the result? If no → greedy works."
```

**Increasing Triplet Subsequence** - find if there exist i < j < k where nums[i] < nums[j] < nums[k]:

```python
def increasingTriplet(self, nums: List[int]) -> bool:
    first = second = float('inf')   # first = smallest, second = second smallest

    for n in nums:
        if n <= first:       # n is new smallest
            first = n
        elif n <= second:    # n > first, but new second-smallest
            second = n
        else:                # n > second > first → found triplet!
            return True

    return False
```

```
  nums = [2, 1, 5, 0, 4, 6]

  n=2: first=2,  second=∞
  n=1: first=1,  second=∞     (new smallest)
  n=5: first=1,  second=5     (new second)
  n=0: first=0,  second=5     (new smallest, but second still valid!)
  n=4: first=0,  second=4     (new second)
  n=6: 6 > 4 > 0 → return True!

  Key insight: even when first updates past second,
  second is still valid because SOME earlier value was smaller than it.
```

**Jump Game** - can you reach the last index? Track the farthest reachable position:

```python
def canJump(self, nums: List[int]) -> bool:
    farthest = 0
    for i, jump in enumerate(nums):
        if i > farthest:         # can't reach this index
            return False
        farthest = max(farthest, i + jump)   # update farthest reachable
    return True
```

```
  nums = [2, 3, 1, 1, 4]
  i=0: farthest = max(0, 0+2) = 2
  i=1: farthest = max(2, 1+3) = 4   → can reach index 4 (the end)!
  
  nums = [3, 2, 1, 0, 4]
  i=0: farthest=3, i=1: farthest=3, i=2: farthest=3, i=3: farthest=3
  i=4: 4 > 3 → return False (stuck at index 3)
```
