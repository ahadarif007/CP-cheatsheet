## Merge Sort

Merge sort divides the array in half, recursively sorts each half, then merges the two sorted halves. Always O(n log n) but uses O(n) extra space.

```
  [38, 27, 43, 3, 9, 82, 10]
           split
  [38, 27, 43, 3]    [9, 82, 10]
       split              split
  [38, 27] [43, 3]   [9, 82] [10]
    split    split     split
  [38][27] [43][3]  [9][82]  [10]
    merge    merge    merge
  [27, 38] [3, 43]  [9, 82]  [10]
      merge             merge
  [3, 27, 38, 43]   [9, 10, 82]
           merge
  [3, 9, 10, 27, 38, 43, 82]
```

```python
from collections import deque

def mergeSort(array):
    def sortArray(nums):
        if len(nums) <= 1:
            return nums
        mid = len(nums) // 2
        left = sortArray(nums[:mid])
        right = sortArray(nums[mid:])
        return merge(left, right)

    def merge(l1, l2):
        result = []
        l1, l2 = deque(l1), deque(l2)    # deque for O(1) popleft
        while l1 and l2:
            if l1[0] <= l2[0]:
                result.append(l1.popleft())
            else:
                result.append(l2.popleft())
        result.extend(l1 or l2)           # append whichever has remaining
        return result

    return sortArray(array)
```
