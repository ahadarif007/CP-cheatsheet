## Quick Sort

Quicksort picks a "pivot" element and partitions the array into elements less than pivot and greater than pivot, then recursively sorts each partition. Average O(n log n), worst O(n^2).

```python
def quickSort(array):
	def sort(arr, l, r):
		if l < r:                    # base case: subarray of size 0 or 1 is sorted
			p = part(arr, l, r)      # partition and get pivot's final position
			sort(arr, l, p-1)        # recursively sort left of pivot
			sort(arr, p+1, r)        # recursively sort right of pivot

	def part(arr, l, r):
		pivot = arr[r]               # choose rightmost element as pivot
		a = l                        # 'a' = boundary between "less than" and "greater than"
		                             # everything at indices [l, a) is < pivot
		for i in range(l, r):        # scan through all elements except pivot
			if arr[i] < pivot:       # element is less than pivot
				arr[i], arr[a] = arr[a], arr[i]  # swap it to the "less than" zone
				a += 1               # expand the "less than" zone
		arr[r], arr[a] = arr[a], arr[r]  # place pivot at boundary (its final position!)
		return a                     # return pivot's final index

	sort(array, 0, len(array)-1)
	return array
```
