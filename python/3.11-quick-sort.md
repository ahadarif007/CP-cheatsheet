## Quick Sort

Quicksort picks a "pivot" element and partitions the array into elements less than pivot and greater than pivot, then recursively sorts each partition. Average O(n log n), worst O(n^2).

```
Partition example (pivot = last element):

  [3, 7, 8, 5, 2, 1, 9, 4]    pivot = 4, a = 0
   i                            3 < 4 → swap(0,0), a=1:  [3, 7, 8, 5, 2, 1, 9, 4]
      i                         7 > 4 → skip
         i                      8 > 4 → skip
            i                   5 > 4 → skip
               i                2 < 4 → swap(1,4), a=2:  [3, 2, 8, 5, 7, 1, 9, 4]
                  i             1 < 4 → swap(2,5), a=3:  [3, 2, 1, 5, 7, 8, 9, 4]
                     i          9 > 4 → skip
  Place pivot at a=3:                                     [3, 2, 1, 4, 7, 8, 9, 5]
                                         < pivot  [pivot]  > pivot
```

```python
def quickSort(array):
    def sort(arr, l, r):
        if l < r:
            p = partition(arr, l, r)
            sort(arr, l, p - 1)        # sort left of pivot
            sort(arr, p + 1, r)        # sort right of pivot

    def partition(arr, l, r):
        pivot = arr[r]                 # choose rightmost as pivot
        a = l                          # boundary: everything [l, a) is < pivot
        for i in range(l, r):
            if arr[i] < pivot:
                arr[i], arr[a] = arr[a], arr[i]
                a += 1                 # expand "less than" zone
        arr[r], arr[a] = arr[a], arr[r]  # place pivot at boundary
        return a

    sort(array, 0, len(array) - 1)
    return array
```
