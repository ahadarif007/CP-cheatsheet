## String Subsequence

Find minimum number of subsequences of `source` needed to spell `target`. Use binary search on character positions for efficient matching.

```python
def shortestWay(self, source: str, target: str) -> int:
    # Precompute: for each character, store all indices where it appears in source
    ref = collections.defaultdict(list)
    for i, c in enumerate(source):
        ref[c].append(i)

    ans = 1          # at least one subsequence needed
    i = -1           # current position in source (start before beginning)
    for c in target:
        if c not in ref:
            return -1                    # character not in source at all = impossible
        offset = ref[c]                  # all positions of char c in source
        j = bisect.bisect_left(offset, i)  # find first position >= i
        if j == len(offset):             # no valid position found (past end of source)
            ans += 1                     # need a new subsequence (restart source)
            i = offset[0] + 1            # start from first occurrence + 1
        else:
            i = offset[j] + 1            # advance past this character

    return ans
```
