## String Subsequence

Find minimum number of subsequences of `source` needed to spell `target`. Use binary search on precomputed character positions.

```
source = "abc",  target = "acdbc"

Precompute positions: a->[0], b->[1], c->[2]

target char   source pos used   action
───────────   ────────────────  ─────────────────────
  'a'         0                 found at 0, advance to 1
  'c'         2                 bisect finds 2 >= 1, use it, advance to 3
  'd'         --                'd' not in source -> return -1

source = "abc",  target = "abcbc"
  'a'         0                 subsequence 1: advance to 1
  'b'         1                 advance to 2
  'c'         2                 advance to 3
  'b'         --                bisect: no position >= 3 -> start new subsequence!
  'b'         1                 subsequence 2: advance to 2
  'c'         2                 advance to 3
Answer: 2 subsequences
```

```python
def shortestWay(self, source: str, target: str) -> int:
    positions = collections.defaultdict(list)
    for i, c in enumerate(source):
        positions[c].append(i)

    ans = 1          # need at least one subsequence
    i = -1           # current position in source (-1 = before start)
    for c in target:
        if c not in positions:
            return -1                          # impossible
        idxs = positions[c]
        j = bisect.bisect_left(idxs, i)        # first occurrence at position >= i
        if j == len(idxs):                     # past end of source
            ans += 1                           # start new subsequence
            i = idxs[0] + 1                    # use first occurrence
        else:
            i = idxs[j] + 1                    # advance past this character
    return ans
```
