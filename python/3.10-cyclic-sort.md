## Cyclic Sort

Cyclic sort places each number at its "correct" index in O(n) time using O(1) space. Works when elements are in range [1, n] or [0, n-1]. The key insight: keep swapping current element to its correct position until current position has the right element.

```python
def cyclic_sort(nums):
  # Goal: place number n at index n-1 (1 goes to index 0, 2 to index 1, etc.)
  i = 0
  while i < len(nums):
    j = nums[i] - 1              # where should nums[i] actually go?
    if nums[i] != nums[j]:      # nums[i] is NOT in its correct position
      nums[i], nums[j] = nums[j], nums[i]  # swap it there
      # DON'T increment i! The swapped-in element might also be wrong
    else:
      i += 1                    # current position is correct (or duplicate), move on
  return nums
# After sort: [1, 2, 3, 4, ...] - each number at its correct index
# Missing numbers can be found where nums[i] != i+1
```
