## All

`all()` returns True if ALL elements in the iterable are truthy. Short-circuits on first False.

```
all([T, T, T, T, T])  →  True   (every element truthy)
all([T, T, F, T, T])  →  False  (stopped at first falsy)
              ↑
              short-circuits here
```

```python
def all(iterable):
    for element in iterable:
        if not element:    # falsy: 0, '', None, [], {}
            return False   # found a falsy one! stop immediately
    return True            # everything was truthy

# Basic examples:
all([1, 2, 3])       # True  (all non-zero)
all([1, 0, 3])       # False (0 is falsy)
all([])              # True  (vacuous truth - nothing to disprove)

# With generator expression:
all(x > 0 for x in [1, 2, 3])  # True (all positive)
```

**CP patterns:**

```python
# Check if a string is all lowercase
all(c.islower() for c in "abcdef")  # True

# Validate sorted order
nums = [1, 3, 5, 7]
all(nums[i] <= nums[i+1] for i in range(len(nums)-1))  # True

# Check if all values in dict meet a condition
degree = {'a': 0, 'b': 0, 'c': 0}
all(v == 0 for v in degree.values())  # True
```

**any vs all quick reference:**

```
any = at least one True  → OR  across elements
all = every one is True  → AND across elements

any([]) = False    (no evidence FOR)
all([]) = True     (no evidence AGAINST)
```
