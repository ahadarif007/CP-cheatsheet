## Shift Array Right

Rotate array right by k positions. The elegant O(n) time, O(1) space solution uses three reversals:

Example: `[1,2,3,4,5]` rotated right by 2 = `[4,5,1,2,3]`
- Step 1: reverse all → `[5,4,3,2,1]`
- Step 2: reverse first k → `[4,5,3,2,1]`
- Step 3: reverse remaining → `[4,5,1,2,3]`

```python
def rotate(self, nums: List[int], k: int) -> None:
    def reverse(l, r, nums):
        while l < r:
            nums[l], nums[r] = nums[r], nums[l]  # swap
            l += 1
            r -= 1
    if len(nums) <= 1: return
    k = k % len(nums)              # handle k > array length (rotating by len = no change)
    reverse(0, len(nums)-1, nums)  # Step 1: reverse entire array
    reverse(0, k-1, nums)          # Step 2: reverse first k elements
    reverse(k, len(nums)-1, nums)  # Step 3: reverse remaining elements
```
