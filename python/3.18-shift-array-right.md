## Shift Array Right

Rotate array right by k positions. The O(n) time, O(1) space solution uses three reversals.

```
Example: [1, 2, 3, 4, 5], k = 2

Original:         [1, 2, 3, | 4, 5]     <- last k=2 elements need to move to front
                             ^^^^^
Step 1 reverse all:  [5, 4, 3, 2, 1]
Step 2 reverse 0..k-1: [4, 5, | 3, 2, 1]
Step 3 reverse k..end: [4, 5, | 1, 2, 3]    <- done!
```

**Why it works:** Reversing the whole array puts the last k elements at the front, but in wrong order. Reversing each half fixes the order within both halves.

```python
def rotate(self, nums: List[int], k: int) -> None:
    def reverse(l, r, nums):
        while l < r:
            nums[l], nums[r] = nums[r], nums[l]
            l += 1
            r -= 1

    if len(nums) <= 1: return
    k = k % len(nums)              # handle k > length (full rotation = no change)
    reverse(0, len(nums) - 1, nums)
    reverse(0, k - 1, nums)
    reverse(k, len(nums) - 1, nums)
```
