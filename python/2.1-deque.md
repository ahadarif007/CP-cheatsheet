## Deque

A deque (double-ended queue) supports efficient append and pop from BOTH ends (O(1)). Regular lists are O(n) for operations at the front. Use deque when you need a queue (FIFO) or need to add/remove from both ends.

```python
from collections import deque
deq = deque([1, 2, 3])
deq.appendleft(5)     # add to LEFT end: deque([5, 1, 2, 3])
deq.append(6)         # add to RIGHT end: deque([5, 1, 2, 3, 6])
deq.popleft()         # remove from LEFT: returns 5, deque([1, 2, 3, 6])
deq.pop()             # remove from RIGHT: returns 6, deque([1, 2, 3])

# Key operations:
# appendleft() + popleft() = stack behavior (LIFO) at left end
# append() + popleft() = queue behavior (FIFO)
# append() + pop() = stack behavior (LIFO) at right end
```
