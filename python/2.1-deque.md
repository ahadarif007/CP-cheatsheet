## Deque

A deque (double-ended queue) supports efficient append and pop from BOTH ends (O(1)). Regular lists are O(n) for operations at the front. Use deque when you need a queue (FIFO) or need to add/remove from both ends.

**How it works:**

```
Regular list (slow at front):
  pop(0) → [■, 2, 3, 4, 5]  →  [2, 3, 4, 5]   O(n) shift everything!
                ← ← ← ←

Deque (fast at both ends):
  LEFT end                    RIGHT end
     ↓                           ↓
   [ 1,   2,   3,   4,   5 ]
     ↑                           ↑
  popleft()  O(1)             pop()  O(1)
  appendleft()  O(1)         append()  O(1)
```

```python
from collections import deque
deq = deque([1, 2, 3])
deq.appendleft(5)     # add to LEFT end:    deque([5, 1, 2, 3])
deq.append(6)         # add to RIGHT end:   deque([5, 1, 2, 3, 6])
deq.popleft()         # remove from LEFT:   returns 5 → deque([1, 2, 3, 6])
deq.pop()             # remove from RIGHT:  returns 6 → deque([1, 2, 3])
```

**Queue vs Stack behavior:**

```
QUEUE (FIFO) — append() + popleft():
  append(1), append(2), append(3)  →  [1, 2, 3]
  popleft() → 1  (first in, first out)

STACK (LIFO) — append() + pop():
  append(1), append(2), append(3)  →  [1, 2, 3]
  pop() → 3  (last in, first out)
```

```python
# Queue (FIFO) — BFS, task scheduling
q = deque()
q.append(1)       # enqueue
q.append(2)
q.popleft()       # dequeue → 1 (first in, first out)

# Stack (LIFO) — DFS, undo
s = deque()
s.append(1)       # push
s.append(2)
s.pop()           # pop → 2 (last in, first out)
```

**Other useful operations:**

```python
deq = deque([1, 2, 3, 4, 5])

deq.rotate(2)     # rotate RIGHT by 2:  deque([4, 5, 1, 2, 3])
deq.rotate(-2)    # rotate LEFT by 2:   deque([1, 2, 3, 4, 5])

len(deq)          # 5
deq.extend([6,7]) # add multiple to RIGHT:  deque([1, 2, 3, 4, 5, 6, 7])
deq.extendleft([0, -1])  # add multiple to LEFT (reversed!): deque([-1, 0, 1, ...])

# Bounded deque — auto-drops oldest when full (sliding window!)
window = deque(maxlen=3)
window.append(1)  # deque([1])
window.append(2)  # deque([1, 2])
window.append(3)  # deque([1, 2, 3])
window.append(4)  # deque([2, 3, 4])  ← 1 dropped automatically
```

**CP pattern — sliding window max/min:**

```python
from collections import deque

def maxSlidingWindow(nums, k):
    """Max of every window of size k"""
    dq = deque()     # stores INDICES, front = index of current max
    result = []
    for i, num in enumerate(nums):
        while dq and dq[0] < i - k + 1:    # remove indices outside window
            dq.popleft()
        while dq and nums[dq[-1]] < num:    # remove smaller elements from back
            dq.pop()
        dq.append(i)
        if i >= k - 1:                      # window is full
            result.append(nums[dq[0]])      # front of deque = max
    return result

# nums = [1, 3, -1, -3, 5, 3, 6, 7], k = 3
# result = [3, 3, 5, 5, 6, 7]
```
