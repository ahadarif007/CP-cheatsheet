## Basic Calculator

Simulate a calculator with a stack. Handles `+`, `-`, `*`, `/`, and parentheses.

The strategy: track current `sign` and `num`. When we hit an operator or closing paren, "commit" the previous number with its sign to the stack. For multiplication/division, combine with top of stack immediately.

```python
class Solution:
    def calculate(self, s: str) -> int:
        s += '$'                     # sentinel to flush last number
        def helper(stk, i):
            sign = '+'              # initial sign is positive
            num = 0                 # current number being built
            while i < len(s):
                c = s[i]
                if c == " ":        # skip whitespace
                    i += 1
                    continue
                elif c.isdigit():   # build multi-digit number
                    num = num * 10 + int(c)
                    i += 1
                elif c == '(':      # recursion: evaluate sub-expression
                    num, i = helper([], i+1)  # returns (result, new_index)
                else:               # operator or ')' or '$': commit previous number
                    if sign == '+':
                        stk.append(num)
                    if sign == '-':
                        stk.append(-num)
                    if sign == '*':
                        stk.append(stk.pop() * num)   # combine with previous
                    if sign == '/':
                        stk.append(int(stk.pop() / num))  # int() truncates toward zero
                    i += 1
                    num = 0
                    if c == ')':
                        return sum(stk), i   # return sub-expression result
                    sign = c                 # remember operator for next number
            return sum(stk)         # final result = sum of all committed values
        return helper([], 0)
```
