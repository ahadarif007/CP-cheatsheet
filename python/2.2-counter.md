## Counter

`Counter` is a specialized dictionary for counting things. Keys are elements, values are counts. Very useful for frequency-based problems.

```python
from collections import Counter

count = Counter("hello")  # Counter({'l': 2, 'h': 1, 'e': 1, 'o': 1})
count['l']        # 2 (access count for any element)
count['l'] += 1   # manually increment
count['l']        # 3
count['z']        # 0 (missing keys return 0, not KeyError!)
```

**Creating Counters:**

```python
Counter("aabbc")                  # from string:  {'a': 2, 'b': 2, 'c': 1}
Counter([1, 1, 2, 3, 3, 3])      # from list:    {3: 3, 1: 2, 2: 1}
Counter({'a': 3, 'b': 1})        # from dict:    {'a': 3, 'b': 1}
Counter(a=3, b=1)                # from kwargs:  {'a': 3, 'b': 1}
```

**most_common(k)** — get the k most frequent elements as (element, count) tuples:

```python
count = Counter([1, 1, 1, 2, 2, 3])
count.most_common(2)    # [(1, 3), (2, 2)]  ← top 2 by frequency
count.most_common()     # [(1, 3), (2, 2), (3, 1)]  ← all, sorted by frequency

# CP: Top K Frequent Elements
def topKFrequent(nums, k):
    return [val for val, cnt in Counter(nums).most_common(k)]
# topKFrequent([1,1,1,2,2,3], 2)  →  [1, 2]
```

**elements()** — expand counts back into individual elements:

```python
c = Counter(a=2, b=3)
list(c.elements())  # ['a', 'a', 'b', 'b', 'b']

# CP: Intersection of Two Arrays (with duplicates)
def intersect(nums1, nums2):
    return list((Counter(nums1) & Counter(nums2)).elements())
# intersect([1,2,2,1], [2,2])  →  [2, 2]
```

**Arithmetic operators** — set-like operations on counts:

```
c = Counter(a=3, b=1)
d = Counter(a=1, b=2)

Operation    Result                    What it does
─────────    ──────                    ────────────
c + d        Counter(a=4, b=3)        ADD counts
c - d        Counter(a=2)             SUBTRACT (drops zero/negative)
c & d        Counter(a=1, b=1)        MIN of each (intersection)
c | d        Counter(a=3, b=2)        MAX of each (union)
```

```python
c = Counter(a=3, b=1)
d = Counter(a=1, b=2)
c + d   # Counter({'a': 4, 'b': 3})   add counts
c - d   # Counter({'a': 2})            subtract (drops zero and negative)
c & d   # Counter({'a': 1, 'b': 1})    intersection (min of each)
c | d   # Counter({'a': 3, 'b': 2})    union (max of each)

# Unary +/- filters by sign
c = Counter(a=2, b=-4)
+c      # Counter({'a': 2})     keep only POSITIVE counts
-c      # Counter({'b': 4})     negate, then keep only positive
```

**CP patterns:**

```python
# Check if two strings are anagrams
Counter("listen") == Counter("silent")  # True

# Check if ransom note can be built from magazine
def canConstruct(ransomNote, magazine):
    return not (Counter(ransomNote) - Counter(magazine))
    # subtraction drops zeros → empty Counter is falsy

# Frequency-based grouping
count = Counter([1,1,2,2,2,3])
# count = {2: 3, 1: 2, 3: 1}
# total() returns sum of all counts
count.total()   # 6  (Python 3.10+)
```
