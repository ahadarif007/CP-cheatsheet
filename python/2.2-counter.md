## Counter

`Counter` is a specialized dictionary for counting things. Keys are elements, values are counts. It provides many useful methods for frequency-based problems.

```python
from collections import Counter
count = Counter("hello")  # Counter({'l': 2, 'h': 1, 'e': 1, 'o': 1})
count['l']        # 2 (access count for any element)
count['l'] += 1   # manually increment
count['l']        # 3
count['z']        # 0 (missing keys return 0, not KeyError!)
```

**most_common(k)** - get the k most frequent elements as (element, count) tuples:

```python
# nums = [1,1,1,2,2,3], k = 2
def topKFrequent(self, nums: List[int], k: int) -> List[int]:
    if len(nums) == k:
        return nums
    # Counter(nums).most_common(k) = [(1, 3), (2, 2)] (top k by frequency)
    return [n[0] for n in Counter(nums).most_common(k)]  # [1, 2] (just the elements)
```

**elements()** - iterate through each element repeated by its count:

```python
def intersect(self, nums1: List[int], nums2: List[int]) -> List[int]:
    c1 = collections.Counter(nums1)  # {1:2, 2:2}
    c2 = collections.Counter(nums2)  # {2:2}
    dif = c1 & c2                    # intersection: min count for each key = {2:2}
    return list(dif.elements())      # [2, 2] (expand counts back to elements)
```

**Arithmetic operators on Counters** - very powerful for set-like operations on counts:

```python
c = Counter(a=3, b=1)
d = Counter(a=1, b=2)
c + d   # Counter({'a': 4, 'b': 3})   ADD counts
c - d   # Counter({'a': 2})            SUBTRACT (drops zero and negative)
c & d   # Counter({'a': 1, 'b': 1})    INTERSECTION (min of each)
c | d   # Counter({'a': 3, 'b': 2})    UNION (max of each)

c = Counter(a=2, b=-4)
+c      # Counter({'a': 2})     keep only POSITIVE counts
-c      # Counter({'b': 4})     negate and keep only positive results
```
