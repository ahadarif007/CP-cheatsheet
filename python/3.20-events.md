## Events (Sweep Line)

The sweep line pattern flattens intervals into +1/-1 events, sorts by time, then sweeps. Useful for: meeting rooms needed, peak concurrent users, employee free time, overlapping intervals.

```
Meetings:  A[1───5]   B[2────6]   C[4──7]

Timeline events:
  time:  1    2    4    5    6    7
event: +1   +1   +1   -1   -1   -1

Active count (balance):
         1    2    3    2    1    0
                   ^
              peak = 3 concurrent meetings

Free time = gaps where balance == 0 (after time 7)
```

```python
def employeeFreeTime(self, schedule: '[[Interval]]') -> '[Interval]':
    events = []
    for emp in schedule:
        for m in emp:
            events.append((m.start, 1))    # +1: someone becomes busy
            events.append((m.end, -1))     # -1: someone becomes free
    events.sort()                          # sort by time (ties: -1 before +1)

    result = []
    prev = None
    balance = 0                            # how many people are currently busy
    for time, change in events:
        if balance == 0 and prev is not None and time != prev:
            result.append(Interval(prev, time))  # gap where nobody is busy = free time
        balance += change
        prev = time
    return result
```
