## Dynamic Programming

DP solves problems by breaking them into overlapping subproblems, solving each once, and storing results. Key indicators: "optimal" (min/max), overlapping subproblems, choice at each step.

```
Top-down (memoization) vs Bottom-up (tabulation):

  Top-down:   start from the answer, recurse down, cache results
  Bottom-up:  start from base cases, build up to the answer

  Top-down is usually easier to write; bottom-up avoids recursion stack
```

**Coin Change** - minimum coins to make an amount (unbounded knapsack):

```
  coins = [1, 3, 4],  amount = 6

  dp[i] = min coins to make amount i
  dp:  [0, 1, 2, 1, 1, 2, 2]
        ↑  ↑  ↑  ↑  ↑  ↑  ↑
        0  1  1+1 3  4  4+1 3+3

  dp[6] = min(dp[6-1]+1, dp[6-3]+1, dp[6-4]+1) = min(3, 2, 3) = 2
  Answer: 2 coins (3 + 3)
```

```python
def coinChange(self, coins: List[int], amount: int) -> int:
    MAX = float('inf')
    dp = [MAX] * (amount + 1)     # dp[i] = min coins for amount i
    dp[0] = 0                      # base case: 0 coins for amount 0

    for c in coins:                # try each coin denomination
        for a in range(c, amount + 1):
            dp[a] = min(dp[a], dp[a - c] + 1)   # use coin c or don't

    return dp[amount] if dp[amount] != MAX else -1
```

**Longest Common Subsequence (LCS)** - classic 2D DP:

```
  text1 = "abcde",  text2 = "ace"

  dp[i][j] = LCS length of text1[:i] and text2[:j]

        ""  a  c  e
    ""   0  0  0  0
    a    0  1  1  1     a==a → dp[0][0]+1 = 1
    b    0  1  1  1     b≠c  → max(left, up)
    c    0  1  2  2     c==c → dp[1][1]+1 = 2
    d    0  1  2  2
    e    0  1  2  3     e==e → dp[3][2]+1 = 3  ← answer!
             ↗
  Match: take diagonal + 1
  No match: take max(left, up)
```

```python
def longestCommonSubsequence(self, text1: str, text2: str) -> int:
    X = len(text1) + 1
    Y = len(text2) + 1
    dp = [[0] * Y for _ in range(X)]

    for i, c in enumerate(text1):
        for j, d in enumerate(text2):
            if c == d:
                dp[i+1][j+1] = 1 + dp[i][j]                  # match → diagonal + 1
            else:
                dp[i+1][j+1] = max(dp[i][j+1], dp[i+1][j])   # no match → best of skip either

    return dp[-1][-1]
```
