## Trie

A Trie (prefix tree) stores strings as paths from root to leaves. Each node is a dict of children. Good for autocomplete, spell check, IP routing, word games.

```
After adding "app", "api", "bat":

         (root)
        /      \
      'a'      'b'
       |        |
      'p'      'a'
      / \       |
    'p'  'i'   't'
     |    |     |
  #=app #=api #=bat        ('#' marks end-of-word, stores full word)
```

```python
class Trie:
    def __init__(self):
        self.root = {}

    def addWord(self, word: str):
        node = self.root
        for c in word:
            if c not in node:
                node[c] = {}
            node = node[c]
        node['#'] = word   # '#' = end-of-word marker, value = full word for easy retrieval

    def matchPrefix(self, prefix: str, node=None):
        """Return all words that start with prefix"""
        if not node:
            node = self.root
        for c in prefix:
            if c not in node:
                return []
            node = node[c]

        result = []
        for key in node:
            if key == '#':
                result.append(node[key])
            else:
                result += self.matchPrefix('', node[key])  # recurse into children
        return result

    def hasPrefix(self, prefix: str):
        """Check if any word starts with prefix"""
        node = self.root
        for c in prefix:
            if c not in node:
                return False
            node = node[c]
        return True
```

**Wildcard search** (`.` matches any character) - uses `'#'` as end-of-word:

```python
def search(self, word: str) -> bool:
    def dfs(word, node):
        for i, c in enumerate(word):
            if c in node:
                node = node[c]
            elif c == '.':                  # wildcard: try ALL children
                return any(dfs(word[i+1:], node[ch]) for ch in node if ch != '#')
            else:
                return False
        return '#' in node
    return dfs(word, self.root)
```
