## Trie

A Trie (prefix tree) is a tree where each path from root to a node represents a prefix. Each node is a dictionary of children. Good for autocomplete, spell checker, IP routing (longest prefix match), word games.

Structure: nested dictionaries. '#' marks end-of-word.

```python
class Trie:
    def __init__(self):
        self.root = {}   # root is an empty dict; children are nested dicts
        # Example after adding "app": {'a': {'p': {'p': {'#': 'app'}}}}

    def addWord(self, s: str):
        tmp = self.root
        for c in s:                    # walk down the trie
            if c not in tmp:           # character not in current node's children
                tmp[c] = {}            # create new child node (empty dict)
            tmp = tmp[c]               # move to child
        tmp['#'] = s   # mark end-of-word with '#', store full word for easy retrieval

    def matchPrefix(self, s: str, tmp=None):
        """Find all words that start with prefix s"""
        if not tmp: tmp = self.root
        for c in s:
            if c not in tmp:
                return []              # prefix not in trie
            tmp = tmp[c]               # walk down following prefix

        rtn = []
        for k in tmp:                  # collect all words below this node
            if k == '#':
                rtn.append(tmp[k])     # found a complete word
            else:
                rtn += self.matchPrefix('', tmp[k])  # recurse into children
        return rtn

    def hasWord(self, s: str):
        """Check if any word starts with prefix s"""
        tmp = self.root
        for c in s:
            if c in tmp:
                tmp = tmp[c]
            else:
                return False
        return True
```

**Wildcard search** (`.` matches any character):

```python
def search(self, word: str) -> bool:
    def searchNode(word, node):
        for i, c in enumerate(word):
            if c in node:
                node = node[c]              # exact character match
            elif c == '.':                  # wildcard: try ALL children
                return any(searchNode(word[i+1:], node[cn]) for cn in node if cn != '$')
            else:
                return False               # character not found, no wildcard
        return '$' in node                 # check if we're at end-of-word
    return searchNode(word, self.trie)
```
