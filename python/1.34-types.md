## Types

Python type hints help document what types functions expect. They don't enforce types at runtime but help IDEs and type checkers catch bugs.

```python
from typing import List, Set, Dict, Tuple, Optional
# List[int] = list of integers
# Dict[str, int] = dict with string keys and int values
# Optional[int] = int or None
# Tuple[int, str] = tuple of (int, str)
```
