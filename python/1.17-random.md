## Random

Python's `random` module provides various randomization functions.

**Common random functions:**

```Python
import random
ints = [0, 1, 2]
random.choice(ints)              # randomly pick one element: 0, 1, or 2
random.choices([1,2,3], [1,1,10])  # weighted random: 3 is 10x more likely
random.randint(0, 2)            # random integer in [0, 2] (INCLUSIVE both ends)
random.randrange(0, 2)          # random integer in [0, 2) = 0 or 1 (EXCLUSIVE end)
random.shuffle(ints)            # shuffle list IN-PLACE, returns None
random.sample(ints, 2)          # pick 2 unique elements (without replacement)
random.random()                 # random float in [0.0, 1.0)
```

**`randint` vs `randrange`** - the key difference is the endpoint:

```
randint(a, b)    →  [a, b]   both ends included
randrange(a, b)  →  [a, b)   end excluded (like range())
randrange(0, 0)  →  ERROR!   empty range
```

**Fisher-Yates shuffle** - produces a uniformly random permutation in O(n):

```
  [A, B, C, D]     Start: entire array is "unshuffled"
   ↑                i=0: swap A with random element from [0..3]
      ↑             i=1: swap B with random element from [1..3]
         ↑          i=2: swap C with random element from [2..3]
                    i=3: done — each element had equal chance at every position
```

```Python
def shuffle(arr):
    for i in range(len(arr)):
        r = random.randrange(i, len(arr))   # random index from [i, end)
        arr[i], arr[r] = arr[r], arr[i]
    return arr
```
