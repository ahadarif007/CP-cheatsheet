## Random

Python's `random` module provides various randomization functions.

**Fisher-Yates shuffle** - produces a uniformly random permutation:

```Python
for i, l in enumerate(shuffle):
  # Pick random index from remaining unshuffled portion [i, end]
  r = random.randrange(0+i, len(shuffle))
  shuffle[i], shuffle[r] = shuffle[r], shuffle[i]  # swap current with random
return shuffle
```

**Common random functions:**

```Python
import random
ints = [0, 1, 2]
random.choice(ints)              # randomly pick one element: 0, 1, or 2
random.choices([1,2,3], [1,1,10])  # weighted random: 3 is 10x more likely
random.randint(0, 2)            # random integer in [0, 2] (INCLUSIVE both ends)
random.randint(0, 0)            # always returns 0
random.randrange(0, 0)          # ERROR! empty range (start must be < stop)
random.randrange(0, 2)          # random integer in [0, 2) = 0 or 1 (EXCLUSIVE end)
random.shuffle(ints)            # shuffle list IN-PLACE, returns None
random.sample(ints, 2)          # pick 2 unique elements (without replacement)
random.random()                 # random float in [0.0, 1.0)
```
