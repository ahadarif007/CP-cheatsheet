## Regular Expression

The `re` module provides pattern matching in strings. Useful for complex string manipulation:

```python
import re
def removeVowels(self, S: str) -> str:
    return re.sub('a|e|i|o|u', '', S)  # replace any vowel with empty string
    # 'a|e|i|o|u' means "match a OR e OR i OR o OR u"
    # '' means replace with nothing (delete the match)
```
