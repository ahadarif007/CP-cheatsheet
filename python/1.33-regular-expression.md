## Regular Expression

The `re` module provides pattern matching in strings.

**Common patterns for CP:**

```python
import re

# Remove all vowels
re.sub('[aeiou]', '', "hello world")      # "hll wrld"

# Find all numbers in a string
re.findall(r'\d+', "abc 123 def 456")     # ['123', '456']

# Find all words
re.findall(r'[a-zA-Z]+', "hi! how? ok.")  # ['hi', 'how', 'ok']

# Split on multiple delimiters
re.split(r'[,;:\s]+', "a,b;c:d e")       # ['a', 'b', 'c', 'd', 'e']

# Check if entire string matches a pattern
re.fullmatch(r'\d{3}-\d{4}', "123-4567") # Match object (truthy)
re.fullmatch(r'\d{3}-\d{4}', "12-4567")  # None (falsy)
```

**Quick regex reference:**

```
\d    digit [0-9]            \D    non-digit
\w    word char [a-zA-Z0-9_] \W    non-word
\s    whitespace             \S    non-whitespace
.     any char (except \n)
+     one or more            *     zero or more
?     zero or one            {n}   exactly n times
[abc] any of a, b, c         [^abc] NOT a, b, c
^     start of string        $     end of string
a|b   a OR b                 ()    capture group
```
