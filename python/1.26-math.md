## Math

**Power with modulo** - essential for large number problems (prevents overflow):

```python
d = pow(a, b, p)   # (a ^ b) % p — much faster than (a**b) % p
```

**Division with quotient AND remainder:**

```python
divmod(8, 3)   # (2, 2)  → 8 = 3*2 + 2  (quotient=2, remainder=2)
divmod(3, 8)   # (0, 3)  → 3 = 8*0 + 3  (quotient=0, remainder=3)
```

**math module essentials:**

```python
import math

math.gcd(12, 8)          # 4 (greatest common divisor)
math.lcm(4, 6)           # 12 (least common multiple, Python 3.9+)
math.isqrt(10)           # 3 (integer square root, floor)
math.sqrt(10)            # 3.1622... (float square root)
math.ceil(3.2)           # 4 (round up)
math.floor(3.8)          # 3 (round down)
math.log2(8)             # 3.0
math.log10(1000)         # 3.0
math.inf                 # positive infinity (useful as initial max/min)
math.factorial(5)        # 120
math.comb(5, 2)          # 10  (5 choose 2, Python 3.8+)
math.perm(5, 2)          # 20  (5 permute 2, Python 3.8+)
```

**GCD of multiple numbers:**

```python
from math import gcd
from functools import reduce

reduce(gcd, [12, 18, 24])  # 6
# Python 3.9+: math.gcd(12, 18, 24)  — accepts multiple args directly
```
