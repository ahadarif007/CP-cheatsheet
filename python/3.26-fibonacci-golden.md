## Fibonacci Golden

Fibonacci numbers can be calculated in O(1) using the [golden ratio](https://demonstrations.wolfram.com/GeneralizedFibonacciSequenceAndTheGoldenRatio/):

```python
def fib(self, N: int) -> int:
    golden_ratio = (1 + 5 ** 0.5) / 2     # phi = (1 + sqrt(5)) / 2 ≈ 1.618
    return int((golden_ratio ** N + 1) / 5 ** 0.5)
    # This is Binet's formula - works for N up to about 70 before float precision issues
```
