## Fibonacci Golden

Fibonacci numbers can be calculated in O(1) using Binet's formula with the [golden ratio](https://demonstrations.wolfram.com/GeneralizedFibonacciSequenceAndTheGoldenRatio/):

```python
def fib(self, N: int) -> int:
    golden_ratio = (1 + 5 ** 0.5) / 2     # phi ≈ 1.618
    return int((golden_ratio ** N + 1) / 5 ** 0.5)
```

Works for N up to ~70 before float precision issues. For larger N, use matrix exponentiation or iterative DP.
