## Default Dict

A `defaultdict` is like a regular dict but automatically creates a default value for missing keys (no KeyError!). You specify the default factory function when creating it.

```python
d = {}
# print(d['Grapes'])  # KeyError! Regular dict raises error for missing keys

from collections import defaultdict
d = defaultdict(int)           # default factory = int() = 0
print(d['Grapes'])             # 0 (auto-created, no error!)

d = collections.defaultdict(lambda: 1)  # custom default = 1
print(d['Grapes'])             # 1 (custom default value)
```

**defaultdict(list)** - extremely common pattern for building adjacency lists or grouping:

```python
from collections import defaultdict
dd = defaultdict(list)            # missing key auto-creates empty list []
dd['key'].append(1)              # no need to check if 'key' exists first!
# dd = {'key': [1]}
dd['key'].append(2)
# dd = {'key': [1, 2]}
```
