## Default Dict

A `defaultdict` is like a regular dict but automatically creates a default value for missing keys (no KeyError!). You specify the default factory function when creating it.

**The problem it solves:**

```
Regular dict:
  d = {}
  d['key'].append(1)   →  KeyError! 'key' doesn't exist yet

defaultdict(list):
  d = defaultdict(list)
  d['key'].append(1)   →  works! auto-creates [] then appends
```

```python
from collections import defaultdict

d = defaultdict(int)           # missing key → int() → 0
d['apples']                    # 0 (auto-created, no error!)
d['apples'] += 5               # 5

d = defaultdict(list)          # missing key → list() → []
d['fruits'].append('apple')    # auto-creates [], then appends

d = defaultdict(set)           # missing key → set() → set()
d['colors'].add('red')         # auto-creates set(), then adds

d = defaultdict(lambda: 1)    # missing key → 1 (custom default)
d['anything']                  # 1
```

**defaultdict(list)** — the most common pattern, great for grouping and adjacency lists:

```python
from collections import defaultdict

# Grouping: group words by first letter
words = ["apple", "bat", "bar", "atom", "cat"]
groups = defaultdict(list)
for w in words:
    groups[w[0]].append(w)
# {'a': ['apple', 'atom'], 'b': ['bat', 'bar'], 'c': ['cat']}

# Adjacency list for graphs
edges = [[0,1], [0,2], [1,2], [2,3]]
graph = defaultdict(list)
for u, v in edges:
    graph[u].append(v)
    graph[v].append(u)     # undirected: add both directions
# {0: [1, 2], 1: [0, 2], 2: [0, 1, 3], 3: [2]}
```

```
Graph built from edges:         Adjacency list:
  0 --- 1                       0 → [1, 2]
  |   / |                       1 → [0, 2]
  |  /  |                       2 → [0, 1, 3]
  2 --- 3                       3 → [2]
```

**defaultdict(int)** — counting pattern (though Counter is often better):

```python
# Count character frequencies
freq = defaultdict(int)
for ch in "hello":
    freq[ch] += 1
# {'h': 1, 'e': 1, 'l': 2, 'o': 1}
```

**defaultdict vs setdefault vs manual check:**

```python
# These all do the same thing — defaultdict is cleanest:

# Manual check (verbose)
d = {}
if key not in d:
    d[key] = []
d[key].append(val)

# setdefault (one-liner but reads oddly)
d = {}
d.setdefault(key, []).append(val)

# defaultdict (cleanest)
d = defaultdict(list)
d[key].append(val)
```

**CP pattern — nested defaultdict:**

```python
# 2D frequency map (e.g., count pairs)
pair_count = defaultdict(lambda: defaultdict(int))
pair_count['a']['b'] += 1
pair_count['a']['c'] += 1
# {'a': {'b': 1, 'c': 1}}
```
