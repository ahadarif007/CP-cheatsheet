# Trie (Prefix Tree)

> A trie stores strings character by character in a tree structure.
> Each node's children represent the next possible characters.
> Enables O(L) prefix search where L = string length.

## Trie Structure

```
Words: "cat", "car", "dog"

root
 ├── c
 │   └── a
 │       ├── t (#="cat")
 │       └── r (#="car")
 └── d
     └── o
         └── g (#="dog")
```

## Trie with Prefix Matching

```js
class Trie {
  constructor() {
    this.root = {};
  }

  addWord(word) {
    let node = this.root;
    for (const char of word) {
      if (!node[char]) node[char] = {};
      node = node[char];
    }
    node['#'] = word;                    // mark end of word, store full word
  }

  matchPrefix(prefix, startNode) {
    let node = startNode || this.root;
    for (const char of prefix) {
      if (!node[char]) return [];        // no words with this prefix
      node = node[char];
    }

    // Collect all words under this prefix node
    const results = [];
    const collect = (current) => {
      for (const key in current) {
        if (key === '#') results.push(current[key]);
        else collect(current[key]);
      }
    };
    collect(node);
    return results;
  }

  hasWord(word) {
    let node = this.root;
    for (const char of word) {
      if (!node[char]) return false;
      node = node[char];
    }
    return '#' in node;
  }
}
```

## Word Dictionary with Wildcard Search

The `.` character matches any single letter.

```js
class WordDictionary {
  constructor() {
    this.root = {};
  }

  addWord(word) {
    let node = this.root;
    for (const char of word) {
      if (!node[char]) node[char] = {};
      node = node[char];
    }
    node['$'] = true;                    // end-of-word marker
  }

  search(word) {
    const dfs = (index, node) => {
      if (index === word.length) return !!node['$'];

      const char = word[index];
      if (char !== '.') {
        return node[char] && dfs(index + 1, node[char]);
      }

      // '.' matches any character
      for (const key in node) {
        if (key !== '$' && dfs(index + 1, node[key])) return true;
      }
      return false;
    };

    return dfs(0, this.root);
  }
}
```
