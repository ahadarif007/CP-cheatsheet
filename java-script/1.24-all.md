# All (every)

> `Array.every()` returns `true` if **all** elements satisfy the predicate.
> Equivalent to Python's `all()`. Short-circuits on the first failure.

```js
const allNonNeg = nums.every((x) => x >= 0);      // true if no negatives
const allLong   = words.every((w) => w.length > 3); // true if all words > 3 chars
```
