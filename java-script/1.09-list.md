# List (Array)

> JavaScript arrays are dynamic -- they grow/shrink automatically.
> Access is O(1) by index, push/pop are amortized O(1), shift/unshift are O(n).

## Create and Initialize

```js
const zeros    = Array(5).fill(0);                         // [0, 0, 0, 0, 0]
const grid     = Array.from({ length: 3 }, () => []);      // [[], [], []] -- independent rows
const prefixed = Array.from({ length: 5 }, (_, i) => i);   // [0, 1, 2, 3, 4]
```

## Common Operations

```js
list.push(value);                    // append to end       O(1)
list.pop();                          // remove from end     O(1)
list.unshift(value);                 // prepend to front    O(n)
list.shift();                        // remove from front   O(n)
list.reverse();                      // reverse in place
```

## Merge Two Arrays

```js
const merged = [...listA, ...listB]; // non-destructive concatenation
```

## 2D Grid Initialization

```
grid = [

  [],   // row 1
]
grid[0].push(1) -> [[1], []]    (rows are independent)
```
