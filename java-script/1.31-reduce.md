# Reduce

> `Array.reduce()` collapses an array into a single value by applying an
> accumulator function from left to right. The second argument is the initial value.

```js
const sum     = nums.reduce((acc, val) => acc + val, 0);
// [1, 2, 3, 4] -> 10

const product = nums.reduce((acc, val) => acc * val, 1);
// [2, 3, 4] -> 24

const max     = nums.reduce((acc, val) => Math.max(acc, val), -Infinity);
// [3, 1, 4, 1, 5] -> 5
```

## Flatten with Reduce

```js
const flat = nested.reduce((acc, arr) => acc.concat(arr), []);
// [[1,2], [3,4]] -> [1, 2, 3, 4]
```
