# Hash (Frequency Map)

> A `Map` is the standard way to build a frequency counter in JavaScript.
> Unlike plain objects, Map keys can be any type and iteration order is guaranteed.

## Build a Character Frequency Map

```
Input:  "banana"
Map:    { 'b' => 1, 'a' => 3, 'n' => 2 }
```

```js
const freq = new Map();
for (const char of text) {
  freq.set(char, (freq.get(char) || 0) + 1);  // increment count
}
```

## Common Map Operations

```js
freq.has('a');          // true if key exists
freq.get('a');          // value, or undefined
freq.delete('a');       // remove key
freq.size;              // number of entries
```
