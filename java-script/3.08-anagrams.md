# Anagrams

> Two strings are anagrams if they contain the same characters with the same frequencies.
> Use a frequency array of size 26 (for lowercase letters) for O(n) checks.

## Check if Two Strings are Anagrams

```js
function isAnagram(strA, strB) {
  if (strA.length !== strB.length) return false;

  const charCount = Array(26).fill(0);
  for (const ch of strA) charCount[ch.charCodeAt(0) - 97]++;     // increment
  for (const ch of strB) {
    if (--charCount[ch.charCodeAt(0) - 97] < 0) return false;   // decrement, check
  }
  return true;
}
```

## Find All Anagram Start Indices (Sliding Window)

Slide a window of size `pattern.length` over the text. Compare frequency arrays.

```
text:    "cbaebabacd"
pattern: "abc"

Window slides (size 3):
  [cba]ebabacd  -> freq matches "abc" -> add index 0
  c[bae]babacd  -> no match
  cb[aeb]abacd  -> no match
  ...
  cbaeba[bac]d  -> freq matches "abc" -> add index 6

Result: [0, 6]
```

```js
function findAnagrams(text, pattern) {
  const need = Array(26).fill(0);
  for (const ch of pattern) need[ch.charCodeAt(0) - 97]++;

  const have    = Array(26).fill(0);
  const indices = [];
  const winSize = pattern.length;

  for (let i = 0; i < text.length; i++) {
    have[text.charCodeAt(i) - 97]++;                             // add right char

    if (i >= winSize) have[text.charCodeAt(i - winSize) - 97]--; // remove left char

    if (need.every((count, j) => count === have[j])) {
      indices.push(i - winSize + 1);
    }
  }
  return indices;
}
```
