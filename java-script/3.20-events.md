# Events (Line Sweep)

> The line sweep technique converts intervals into start/end events,
> sorts them by time, and processes them to find gaps or overlaps.

## Employee Free Time

Find gaps when no employee is busy. Create +1 events for starts and -1 for ends.

```
Employee 1: [1,3], [6,7]
Employee 2: [2,4]
Employee 3: [5,8]

Events sorted: (1,+1) (2,+1) (3,-1) (4,-1) (5,+1) (6,+1) (7,-1) (8,-1)
Balance:        1      2      1      0       1      2      1      0
                                     ^-- gap at [4,5]

Free time: [[4, 5]]
```

```js
function employeeFreeTime(schedules) {
  const events = [];
  for (const employee of schedules) {
    for (const meeting of employee) {
      events.push([meeting.start,  1]);    // +1 = someone becomes busy
      events.push([meeting.end,   -1]);    // -1 = someone becomes free
    }
  }
  events.sort((a, b) => a[0] - b[0]);

  const freeIntervals = [];
  let prevTime = null, busyCount = 0;

  for (const [time, change] of events) {
    if (busyCount === 0 && prevTime !== null && time !== prevTime) {
      freeIntervals.push([prevTime, time]);    // gap found
    }
    busyCount += change;
    prevTime = time;
  }
  return freeIntervals;
}
```
