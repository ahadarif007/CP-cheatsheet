# Map (Transform)

> `Array.map()` creates a **new array** by applying a function to every element.
> Does not mutate the original. Equivalent to Python's `map()` or list comprehension.

```js
const uppercased = words.map((word) => word.toUpperCase());
// ["hello", "world"] -> ["HELLO", "WORLD"]

const doubled = nums.map((n) => n * 2);
// [1, 2, 3] -> [2, 4, 6]

const lengths = words.map((word) => word.length);
// ["hi", "hello"] -> [2, 5]
```
