# Lambda (Arrow Functions)

> Arrow functions `() => {}` are JavaScript's lambda syntax. They are commonly used
> as inline callbacks for array methods like `map`, `filter`, `reduce`, and `sort`.

## Find Closest Value to Target

```js
const closest = [valueA, valueB].reduce(
  (best, current) =>
    Math.abs(target - current) < Math.abs(target - best) ? current : best
);
```

## Common Arrow Function Patterns

```js
// Single expression (implicit return)
const double = (x) => x * 2;

// Multi-line (explicit return)
const clamp = (val, lo, hi) => {
  if (val < lo) return lo;
  if (val > hi) return hi;
  return val;
};

// As a comparator
nums.sort((a, b) => a - b);
```
