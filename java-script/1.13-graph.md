# Graph

> Graphs are represented as adjacency lists. For `N` nodes with edges `[u, v]`,
> build an array of neighbor lists. DFS uses recursion/stack, BFS uses a queue.

## Adjacency List Construction

```
Edges: [[0,1], [0,2], [1,3]]

Node 0 -> [1, 2]
Node 1 -> [0, 3]
Node 2 -> [0]
Node 3 -> [1]
```

```js
const adjList = Array.from({ length: nodeCount }, () => []);
for (const [nodeA, nodeB] of edges) {
  adjList[nodeA].push(nodeB);         // undirected: add both directions
  adjList[nodeB].push(nodeA);
}
```

## Depth-First Search (DFS)

```js
function dfs(node, parent) {
  for (const neighbor of adjList[node]) {
    if (neighbor !== parent) {        // avoid revisiting parent in undirected graph
      dfs(neighbor, node);
    }
  }
}
```

## Breadth-First Search (BFS)

```
BFS from node 0 on graph: 0--1--3, 0--2

Queue steps:
  [0]     -> visit 0, enqueue 1, 2
  [1, 2]  -> visit 1, enqueue 3
  [2, 3]  -> visit 2
  [3]     -> visit 3

Order: [0, 1, 2, 3]
```

```js
function bfs(startNode) {
  const order   = [];
  const visited = Array(nodeCount).fill(false);
  const queue   = [startNode];
  visited[startNode] = true;

  for (let qi = 0; qi < queue.length; qi++) {
    const node = queue[qi];
    order.push(node);

    for (const neighbor of adjList[node]) {
      if (!visited[neighbor]) {
        visited[neighbor] = true;
        queue.push(neighbor);
      }
    }
  }
  return order;
}
```
