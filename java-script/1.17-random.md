# Random

> `Math.random()` returns a float in [0, 1). Combine with `Math.floor()`
> to get random integers. Use Fisher-Yates for unbiased shuffling.

## Random Integer in Range [lo, hi)

```js
const randomInt = Math.floor(Math.random() * (hi - lo)) + lo;
```

## Fisher-Yates Shuffle (in-place, O(n))

Produces a uniformly random permutation by swapping each element with
a randomly chosen element from the remaining unshuffled portion.

```
[A, B, C, D]
 i=3: swap D with random(0..3)  -> [A, D, C, B]
 i=2: swap C with random(0..2)  -> [C, D, A, B]
 i=1: swap D with random(0..1)  -> [D, C, A, B]
```

```js
function shuffle(arr) {
  for (let i = arr.length - 1; i > 0; i--) {
    const randomIdx = Math.floor(Math.random() * (i + 1));
    [arr[i], arr[randomIdx]] = [arr[randomIdx], arr[i]];
  }
  return arr;
}
```
