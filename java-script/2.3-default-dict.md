# Default Dict (Auto-Initializing Map)

> Python's `defaultdict` auto-creates values for missing keys. In JavaScript,
> use a helper function that initializes the value on first access.

## Get-or-Initialize Helper

```js
function getOrInit(map, key, factory) {
  if (!map.has(key)) map.set(key, factory());
  return map.get(key);
}
```

## Example: Group Values by Key

```
Input:  [[1, 'a'], [2, 'b'], [1, 'c']]
Result: { 1 => ['a', 'c'], 2 => ['b'] }
```

```js
const groups = new Map();
getOrInit(groups, 1, () => []).push('a');   // groups: {1: ['a']}
getOrInit(groups, 2, () => []).push('b');   // groups: {1: ['a'], 2: ['b']}
getOrInit(groups, 1, () => []).push('c');   // groups: {1: ['a', 'c'], 2: ['b']}
```

## Alternative: Nullish Coalescing Pattern

```js
const adjacency = new Map();
if (!adjacency.has(node)) adjacency.set(node, []);
adjacency.get(node).push(neighbor);
```
