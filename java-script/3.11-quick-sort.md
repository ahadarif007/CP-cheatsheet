# Quick Sort

> Quick sort picks a pivot, partitions elements into < pivot and >= pivot,
> then recursively sorts each side. Average O(n log n), worst O(n^2).

## Algorithm Visualization

```
[3, 6, 8, 10, 1, 2, 1]   pivot = 1 (last element)

Partition (elements < pivot go left):
  [1, 6, 8, 10, 3, 2, 1]   swap 3 and 1
  ... after partition: [1, 1, 8, 10, 3, 2, 6]  pivot at index 1

Recursively sort left [1] and right [8, 10, 3, 2, 6]
```

```js
function quickSort(arr) {
  sortRange(arr, 0, arr.length - 1);
}

function sortRange(arr, left, right) {
  if (left >= right) return;
  const pivotIdx = partition(arr, left, right);
  sortRange(arr, left, pivotIdx - 1);     // sort left partition
  sortRange(arr, pivotIdx + 1, right);    // sort right partition
}

function partition(arr, left, right) {
  const pivotVal  = arr[right];           // choose last element as pivot
  let insertPos   = left;                 // where next smaller element goes

  for (let scan = left; scan < right; scan++) {
    if (arr[scan] < pivotVal) {
      [arr[scan], arr[insertPos]] = [arr[insertPos], arr[scan]];
      insertPos++;
    }
  }
  [arr[right], arr[insertPos]] = [arr[insertPos], arr[right]];  // place pivot
  return insertPos;
}
```
