# Tuple (Pair / Fixed-Size Grouping)

> JavaScript has no built-in tuple type. Use a small array or an object literal
> to group related values. Destructuring makes either approach ergonomic.

## Array-Style Pair

```js
const pair = [2, 3];
const [x, y] = pair;                   // destructure: x = 2, y = 3
```

## Object-Style Pair (self-documenting)

```js
const point = { row: 2, col: 3 };
const { row, col } = point;            // destructure by name
```

## Returning Multiple Values

```js
function minMax(nums) {
  return [Math.min(...nums), Math.max(...nums)];
}
const [lo, hi] = minMax([5, 1, 9]);    // lo = 1, hi = 9
```
