# Binary Search Tree (BST)

> A BST maintains the invariant: left.val < node.val < right.val for every node.
> Inorder traversal of a valid BST yields values in sorted order.

## Complete Tree Check by Index

A complete binary tree has all levels fully filled except possibly the last,
which is filled from left to right. Assign index `1` to root; left child = `2*i`,
right child = `2*i + 1`. The tree is complete if max index equals total node count.

```
Complete:           Not complete:
      1                   1
     / \                 / \
    2   3               2   3
   / \                   \
  4   5                   5
indexes: 1,2,3,4,5      indexes: 1,2,3,5
max=5, count=5 OK        max=5, count=4 FAIL
```

```js
function isComplete(root) {
  let totalNodes = 0n, maxIndex = 0n;

  (function dfs(node, index) {
    if (!node) return;
    totalNodes++;
    if (index > maxIndex) maxIndex = index;
    dfs(node.left,  index * 2n);
    dfs(node.right, index * 2n + 1n);
  })(root, 1n);

  return totalNodes === maxIndex;
}
```

## Range Sum in BST

Sum all node values in range [low, high]. Prune branches that can't contain values in range.

```js
function rangeSumBST(root, low, high) {
  let sum = 0;

  const dfs = (node) => {
    if (!node) return;
    if (low <= node.val && node.val <= high) sum += node.val;
    if (node.val > low)  dfs(node.left);    // left subtree may have values >= low
    if (node.val < high) dfs(node.right);   // right subtree may have values <= high
  };

  dfs(root);
  return sum;
}
```

## Validate BST (Iterative Inorder)

Inorder traversal of a valid BST produces strictly increasing values.

```
Valid BST:        Invalid BST:
    2                 2
   / \               / \
  1   3             3   1
inorder: 1,2,3     inorder: 3,2,1 (not increasing)
```

```js
function isValidBST(root) {
  const stack = [];
  let current   = root;
  let prevValue = null;

  while (current || stack.length) {
    while (current) {
      stack.push(current);
      current = current.left;
    }
    current = stack.pop();
    if (prevValue !== null && current.val <= prevValue) return false;
    prevValue = current.val;
    current   = current.right;
  }
  return true;
}
```
