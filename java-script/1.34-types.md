# Types (Common Data Structures)

> Quick reference for initializing common typed structures in competitive programming.

## 2D Grid (R rows x C columns)

```js
const grid = Array.from({ length: rows }, () => Array(cols).fill(0));
```

## Adjacency List (Map-based)

```js
const adjList = new Map();  // Map<number, number[]>
```

## Type Checking

```js
typeof 42;             // "number"
typeof 'hi';           // "string"
typeof true;           // "boolean"
typeof undefined;      // "undefined"
typeof null;           // "object" (historical quirk)
Array.isArray([1,2]);  // true
```
