# Iter (Iterators)

> Any object with a `[Symbol.iterator]` method is iterable in JavaScript.
> Arrays, strings, Maps, Sets are all iterable. Use `for...of` to consume them.

## For-Of Loop (preferred)

```js
for (const value of list) {
  // process each value
}
```

## Manual Iterator Protocol

```js
const iterator = list[Symbol.iterator]();
const first    = iterator.next();        // { value: ..., done: false }
const second   = iterator.next();        // { value: ..., done: false }
// iterator.next() returns { done: true } when exhausted
```

## Iterating with Index (entries)

```js
for (const [index, value] of list.entries()) {
  // index = 0, 1, 2, ...
}
```
