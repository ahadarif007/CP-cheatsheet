# Deque (Double-Ended Queue)

> A deque supports O(1) insertion and removal at both ends. JavaScript arrays
> can serve as a deque, but `shift()`/`unshift()` are O(n). For competitive
> programming with large inputs, consider a linked-list-based deque.

```
Operations:
  pushFront(5)  [5]
  pushBack(6)   [5, 6]
  popFront()    [6]      returns 5
  popBack()     []       returns 6
```

```js
const deque = [];

deque.unshift(5);          // push to front   O(n)
deque.push(6);             // push to back    O(1)
const front = deque.shift();   // pop from front  O(n)
const back  = deque.pop();     // pop from back   O(1)
```

## Peek Without Removing

```js
const peekFront = deque[0];
const peekBack  = deque[deque.length - 1];
```
