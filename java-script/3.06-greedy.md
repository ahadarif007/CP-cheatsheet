# Greedy

> Greedy algorithms make the locally optimal choice at each step.
> They work when a problem has the greedy-choice property (local optimum leads to global optimum).

## Increasing Triplet Subsequence

Find if there exist indices i < j < k such that nums[i] < nums[j] < nums[k].
Track the smallest and second-smallest values seen so far.

```
Input: [2, 1, 5, 0, 4, 6]

Step through:
  x=2: first=2, second=INF
  x=1: first=1, second=INF     (1 < 2, update first)
  x=5: first=1, second=5       (5 > first, update second)
  x=0: first=0, second=5       (0 < 1, update first)
  x=4: first=0, second=4       (4 > first but < 5, update second)
  x=6: 6 > second(4) -> FOUND TRIPLET! return true
```

```js
function increasingTriplet(nums) {
  let first  = Infinity;     // smallest value seen
  let second = Infinity;     // smallest value greater than first

  for (const num of nums) {
    if (num <= first) first = num;          // update smallest
    else if (num <= second) second = num;   // update second smallest
    else return true;                       // found third value > second > first
  }
  return false;
}
```
