# Loops

```js
for (let i = 0; i < n; i++) {}
for (let i = n - 1; i >= 0; i--) {}
for (const x of nums) {
}
let i = 0;
while (i < n) {
  i++;
}
```

Reverse in-place

```js
let l = 0,
  r = nums.length - 1;
while (l < r) {
  [nums[l], nums[r]] = [nums[r], nums[l]];
  l++;
  r--;
}
```

Ranges

```js
for (let i = 0; i < 3; i++) {}
for (let i = 3; i >= 0; i--) {}
```
