# Loops

> JavaScript provides `for`, `for...of`, `while`, and `do...while` loops.
> Use `for...of` to iterate values; use classic `for` when you need the index.

## Classic For Loop

```js
// Forward: 0, 1, 2, ..., n-1
for (let index = 0; index < length; index++) { }

// Backward: n-1, n-2, ..., 0
for (let index = length - 1; index >= 0; index--) { }
```

## For-Of Loop (iterate values)

```js
for (const value of nums) {
  // process each value
}
```

## While Loop

```js
let index = 0;
while (index < length) {
  index++;
}
```

## Reverse an Array In-Place (Two Pointers)

```
Before: [1, 2, 3, 4, 5]
         L ->       <- R     swap(L, R), then move inward
After:  [5, 4, 3, 2, 1]
```

```js
let left = 0, right = nums.length - 1;
while (left < right) {
  [nums[left], nums[right]] = [nums[right], nums[left]]; // swap
  left++;
  right--;
}
```

## Range Patterns

```js
// Equivalent of range(0, 3) -> 0, 1, 2
for (let i = 0; i < 3; i++) { }

// Equivalent of range(3, -1, -1) -> 3, 2, 1, 0
for (let i = 3; i >= 0; i--) { }
```
