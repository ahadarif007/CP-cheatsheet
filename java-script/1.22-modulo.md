# Modulo

> JavaScript's `%` operator can return negative results for negative dividends
> (e.g., `-7 % 3 === -1`). Use this helper to get a non-negative remainder,
> which is needed for circular array indexing and modular arithmetic.

```
Standard %:     -7 % 3 = -1      (follows sign of dividend)
Safe mod:       mod(-7, 3) = 2   (always non-negative)
```

```js
function mod(value, modulus) {
  const remainder = value % modulus;
  return remainder < 0 ? remainder + modulus : remainder;
}
```

## Common Use: Circular Array Index

```js
const nextIndex = mod(currentIndex + step, arr.length);
```
