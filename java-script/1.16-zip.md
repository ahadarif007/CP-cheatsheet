# Zip (Parallel Iteration)

> JavaScript has no built-in `zip`. Use a shared index loop to iterate
> two arrays in parallel. Stop at the shorter array's length.

## Parallel Iteration

```js
for (let i = 0; i < Math.min(listA.length, listB.length); i++) {
  const a = listA[i];
  const b = listB[i];
  // process pair (a, b)
}
```

## Rotate Matrix 90 Degrees Clockwise

Transpose the matrix (swap rows and columns), then reverse each row.

```
Original:       Transpose:      Reverse rows:
1  2  3         1  4  7         7  4  1
4  5  6    ->   2  5  8    ->   8  5  2
7  8  9         3  6  9         9  6  3
```

```js
function rotateMatrix(matrix) {
  const size = matrix.length;

  // Step 1: transpose (swap matrix[i][j] with matrix[j][i])
  for (let i = 0; i < size; i++) {
    for (let j = i + 1; j < size; j++) {
      const temp    = matrix[i][j];
      matrix[i][j]  = matrix[j][i];
      matrix[j][i]  = temp;
    }
  }

  // Step 2: reverse each row
  for (let i = 0; i < size; i++) {
    matrix[i].reverse();
  }
}
```
