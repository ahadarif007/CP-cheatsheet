# Continuous Subarrays with Sum k

> Count subarrays whose elements sum to exactly k.
> Uses a prefix sum + hash map approach in O(n) time.

## Key Insight

If `prefixSum[j] - prefixSum[i] = k`, then subarray `(i, j]` sums to k.
Store how many times each prefix sum has occurred.

```
nums = [1, 1, 1], k = 2

prefix:  0  1  2  3
               ^  ^
         prefixSum[2] - prefixSum[0] = 2 = k  -> count++
               prefixSum[3] - prefixSum[1] = 2 = k  -> count++
Answer: 2
```

```js
function subarraySum(nums, k) {
  const prefixCount = new Map([[0, 1]]);   // base case: empty prefix sums to 0
  let count     = 0;
  let prefixSum = 0;

  for (const num of nums) {
    prefixSum += num;
    count += prefixCount.get(prefixSum - k) || 0;           // how many earlier prefixes match
    prefixCount.set(prefixSum, (prefixCount.get(prefixSum) || 0) + 1);
  }
  return count;
}
```
