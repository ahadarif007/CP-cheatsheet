# Counter (Frequency Map)

> JavaScript has no built-in Counter class like Python. Use a `Map` to count
> occurrences of each element.

## Count Character Frequencies

```
Input:  "hello"
Result: { 'h' => 1, 'e' => 1, 'l' => 2, 'o' => 1 }
```

```js
const counter = new Map();
for (const char of text) {
  counter.set(char, (counter.get(char) || 0) + 1);
}
```

## Top-K Most Frequent Elements

Sort entries by count descending, then take the first k.

```
Input:  nums = [1,1,1,2,2,3], k = 2
Step 1: Count -> {1:3, 2:2, 3:1}
Step 2: Sort by frequency desc -> [[1,3], [2,2], [3,1]]
Step 3: Take first k -> [1, 2]
```

```js
function topKFrequent(nums, k) {
  const freqMap = new Map();
  for (const num of nums) {
    freqMap.set(num, (freqMap.get(num) || 0) + 1);
  }

  return Array.from(freqMap.entries())
    .sort((a, b) => b[1] - a[1])      // sort by frequency descending
    .slice(0, k)                       // take top k
    .map((entry) => entry[0]);         // extract the element
}
```
