# Sliding Window

> The sliding window pattern maintains a window [left, right] over an array/string,
> expanding right to include new elements and shrinking left to maintain constraints.
> Solves substring/subarray problems in O(n).

## Longest Substring Without Repeating Characters

```
Input: "abcabcbb"

Window slides:
  [a]bcabcbb       len=1
  [ab]cabcbb       len=2
  [abc]abcbb       len=3
  a[bca]bcbb       len=3  (hit duplicate 'a', move left past first 'a')
  ab[cab]cbb       len=3
  abc[abc]bb       len=3
  ...
Answer: 3
```

```js
function lengthOfLongestSubstring(text) {
  const lastSeen = Array(256).fill(-1);  // last index of each ASCII char
  let maxLen = 0, left = 0;

  for (let right = 0; right < text.length; right++) {
    const charCode = text.charCodeAt(right);

    // If char was seen inside current window, shrink window past it
    if (lastSeen[charCode] >= left) {
      left = lastSeen[charCode] + 1;
    }

    lastSeen[charCode] = right;
    maxLen = Math.max(maxLen, right - left + 1);
  }
  return maxLen;
}
```
