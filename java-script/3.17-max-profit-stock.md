# Max Profit Stock

> Stock problems use state machines: `notHolding` (cash) and `holding` (own stock).
> At each price, decide to buy, sell, or hold.

## State Machine

```
           buy (-price)
  notHolding ---------> holding
      ^                    |
      |    sell (+price)   |
      +--------------------+
```

## Infinite Transactions (Buy/Sell Any Number of Times)

```js
function maxProfit(prices) {
  let notHolding = 0;              // max profit when not holding stock
  let holding    = -Infinity;      // max profit when holding stock

  for (const price of prices) {
    const prevNotHolding = notHolding;
    notHolding = Math.max(notHolding, holding + price);     // sell
    holding    = Math.max(holding, prevNotHolding - price);  // buy
  }
  return notHolding;
}
```

## Single Transaction (Buy Once, Sell Once)

```js
function maxProfitOnce(prices) {
  let notHolding = 0;
  let holding    = -Infinity;

  for (const price of prices) {
    notHolding = Math.max(notHolding, holding + price);  // sell
    holding    = Math.max(holding, -price);               // buy (from 0 profit)
  }
  return notHolding;
}
```

## At Most K Transactions

```js
function maxProfitK(maxTxns, prices) {
  const notHolding = Array(maxTxns + 1).fill(0);
  const holding    = Array(maxTxns + 1).fill(-1e15);

  for (const price of prices) {
    for (let txn = maxTxns; txn >= 1; txn--) {
      notHolding[txn] = Math.max(notHolding[txn], holding[txn] + price);
      holding[txn]    = Math.max(holding[txn], notHolding[txn - 1] - price);
    }
  }
  return notHolding[maxTxns];
}
```
