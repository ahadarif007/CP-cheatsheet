# Constants

> JavaScript provides special numeric constants for boundary values.
> Use these when initializing min/max trackers or representing infinity.

```js
const INF      = Number.POSITIVE_INFINITY;   // +Infinity
const NEG_INF  = Number.NEGATIVE_INFINITY;   // -Infinity
const MAX_SAFE = Number.MAX_SAFE_INTEGER;    // 2^53 - 1  (9007199254740991)
const MIN_SAFE = Number.MIN_SAFE_INTEGER;    // -(2^53 - 1)
```

## Common Usage

```js
let minVal = Infinity;       // will be replaced by any real number
let maxVal = -Infinity;      // will be replaced by any real number

for (const num of nums) {
  minVal = Math.min(minVal, num);
  maxVal = Math.max(maxVal, num);
}
```
