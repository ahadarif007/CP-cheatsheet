# Ternary Operator

> The ternary `condition ? valueIfTrue : valueIfFalse` is JavaScript's inline
> conditional expression. Use it for simple assignments; prefer `if/else` for
> complex logic.

```js
const result = condition ? valueA : valueB;
```

## Examples

```js
const absVal  = x >= 0 ? x : -x;
const label   = score >= 60 ? 'pass' : 'fail';
const clamped = val > max ? max : (val < min ? min : val);
```
