# Any (some)

> `Array.some()` returns `true` if **at least one** element satisfies the predicate.
> Equivalent to Python's `any()`. Short-circuits on the first match.

```js
const hasPositive = nums.some((x) => x > 0);    // true if any element > 0
const hasEmpty    = words.some((w) => w === ''); // true if any word is empty
```
