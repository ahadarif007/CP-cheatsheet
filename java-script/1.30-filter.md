# Filter

> `Array.filter()` creates a **new array** with only the elements that pass
> the predicate. Does not mutate the original.

```js
const passing = scores.filter((score) => score > 75);
// [90, 50, 80, 60] -> [90, 80]

const evens = nums.filter((n) => n % 2 === 0);
// [1, 2, 3, 4] -> [2, 4]

const nonEmpty = words.filter((w) => w.length > 0);
// ["hi", "", "bye"] -> ["hi", "bye"]
```
