# Convert Base

> Convert a number to a different base by repeatedly dividing and collecting remainders.
> For negative numbers in hexadecimal, use two's complement (add 2^32).

## Integer to Hexadecimal

```
Input: 26
  26 / 16 = 1 remainder 10 ('a')
   1 / 16 = 0 remainder  1 ('1')
Result: "1a"

Input: -1 (two's complement: 2^32 - 1 = 4294967295)
  4294967295 in hex = "ffffffff"
```

```js
function toHex(num) {
  if (num === 0) return '0';
  if (num < 0) num = 2 ** 32 + num;         // two's complement for negatives

  const hexDigits = '0123456789abcdef';
  let result = '';

  while (num > 0) {
    const digit = num % 16;
    result = hexDigits[digit] + result;      // prepend digit
    num = Math.floor(num / 16);
  }
  return result;
}
```
