# Slicing

> `slice(start, end)` returns a shallow copy from index `start` up to (but not including)
> `end`. Works on both strings and arrays. Negative indices count from the end.

```
Index:    0   1   2   3   4
Array:  [ a , b , c , d , e ]
              └── slice(1, 4) ──┘  -> [b, c, d]
```

## String Slicing

```js
const substring = text.slice(2, 4);       // characters at index 2 and 3
const fromEnd   = text.slice(-3);         // last 3 characters
```

## Array Slicing

```js
const subArray = arr.slice(2, 5);         // elements at index 2, 3, 4
const copy     = arr.slice();             // shallow copy of entire array
```
