# Merge K Sorted Arrays

> Merge k sorted arrays into one sorted array using a min-heap.
> The heap always holds one element from each array, and we repeatedly
> extract the minimum. Time: O(N log k) where N = total elements.

## Algorithm Visualization

```
Arrays: [[1, 4, 7],
         [2, 5, 8],
         [3, 6, 9]]

Heap (value, arrayIdx, elementIdx):
  Init:  [(1,0,0), (2,1,0), (3,2,0)]
  Pop 1, push 4 from array 0:  [(2,1,0), (3,2,0), (4,0,1)]
  Pop 2, push 5 from array 1:  [(3,2,0), (4,0,1), (5,1,1)]
  ...
Result: [1, 2, 3, 4, 5, 6, 7, 8, 9]
```

```js
function mergeKSortedArrays(arrays) {
  const minHeap = new MinHeap((a, b) => a[0] - b[0]);

  // Initialize heap with the first element of each array
  for (let arrIdx = 0; arrIdx < arrays.length; arrIdx++) {
    if (arrays[arrIdx].length > 0) {
      minHeap.push([arrays[arrIdx][0], arrIdx, 0]);  // [value, arrayIndex, elementIndex]
    }
  }

  const result = [];
  while (minHeap.size()) {
    const [value, arrIdx, elemIdx] = minHeap.pop();
    result.push(value);

    // If this array has more elements, push the next one
    if (elemIdx + 1 < arrays[arrIdx].length) {
      minHeap.push([arrays[arrIdx][elemIdx + 1], arrIdx, elemIdx + 1]);
    }
  }
  return result;
}
```
