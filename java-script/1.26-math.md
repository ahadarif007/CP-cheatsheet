# Math

> JavaScript numbers are 64-bit doubles. Use `BigInt` for arbitrary precision.
> `Math.trunc()` truncates toward zero (like integer division in most languages).

## Modular Exponentiation (BigInt)

```js
const result = BigInt(base) ** BigInt(exp) % BigInt(modulus);
```

## Integer Division (truncate toward zero)

```
 7 / 2  -> 3.5  -> Math.trunc(3.5)  = 3
-7 / 2  -> -3.5 -> Math.trunc(-3.5) = -3
```

```js
const quotient = Math.trunc(a / b);
```

## Non-Negative Remainder

```js
const remainder = ((a % b) + b) % b;    // always >= 0
```

## Other Useful Math Functions

```js
Math.abs(-5);          // 5
Math.max(1, 2, 3);     // 3
Math.min(1, 2, 3);     // 1
Math.floor(2.7);       // 2   (round down)
Math.ceil(2.1);        // 3   (round up)
Math.sqrt(16);         // 4
Math.log2(8);          // 3
```
