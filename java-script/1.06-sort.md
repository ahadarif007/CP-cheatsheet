# Sort

> `Array.sort()` sorts **in place** and defaults to lexicographic order.
> Always pass a comparator `(a, b) => a - b` for numeric sorting.
> Returns negative to place `a` first, positive for `b` first, zero for equal.

## Numeric Sort

```js
nums.sort((a, b) => a - b);                 // ascending:  [1, 3, 5, 9]
nums.sort((a, b) => b - a);                 // descending: [9, 5, 3, 1]
```

## Sort a Subarray (first k elements)

```js
const sortedPrefix = nums.slice(0, k).sort((a, b) => a - b);
```

## Sort by Custom Key

```js
// Sort strings by length
words.sort((a, b) => a.length - b.length);

// Sort pairs by second element, break ties by first
pairs.sort((a, b) => a[1] - b[1] || a[0] - b[0]);
```

## Sort a Map by Value

```js
const freqMap = new Map(Object.entries({ a: 2, b: 1, c: 3 }));
const sorted  = Array.from(freqMap.entries())
  .sort((x, y) => x[1] - y[1]);           // ascending by value
// -> [["b", 1], ["a", 2], ["c", 3]]
```

## Keep Original Indices After Sort

Useful for problems that ask "which index had the k-th smallest value?"

```js
const indexedValues = nums
  .map((value, index) => [index, value])
  .sort((a, b) => a[1] - b[1]);           // sort by value, keep original index
```
