# Fast Power (Binary Exponentiation)

> Compute x^n in O(log n) by squaring the base and halving the exponent.
> If the exponent bit is 1, multiply the result by the current base.

## How It Works

```
x = 2, n = 10 (binary: 1010)

Step   exponent(binary)  base    result
----   ----------------  ----    ------
init   1010              2       1
bit=0  0101              4       1       (even: just square base)
bit=1  0010              16      4       (odd: multiply result by base)
bit=0  0001              256     4       (even: just square base)
bit=1  0000              65536   1024    (odd: multiply result by base)

Answer: 2^10 = 1024
```

```js
function myPow(x, n) {
  let exponent = BigInt(n);
  let base     = Number.isInteger(x) ? BigInt(x) : x;

  if (exponent < 0n) {
    base     = 1 / Number(base);
    exponent = -exponent;
  }

  let result    = 1;
  let baseValue = Number(base);

  while (exponent > 0n) {
    if ((exponent & 1n) === 1n) result *= baseValue;   // odd exponent: multiply
    baseValue *= baseValue;                             // square the base
    exponent >>= 1n;                                    // halve the exponent
  }
  return result;
}
```
