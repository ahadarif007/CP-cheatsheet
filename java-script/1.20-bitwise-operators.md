# Bitwise Operators

> Bitwise operators work on 32-bit integers. Useful for flags, masks,
> and competitive programming tricks. `>>>` is unsigned right shift.

## Operator Reference

```
Operator   Name              Example (binary)           Result
--------   ----              ----------------           ------
  &        AND               1100 & 1010                1000
  |        OR                1100 | 1010                1110
  ^        XOR               1100 ^ 1010                0110
  >>       Right shift       1100 >> 2                  0011
  <<       Left shift        0011 << 2                  1100
  >>>      Unsigned right    -1 >>> 1                   2147483647
  ~        NOT (complement)  ~0b1100                    ...0011
```

```js
const andResult = 0b1100 & 0b1010;    // 0b1000 = 8
const orResult  = 0b1100 | 0b1010;    // 0b1110 = 14
const xorResult = 0b1100 ^ 0b1010;    // 0b0110 = 6
const shrResult = 0b1100 >> 2;        // 0b0011 = 3
const shlResult = 0b0011 << 2;        // 0b1100 = 12
const ushr      = -1 >>> 1;          // 2147483647 (all bits except sign)
```

## Common Bit Tricks

```js
const isEven    = (n & 1) === 0;           // check last bit
const isPow2    = n > 0 && (n & (n-1)) === 0;  // power of 2 check
const swapBit   = n ^ mask;                // toggle bits in mask
const clearLow  = n & (n - 1);            // clear lowest set bit
const lowestBit = n & (-n);               // isolate lowest set bit
```
