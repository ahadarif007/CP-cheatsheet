# Dutch National Flag (3-Way Partition)

> Sort an array of 0s, 1s, and 2s in one pass using three pointers.
> Also known as the Dutch National Flag problem (Dijkstra).

## How It Works

```
Input: [2, 0, 2, 1, 1, 0]

Three regions:  [0..zeroEnd)  [zeroEnd..current)  (twoStart..end]
                   all 0s         all 1s              all 2s

Step through:
  [2, 0, 2, 1, 1, 0]   z=0, c=0, t=5
   c=0: val=2, swap with t=5 -> [0, 0, 2, 1, 1, 2]  t=4
   c=0: val=0, swap with z=0 -> [0, 0, 2, 1, 1, 2]  z=1, c=1
   c=1: val=0, swap with z=1 -> [0, 0, 2, 1, 1, 2]  z=2, c=2
   c=2: val=2, swap with t=4 -> [0, 0, 1, 1, 2, 2]  t=3
   c=2: val=1, skip           -> c=3
   c=3: val=1, skip           -> c=4  (c > t, done)

Result: [0, 0, 1, 1, 2, 2]
```

```js
function sortColors(nums) {
  let zeroEnd = 0;                    // next position for 0
  let current = 0;                    // scanning pointer
  let twoStart = nums.length - 1;    // next position for 2

  while (current <= twoStart) {
    if (nums[current] === 0) {
      [nums[zeroEnd], nums[current]] = [nums[current], nums[zeroEnd]];
      zeroEnd++;
      current++;
    } else if (nums[current] === 2) {
      [nums[current], nums[twoStart]] = [nums[twoStart], nums[current]];
      twoStart--;                     // don't advance current -- swapped value needs checking
    } else {
      current++;                      // 1 is already in the middle section
    }
  }
}
```
