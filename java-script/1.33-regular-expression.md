# Regular Expression

> JavaScript regex uses `/pattern/flags` syntax. Common flags: `g` (global),
> `i` (case-insensitive), `m` (multiline). Use `test()` to check, `match()` to extract.

## Remove All Vowels

```js
const noVowels = text.replace(/[aeiou]/g, '');
// "hello" -> "hll"
```

## Common Patterns

```js
// Test if string matches
/^\d+$/.test('123');            // true -- all digits

// Extract matches
'a1b2c3'.match(/\d/g);          // ["1", "2", "3"]

// Replace with callback
'hello'.replace(/[aeiou]/g, (ch) => ch.toUpperCase());
// "hEllO"

// Split by regex
'one, two;  three'.split(/[,;]\s*/);  // ["one", "two", "three"]
```
