# Binary Search

> Binary search finds a target in O(log n) by halving the search space each step.
> Works on sorted arrays or monotonic predicates (true/false boundary).

## Template: First Bad Version

Find the smallest index where `isBad(m)` is true.

```
Versions:  1  2  3  4  5  6  7
isBad:     F  F  F  T  T  T  T
                     ^-- answer = 4
```

```js
function firstBadVersion(totalVersions, isBad) {
  let left = 1, right = totalVersions;

  while (left < right) {
    const mid = left + ((right - left) >> 1);  // avoid overflow
    if (isBad(mid)) right = mid;               // answer is mid or earlier
    else left = mid + 1;                       // answer is after mid
  }
  return left;
}
```

## Template: Integer Square Root

Find the largest integer whose square is <= x.

```
x = 8
Candidates: 1  2  3  4 ...
Squares:    1  4  9  16
                ^-- 2*2=4 <= 8, 3*3=9 > 8 => answer = 2
```

```js
function mySqrt(x) {
  if (x <= 1) return x;
  let left = 1, right = x;

  while (left < right) {
    const mid = left + ((right - left) >> 1);
    if (mid * mid > x) right = mid;    // too large
    else left = mid + 1;               // might be answer, keep searching right
  }
  return left - 1;                     // left is first value where mid*mid > x
}
```
