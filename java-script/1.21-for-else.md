# For-Else Pattern

> Python has `for...else` where the `else` runs only if the loop completes without
> `break`. JavaScript has no equivalent syntax, so use a boolean flag.

```js
let didBreak = false;
for (let i = 0; i < length; i++) {
  if (condition) {
    didBreak = true;
    break;
  }
}
if (!didBreak) {
  // runs only if the loop finished without breaking
}
```

## Alternative: Use a Function with Early Return

```js
function findFirst(nums, target) {
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] === target) return i;    // found -> early exit
  }
  return -1;                              // "else" branch: not found
}
```
