# Kadane's Algorithm

> Finds the maximum sum contiguous subarray in O(n).
> At each position, decide: extend the current subarray or start fresh.

```
Input: [-2, 1, -3, 4, -1, 2, 1, -5, 4]

Index:   0   1   2   3   4   5   6   7   8
Value:  -2   1  -3   4  -1   2   1  -5   4
Sum:    -2   1  -2   4   3   5   6   1   5
Best:   -2   1   1   4   4   5   6   6   6
                         ^-----------^
                    max subarray: [4, -1, 2, 1] = 6
```

```js
function maxSubArray(nums) {
  let maxSum = nums[0];

  for (let i = 1; i < nums.length; i++) {
    if (nums[i - 1] > 0) nums[i] += nums[i - 1];  // extend if previous sum positive
    maxSum = Math.max(maxSum, nums[i]);
  }
  return maxSum;
}
```
