# Dynamic Programming

> DP solves problems by breaking them into overlapping subproblems and storing results.
> Two approaches: top-down (recursion + memoization) or bottom-up (tabulation).

## Coin Change (Minimum Coins)

Find the fewest coins needed to make `amount`. Unbounded: each coin can be used multiple times.

```
coins = [1, 3, 4], amount = 6

dp[i] = min coins to make amount i
dp: [0, 1, 2, 1, 1, 2, 2]
     ^  ^  ^  ^  ^  ^  ^
     0  1  1+1 3  4  4+1 3+3

Answer: dp[6] = 2 (coins 3+3)
```

```js
function coinChange(coins, amount) {
  const IMPOSSIBLE = 1e9;
  const minCoins   = Array(amount + 1).fill(IMPOSSIBLE);
  minCoins[0] = 0;                              // base case: 0 coins for amount 0

  for (const coin of coins) {
    for (let amt = coin; amt <= amount; amt++) {
      minCoins[amt] = Math.min(minCoins[amt], minCoins[amt - coin] + 1);
    }
  }

  return minCoins[amount] >= IMPOSSIBLE ? -1 : minCoins[amount];
}
```

## Longest Common Subsequence (LCS)

```
a = "abcde", b = "ace"

DP table (dp[i][j] = LCS length of a[0..i-1] and b[0..j-1]):

      ""  a  c  e
  ""   0  0  0  0
  a    0  1  1  1    a == a -> dp[0][0]+1 = 1
  b    0  1  1  1
  c    0  1  2  2    c == c -> dp[1][1]+1 = 2
  d    0  1  2  2
  e    0  1  2  3    e == e -> dp[3][2]+1 = 3

Answer: 3 ("ace")
```

```js
function longestCommonSubsequence(textA, textB) {
  const lenA = textA.length, lenB = textB.length;
  const dp = Array.from({ length: lenA + 1 }, () => Array(lenB + 1).fill(0));

  for (let i = 0; i < lenA; i++) {
    for (let j = 0; j < lenB; j++) {
      dp[i + 1][j + 1] = textA[i] === textB[j]
        ? dp[i][j] + 1                                    // chars match: extend LCS
        : Math.max(dp[i][j + 1], dp[i + 1][j]);           // skip one char from either
    }
  }

  return dp[lenA][lenB];
}
```
