# Candy Crush (Remove Consecutive Duplicates)

> Remove groups of k consecutive identical characters, like Candy Crush.
> Uses a stack of [character, count] pairs. When count reaches k, pop the group.

## How It Works

```
Input: "deeedbbcccbdaa", k = 3

Stack trace:
  'd' -> [(d,1)]
  'e' -> [(d,1),(e,1)]
  'e' -> [(d,1),(e,2)]
  'e' -> [(d,1),(e,3)] -> count=3=k, POP! -> [(d,1)]
  'd' -> [(d,2)]
  'b' -> [(d,2),(b,1)]
  'b' -> [(d,2),(b,2)]
  'c' -> [(d,2),(b,2),(c,1)]
  'c' -> [(d,2),(b,2),(c,2)]
  'c' -> [(d,2),(b,2),(c,3)] -> POP! -> [(d,2),(b,2)]
  'b' -> [(d,2),(b,3)] -> POP! -> [(d,2)]
  'd' -> [(d,3)] -> POP! -> []
  'a' -> [(a,1)]
  'a' -> [(a,2)]

Result: "aa"
```

```js
function removeDuplicates(text, k) {
  const stack = [];    // each entry: [character, count]

  for (const char of text) {
    if (stack.length && stack[stack.length - 1][0] === char) {
      stack[stack.length - 1][1]++;
      if (stack[stack.length - 1][1] === k) stack.pop();   // remove group of k
    } else {
      stack.push([char, 1]);
    }
  }

  let result = '';
  for (const [char, count] of stack) {
    result += char.repeat(count);
  }
  return result;
}
```
