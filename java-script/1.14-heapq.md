# PriorityQueue (Min-Heap)

> JavaScript has no built-in heap. This MinHeap class provides O(log n) push/pop
> and O(1) peek. Pass a custom comparator to change ordering (e.g., max-heap).

## Heap Structure

```
Min-Heap array: [1, 3, 2, 7, 6, 4, 5]

        1             Parent of index i: (i-1) >> 1
       / \            Left child:        2*i + 1
      3   2           Right child:       2*i + 2
     / \ / \
    7  6 4  5
```

## MinHeap Implementation

```js
class MinHeap {
  constructor(comparator = (a, b) => a - b) {
    this.data       = [];
    this.comparator = comparator;
  }

  size() { return this.data.length; }
  peek() { return this.data[0]; }

  push(value) {
    this.data.push(value);
    this._siftUp(this.data.length - 1);
  }

  pop() {
    const heap = this.data;
    if (heap.length === 0) return undefined;

    const top  = heap[0];
    const last = heap.pop();
    if (heap.length) {
      heap[0] = last;                 // move last element to root
      this._siftDown(0);              // restore heap property
    }
    return top;
  }

  _siftUp(index) {
    const heap = this.data;
    while (index > 0) {
      const parentIdx = (index - 1) >> 1;
      if (this.comparator(heap[index], heap[parentIdx]) < 0) {
        [heap[index], heap[parentIdx]] = [heap[parentIdx], heap[index]];
        index = parentIdx;
      } else break;
    }
  }

  _siftDown(index) {
    const heap = this.data;
    while (true) {
      let leftIdx  = index * 2 + 1;
      let rightIdx = leftIdx + 1;
      let smallest = index;

      if (leftIdx < heap.length && this.comparator(heap[leftIdx], heap[smallest]) < 0)
        smallest = leftIdx;
      if (rightIdx < heap.length && this.comparator(heap[rightIdx], heap[smallest]) < 0)
        smallest = rightIdx;

      if (smallest !== index) {
        [heap[index], heap[smallest]] = [heap[smallest], heap[index]];
        index = smallest;
      } else break;
    }
  }
}
```

## Top K Frequent Elements

```
Input:  nums = [1,1,1,2,2,3], k = 2
Step 1: Count frequencies    -> {1:3, 2:2, 3:1}
Step 2: Keep min-heap of size k, pop smallest when size > k
Result: [1, 2]
```

```js
function topKFrequent(nums, k) {
  // Step 1: count frequencies
  const freqMap = new Map();
  for (const num of nums) freqMap.set(num, (freqMap.get(num) || 0) + 1);

  // Step 2: maintain a min-heap of size k (by frequency)
  const minHeap = new MinHeap((a, b) => a[1] - b[1]);
  for (const entry of freqMap.entries()) {
    minHeap.push(entry);
    if (minHeap.size() > k) minHeap.pop();    // evict least frequent
  }

  // Step 3: extract results
  const result = [];
  while (minHeap.size()) result.push(minHeap.pop()[0]);
  return result.reverse();
}
```
