# Dict (Map)

> `Map` is JavaScript's general-purpose dictionary. Keys can be any type.
> For string-only keys, plain objects `{}` also work but lack `.size` and guaranteed order.

## Basic Operations

```js
const dict = new Map();

dict.set('key', 1);                     // insert or update
const value = dict.get('key') ?? 0;     // get with default (0 if missing)
dict.has('key');                         // true if key exists
dict.delete('key');                      // remove entry
dict.size;                              // number of entries
```

## Iteration

```js
// Iterate keys only
for (const key of dict.keys()) { }

// Iterate key-value pairs
for (const [key, value] of dict.entries()) { }
```

## Get-or-Initialize Pattern

```js
if (!dict.has('key')) dict.set('key', 0);  // initialize if absent
dict.set('key', dict.get('key') + 1);      // then update
```
