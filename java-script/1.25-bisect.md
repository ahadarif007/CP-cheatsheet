# Bisect (Binary Search on Sorted Array)

> JavaScript has no built-in bisect. These two functions find insertion points
> in a **sorted** array in O(log n). Equivalent to Python's `bisect_left` / `bisect_right`.

## Lower Bound (bisect_left)

Returns the **leftmost** index where `target` can be inserted to keep the array sorted.
All elements before this index are strictly less than `target`.

```
Array:   [1, 3, 3, 5, 7]
Target:  3
Result:  1  (index of first 3)
```

```js
function lowerBound(sortedArr, target) {
  let left = 0, right = sortedArr.length;
  while (left < right) {
    const mid = (left + right) >> 1;
    if (sortedArr[mid] < target) left = mid + 1;
    else right = mid;
  }
  return left;
}
```

## Upper Bound (bisect_right)

Returns the **rightmost** index where `target` can be inserted to keep the array sorted.
All elements before this index are less than or equal to `target`.

```
Array:   [1, 3, 3, 5, 7]
Target:  3
Result:  3  (index after last 3)
```

```js
function upperBound(sortedArr, target) {
  let left = 0, right = sortedArr.length;
  while (left < right) {
    const mid = (left + right) >> 1;
    if (sortedArr[mid] <= target) left = mid + 1;
    else right = mid;
  }
  return left;
}
```
