# Fibonacci (Golden Ratio Formula)

> The N-th Fibonacci number can be computed in O(1) using the golden ratio
> (Binet's formula). Accurate for small N due to floating-point precision.

## Golden Ratio

```
phi = (1 + sqrt(5)) / 2 = 1.6180339...

F(N) = round(phi^N / sqrt(5))

F(0)=0, F(1)=1, F(2)=1, F(3)=2, F(4)=3, F(5)=5, F(6)=8, ...
```

```js
function fibonacci(n) {
  const phi = (1 + Math.sqrt(5)) / 2;     // golden ratio
  return Math.round(phi ** n / Math.sqrt(5));
}
```

## Iterative Approach (exact, for large N)

```js
function fibIterative(n) {
  if (n <= 1) return n;
  let prev = 0, curr = 1;
  for (let i = 2; i <= n; i++) {
    [prev, curr] = [curr, prev + curr];
  }
  return curr;
}
```
