# Merge Meetings (Insert Interval)

> Insert a new interval into a list of non-overlapping intervals,
> merging any that overlap. Sort all intervals by start time, then merge adjacent.

```
Existing: [[1,3], [6,9]]
New:      [2,5]
Merged:   [[1,5], [6,9]]

Timeline:
  1---3       6---9
    2---5
  [1-----5]  [6---9]   (merged [1,3] and [2,5])
```

```js
function insertInterval(intervals, newInterval) {
  const allIntervals = intervals.slice();
  allIntervals.push(newInterval);
  allIntervals.sort((a, b) => a[0] - b[0]);       // sort by start time

  const merged = [allIntervals[0]];
  for (const current of allIntervals) {
    const last = merged[merged.length - 1];
    if (current[0] <= last[1]) {
      last[1] = Math.max(last[1], current[1]);     // extend end if overlapping
    } else {
      merged.push(current);                        // no overlap, add new interval
    }
  }
  return merged;
}
```
