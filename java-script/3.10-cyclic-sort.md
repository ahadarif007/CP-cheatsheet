# Cyclic Sort

> When an array contains numbers in range [1, n], each value tells you its correct
> position (value `k` belongs at index `k-1`). Swap each element to its correct
> position in O(n) time and O(1) space.

```
Input:  [3, 1, 5, 4, 2]

Step through:
  i=0: nums[0]=3, correct pos=2, swap -> [5, 1, 3, 4, 2]
  i=0: nums[0]=5, correct pos=4, swap -> [2, 1, 3, 4, 5]
  i=0: nums[0]=2, correct pos=1, swap -> [1, 2, 3, 4, 5]
  i=0: nums[0]=1, correct pos=0, already there, i++
  i=1..4: all in place

Result: [1, 2, 3, 4, 5]
```

```js
function cyclicSort(nums) {
  let i = 0;
  while (i < nums.length) {
    const correctIdx = nums[i] - 1;                    // where nums[i] should go
    if (nums[i] !== nums[correctIdx]) {
      [nums[i], nums[correctIdx]] = [nums[correctIdx], nums[i]];  // swap to correct pos
    } else {
      i++;                                             // already in place, move on
    }
  }
  return nums;
}
```
