# Set

> A `Set` stores unique values with O(1) add, delete, and lookup.
> Useful for deduplication and membership checks.

```js
const seen = new Set();

seen.add(value);                     // insert
seen.delete(value);                  // remove
const exists = seen.has(value);      // O(1) membership check
const first  = seen.values().next().value;  // peek at any element
seen.size;                           // number of elements
```

## Common Patterns

```js
// Deduplicate an array
const unique = [...new Set(nums)];

// Set from array
const charSet = new Set('hello'.split(''));  // {'h','e','l','o'}

// Intersection of two sets
const both = new Set([...setA].filter(x => setB.has(x)));
```
