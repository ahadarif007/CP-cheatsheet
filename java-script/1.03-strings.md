# Strings

> Strings in JavaScript are immutable sequences of UTF-16 code units.
> Common operations return new strings -- they never modify the original.

## Searching

```js
const firstPos = text.indexOf('x');       // first occurrence, -1 if not found
const lastPos  = text.lastIndexOf('x');   // last occurrence, -1 if not found
```

## Splitting and Joining

```js
const tokens = line.split(':');            // "a:b:c" -> ["a", "b", "c"]
const joined = ['a', 'b'].join(' ');       // -> "a b"
```

## Character Inspection

```js
const char      = text.charAt(0);                  // first character
const isUpper   = char === char.toUpperCase();      // is uppercase?
const isLower   = char === char.toLowerCase();      // is lowercase?
const isDigit   = /\d/.test(char);                  // is digit 0-9?
```

## Replace and Check Prefix / Suffix

```js
const cleaned   = text.replaceAll('()', '');        // remove all "()"
const hasPrefix = text.startsWith('pre');           // starts with "pre"?
const hasSuffix = text.endsWith('suf');             // ends with "suf"?
```

## Template Literals (String Interpolation)

```js
const greeting = `My name is ${name}, I'm ${age}`;
```

## Manual Split by Letter Characters

Splits a string into words, where a "word" is a consecutive run of letters (A-Z, a-z).

```
Input:  "hello--world!!foo"
Output: ["hello", "world", "foo"]
```

```js
function splitByLetters(input) {
  const words = [];
  let wordStart = -1;

  for (let i = 0; i < input.length; i++) {
    const char = input[i];
    if (/[A-Za-z]/.test(char)) {
      if (wordStart === -1) wordStart = i;       // start of a new word
    } else {
      if (wordStart !== -1) {
        words.push(input.slice(wordStart, i));    // end of current word
        wordStart = -1;
      }
    }
  }
  if (wordStart !== -1) words.push(input.slice(wordStart)); // trailing word
  return words;
}
```
