# Union Find / Disjoint Set Union (DSU)

> Tracks connected components. `find(x)` returns the root of x's component.
> `union(a, b)` merges two components. With path compression and union by size,
> both operations are nearly O(1) amortized.

## How It Works

```
Initial: each node is its own parent
  parent: [0, 1, 2, 3, 4]

union(0, 1): parent[1] = 0       -> {0,1} {2} {3} {4}
union(2, 3): parent[3] = 2       -> {0,1} {2,3} {4}
union(0, 3): parent[2] = 0       -> {0,1,2,3} {4}

find(3): 3 -> parent[3]=2 -> parent[2]=0 (root)
         path compression: parent[3] = 0  (shortcut for next time)
```

```js
class DSU {
  constructor(count) {
    this.parent = Array(count).fill(0).map((_, i) => i);  // each node is its own root
    this.size   = Array(count).fill(1);                    // each component starts at size 1
  }

  find(node) {
    if (this.parent[node] !== node) {
      this.parent[node] = this.find(this.parent[node]);   // path compression
    }
    return this.parent[node];
  }

  union(nodeA, nodeB) {
    let rootA = this.find(nodeA);
    let rootB = this.find(nodeB);
    if (rootA === rootB) return false;                     // already connected

    // Union by size: attach smaller tree under larger tree
    if (this.size[rootA] < this.size[rootB]) [rootA, rootB] = [rootB, rootA];
    this.parent[rootB] = rootA;
    this.size[rootA]  += this.size[rootB];
    return true;
  }
}
```
