# Shift Array Right (Rotate)

> Rotate array right by k positions in O(n) time and O(1) space
> using three reversals.

## Three-Reversal Trick

```
Input:  [1, 2, 3, 4, 5, 6, 7], k = 3

Step 1: Reverse entire array    -> [7, 6, 5, 4, 3, 2, 1]
Step 2: Reverse first k(=3)     -> [5, 6, 7, 4, 3, 2, 1]
Step 3: Reverse remaining n-k   -> [5, 6, 7, 1, 2, 3, 4]
```

```js
function rotateRight(arr, k) {
  const length = arr.length;
  k %= length;                              // handle k > length
  reverse(arr, 0, length - 1);              // reverse all
  reverse(arr, 0, k - 1);                   // reverse first k
  reverse(arr, k, length - 1);              // reverse rest
}

function reverse(arr, left, right) {
  while (left < right) {
    [arr[left], arr[right]] = [arr[right], arr[left]];
    left++;
    right--;
  }
}
```
