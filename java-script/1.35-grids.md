# Grids (2D Matrix DFS/BFS)

> Grid problems model a 2D matrix as a graph where each cell connects to its
> 4 neighbors (up, down, left, right). Use DFS or BFS to explore connected regions.

## Grid Traversal Setup

```
Grid (R=3, C=4):
  0 1 1 0
  0 1 0 0         Directions: (row, col)
  1 1 0 0         Down(1,0) Up(-1,0) Right(0,1) Left(0,-1)
```

```js
const rows = grid.length;
const cols = grid[0].length;

const directions = [
  [1, 0],    // down
  [-1, 0],   // up
  [0, 1],    // right
  [0, -1],   // left
];
```

## DFS Flood Fill

Marks all connected `1` cells as visited (value `2`).

```js
function dfs(row, col) {
  grid[row][col] = 2;                       // mark visited

  for (const [dRow, dCol] of directions) {
    const newRow = row + dRow;
    const newCol = col + dCol;
    const inBounds = newRow >= 0 && newRow < rows && newCol >= 0 && newCol < cols;

    if (inBounds && grid[newRow][newCol] === 1) {
      dfs(newRow, newCol);
    }
  }
}
```
