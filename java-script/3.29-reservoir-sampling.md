# Reservoir Sampling

```js
class Picker {
  constructor(nums) {
    this.nums = nums;
  }
  pick(target) {
    let res = null,
      cnt = 0;
    for (let i = 0; i < this.nums.length; i++) {
      if (this.nums[i] === target) {
        cnt++;
        const chance = Math.floor(Math.random() * cnt) + 1;
        if (chance === 1) res = i;
      }
    }
    return res;
  }
}
```
