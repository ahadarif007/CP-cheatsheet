# Reservoir Sampling

> Randomly pick an element from a stream with equal probability, without
> knowing the total count in advance. Each matching element has a 1/k chance
> of being selected (where k = count of matches seen so far).

## How It Works

```
Target: 3
Stream: [1, 3, 5, 3, 7, 3]
         ^     ^        ^
         idx1  idx3     idx5

match 1 (idx=1): 1/1 chance -> pick idx 1
match 2 (idx=3): 1/2 chance -> maybe replace with idx 3
match 3 (idx=5): 1/3 chance -> maybe replace with idx 5

Each index has exactly 1/3 probability of being the final pick.
```

```js
class RandomPicker {
  constructor(nums) {
    this.nums = nums;
  }

  pick(target) {
    let result = null, matchCount = 0;

    for (let i = 0; i < this.nums.length; i++) {
      if (this.nums[i] === target) {
        matchCount++;
        // Replace result with probability 1/matchCount
        if (Math.floor(Math.random() * matchCount) + 1 === 1) {
          result = i;
        }
      }
    }
    return result;
  }
}
```
